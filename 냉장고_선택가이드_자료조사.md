> **각도 B: 형태·배치·Decision Tree.** 특정 제품/브랜드 추천 없이 '선택 기준'만 정리한 중립 매거진용 자료입니다.

## 1. 타입별 특징 — 장단점·용량대

> 핵심 구분축은 **문 배치 구조(상냉장/하냉동, 좌우분할)**입니다. 이 구조가 "무엇을 어디서 꺼내느냐"라는 사용성을 결정합니다.

### 1-1. 타입 비교표

| 타입 | 구조 | 일반 용량대 | 강점 | 약점 | 신뢰도 |
|---|---|---|---|---|---|
| **양문형 (Side-by-Side)** | 좌측 냉동·우측 냉장으로 **세로 분할**, 두 문이 나란히 | 약 400~850L대(800~850L급이 다수 판매) | 냉장·냉동 모두 상단부터 서서 사용, 동급 4도어보다 저가(가성비) | 칸 폭이 좁아 넓은 식재료 보관·안쪽 인출 불편, 부가기능 탑재 상대적 미흡 | 🔸 |
| **4도어 / FDR (상냉장 하냉동)** | 상단 냉장(2도어)·하단 냉동(2도어) | 약 800~900L대, 일부 900L 초과 | 냉장실 좌우 통합으로 폭넓은 식재료 수납·인출 유리, 최대 용량 선택 가능, 부가기능 풍부 | 냉동실이 전부 하단이라 허리 굽힘 필요, 동급 양문형보다 고가 | 🔸 |
| **상냉장 하냉동 2도어(일반형)** | 상칸 냉장·하칸 냉동 단순 분할 | 약 300~600L대 | 저가, 설치 공간 제약 적음, 신선식품이 눈높이 | (대용량 선택 폭 좁음) 냉동 인출 시 허리 굽힘 | 🔸 |
| **1도어 / 소형(미니)** | 단문, 상단 소형 냉동칸 | 약 20~200L대(미니 분류 100L 미만·높이 100cm 이하) | 이동·설치 자유, 협소 공간·보조용 | 상단 칸은 '강력 냉장' 수준(진짜 냉동 아닌 경우)·아이스크림 보관 한계, 본 냉장고 대체 불가 | 🔸 |

> ⚠️ 위 용량대는 시장에서 흔히 보이는 범위로, 모델별로 상하 폭이 있습니다. 단일 숫자로 단정하지 말고 범위로 제시하세요.

### 1-2. 하냉동 vs 상냉동 (해외 일반론)

미국 cu.ft 기준 자료라 한국 리터 시장과 직접 일치하진 않으나, 구조적 장단점은 보편적입니다.

| 구분 | 특징 | 신뢰도 |
|---|---|---|
| 하냉동형(Bottom-Freezer) | 신선식품이 눈높이에 와 접근 쉬움, 총 수납 큰 편, 가격대 높고 냉동 인출 시 허리 굽힘 | 🔸 |
| 상냉동형(Top-Freezer) | 더 저렴, 동급 하냉동 대비 에너지 약 10~25% 적게 쓰는 편(⚠️ 모델별 상이) | ⚠️ |

### 1-3. 김치냉장고 병용 관점

| 항목 | 내용 | 신뢰도 |
|---|---|---|
| 뚜껑형 vs 스탠드형 | 뚜껑형=직접냉각·정온성능 우수(김치 보관 최적), 스탠드형=수납 편의 우수 | ✅ |
| 실저장 비율 | 표기용량 대비 실제 김치저장 비율: **스탠드형 약 40% / 뚜껑형 약 67%** (한국소비자원) | ✅ |
| 실저장 용량 | 소비자원 6종 비교: 스탠드형 평균 159.1L vs 뚜껑형 140.7L (실차이 약 1.1배, 전체 유효용량은 스탠드형이 큼) | ✅ |
| 안전사용 권장 | 권장 사용기간 7년(화재사고 다수가 노후 제품) | 🔸 |

> **병용 판단:** 김치냉장고를 별도로 둘 계획이면 메인 냉장고는 한 단계 작게 잡아도 무방(🔸). 빌트인·키친핏처럼 용량이 작은 메인을 쓰면 김치냉장고로 보충하는 패턴이 흔함(🔸).

## 2. 설치 조건 — 폭·높이·방열·문 열림·빌트인

> **순서가 중요합니다: ① 반입 경로 → ② 설치 공간(방열 포함) → ③ 문 열림 반경.** 이 셋 중 하나라도 막히면 용량·기능 선택은 의미가 없습니다.

### 2-1. 방열 공간(이격거리)

| 출처(설치형태) | 후면 | 측면 | 상단 | 신뢰도 |
|---|---|---|---|---|
| **LG 공식**(프리스탠딩) | 10cm | 5cm | 최소 2cm | ✅ |
| **삼성 공식** | 약 5cm | 약 5cm | — | ✅ |
| LG 공식(가구장 빌트인형, 특정 모델) | 약 25mm | 4mm 이상 | 도어 상단 15mm 이상 | ✅(모델 의존) |
| GE 공식(미국) | 약 2.5~5cm | 약 0.3~2.5cm | 약 2.5cm | ✅ |

> 🔸 **종합:** 국내 프리스탠딩 기준 대략 **후면 5~10cm, 측면 5cm, 상단 2cm 내외**가 흔히 제시되는 범위. 단, 가구장 매립(키친핏)은 수치 체계가 다르므로 반드시 **해당 모델 설치설명서**를 따라야 합니다.
> ⚠️ 주의: "방열용 통풍 간격"(수 mm 단위)과 "도어 작동용 측면 여유"(수 cm 단위)는 별개 개념 — 단위 혼동 금지.

### 2-2. 문 열림 반경 / 도어 스윙

| 항목 | 내용 | 신뢰도 |
|---|---|---|
| 측면 여유(서랍·선반 인출) | LG 특정 모델: 도어 110도 개방 시 측면 최소 약 122~135mm 여유 필요 | ✅(모델별) |
| 벽 옆 설치 | KitchenAid: 손잡이가 벽에 닿지 않도록 측면 약 6.35cm 여유 권장 | ✅ |
| 양문형 전면 공간 | 문 폭이 넓어 열 때 앞쪽 60cm 이상 공간 필요(🔸) — 4도어는 문 하나가 작아 좁은 주방 유리 | 🔸 |
| 일반 원칙 | 도어가 완전히 열리지 않으면 내부 트레이·서랍 인출 곤란, 무리한 개폐는 힌지 마모 가속 | 🔸 |

### 2-3. 빌트인 vs 프리스탠딩 vs 키친핏(카운터뎁스)

| 구분 | 깊이 기준 | 의미 | 신뢰도 |
|---|---|---|---|
| 프리스탠딩 | 약 930mm(일부 자료 ~98cm) | 가구장 밖으로 보통 한 뼘 돌출 | 🔸 |
| 키친핏(세미빌트인) | 약 700mm | 가구장 라인에 맞춰 빌트인처럼 연출, 돌출 최소 | ✅(삼성 사양) |
| 빌트인 vs 카운터뎁스 개념 | — | "빌트인"은 설치 방식(스타일), "카운터뎁스/키친핏"은 앞뒤 깊이 치수 — 별개 개념 | 🔸 |

> **키친핏 트레이드오프:** 깊이를 줄이면 일반적으로 내부 용량 여유가 줄어듭니다(🔸). ⚠️ 슬림형의 정확한 리터 손실을 정량화한 신뢰 출처는 확인되지 않음 → 정성적 서술만 권장.
> **키친핏 설치 사양(삼성, ✅):** 매립 시 깊이 700mm 이상 냉장고장 필요(콘센트 내장 시 720mm 이상), 높이 1,900mm 이상 냉장고장, 제품-가구장 사이 최소 12mm.

### 2-4. 반입 경로·실무 노트

| 항목 | 내용 | 신뢰도 |
|---|---|---|

| 수평 설치 | 바닥 미수평 시 진동·소음·도어 어긋남·전도 위험 → 수평 필수, 바닥은 적재 하중 견뎌야 함 | ✅ |

| 콘센트 | ⚠️ 공식 cm 지침 미확인 → "제품 후면/측면 도달 위치에 사전 확보 권장" 수준으로만 서술 |

## 3. 인버터 컴프레서 & 에너지효율

### 3-1. 인버터 vs 정속형

| 구분 | 작동 방식 | 장점 | 트레이드오프 | 신뢰도 |
|---|---|---|---|---|
| **정속형(일반)** | 냉기 부족 시 최대 출력, 도달 시 OFF — **ON/OFF 반복** | 저가, 빠른 냉각, 구조 단순(정비 용이) | ON/OFF 전류 피크로 전력 소모·소음·온도편차 큼 | 🔸 |
| **인버터** | 멈추지 않고 부하에 비례해 **회전속도/출력 가변** | 전력 최소화, 정밀 온도 제어, 저소음·저진동 | 단가 ↑, 구조 복잡 | 🔸 |

> ⚠️ 해외 출처의 "인버터 25~32% 절감", "수명 20~25%↑", "소음 35~40dB vs 45~50dB" 등은 추정치 → 한국 모델 일반화 시 "(일반적/추정)" 표기.

### 3-2. 에너지소비효율등급(한국)

| 항목 | 내용 | 신뢰도 |
|---|---|---|
| 등급 체계 | 1~5등급(1등급일수록 절약, 색상 녹색→빨강) | ✅ |
| 라벨 표시 | 효율등급, **월간소비전력량(kWh)**, 연간 전기요금(원), CO₂ 배출량 | ✅ |
| 연간 요금 계산 | **연간소비전력량(kWh) × 약 160원** (예: 월 42.8kWh → 연 약 82,000원) | ✅ |
| 실사용 변수 | 동일 제품도 주위온도 16℃→32℃ 상승 시 소비량 최대 약 2.7배 증가 → 라벨 kWh와 사용환경 함께 봐야 | 🔸 |

> **핵심:** 등급(별표)보다 **라벨의 월간/연간 소비전력량 실수치(kWh)**가 비용 비교에 더 직접적입니다.
> ⚠️ 등급별 kWh 예시 수치(예: 830L 1등급 369.8kWh 등)는 과거 기준연도 값일 수 있어 기준이 지속 강화됨 — 인용 시 연도 명시 필요.

### 3-3. 타입별 소비전력·소음

| 항목 | 내용 | 신뢰도 |
|---|---|---|
| 타입 효율 | (미국 ENERGY STAR) **상부냉동(탑마운트)이 가장 효율적**, 양문형은 **효율이 낮은 편**(자동제빙·디스펜서 등 부가기능 영향) | ✅(단, "양문형이 최악"은 명시 아닌 함의 → "낮은 편"으로 완충) |
| 국내 비교시험 | 동급 4도어/양문형 간 월 소비전력 최대 1.5배(약 26~38kWh) 차이 사례 | 🔸 |
| 소음 | 국내 비교시험 약 36~40dB 범위 | 🔸 |

### 3-4. 컴프레서 보증

> "10년/20년 컴프레서 보증"은 선택 단서지만, 부품·인건비·냉매 보장 범위가 첫 1년만인 경우도 있어 **보증 범위(부품 vs 인건비)를 확인**해야 합니다(⚠️ 해외 사례 기준, 균형 서술용).

| **도어인도어/홈바/매직스페이스** | 문을 자주 여닫음(자녀·잦은 간식), 편의 우선 | 키 큰 양념통·물통 보관 많음, 예산 우선 | 냉기 손실 감소(정성 ✅, "30% 감소" 수치는 🔸). **Consumer Reports: 도어인도어가 냉장 성능 저해 안 함**(✅). 홈바는 가격차 큰 항목 | ✅/🔸 |
| **변온실(컨버터블)** | 식재료 종류 다양·소형이라 김치냉장고 미보유 | 보관 패턴 단순 | 칸별 맞춤온도로 김치냉장고 부분 대체 가능 | ✅ |
| **무빙바스켓·야채실 습도/신선보관** | 채소·과일 보관 많음 | — | 신선도·공간 최적화에 도움이나 핵심 대비 부차적 | 🔸 |
| **Wi-Fi/스마트(IoT)/AI 절전** | 원격 알림(문 열림·정전·유통기한)·식품관리 실제 활용 | 알림 잘 안 봄·단순 보관 위주 | 대다수 주류 모델 기본 탑재 → 차별화 요소 아님. "10~15% 절전"은 ⚠️ 단일 상업 수치 | 🔸/⚠️ |

> **유지비 관점:** 정수·제빙을 쓰면 6개월 주기 필터 교체비(소모품) + 누수 수리 리스크가 상시 유지비로 따라옵니다. 효율등급 1단계 차이의 금전 ROI는 4도어 기준 연 약 2.5만 원·10년 누적 30만 원 미만으로 낮은 편(🔸).

## 5. Decision Tree — 형태·배치 의사결정 분기

> **분기 원칙: 제약(설치) → 규모(가구수/용량) → 패턴(사용습관) 순으로 좁힙니다.** 뒤 단계가 앞 단계를 거스를 수 없습니다.

```
[START]
  │
┌─ 1단계. 설치 제약 (먼저 막히는 곳부터) ───────────────────────────┐
│                                                                   │
│  Q1. 반입 경로(현관·복도·엘리베이터·주방문) 폭이 제품 가로보다 큰가?
│       ├─ 아니오 → 폭 좁은 모델/일반형으로 후보 한정 (또는 사다리차 검토)
│       └─ 예 ↓
│
│  Q2. 설치 폭에 방열공간(측면 5cm·후면 5~10cm·상단 2cm 내외) 포함해 여유 있나?
│       ├─ 빠듯함 → 폭 좁은 타입(일반형/슬림 양문형) 우선
│       └─ 여유 ↓
│
│  Q3. 가구장과 평평하게(빌트인처럼) 맞추고 싶은가?
│       ├─ 예 → 키친핏(깊이 약 700mm) — 단 용량 여유 줄어듦, 김치냉장고 병용 고려
│       └─ 아니오 → 프리스탠딩(깊이 약 930mm, 돌출 감수)
│
│  Q4. 문 여는 쪽에 벽/주변가구 간섭이 있나?
│       ├─ 예 → 도어 방향(좌우 힌지) 변경 가능 모델 확인 (3단계 참조)
│       └─ 아니오 ↓
└───────────────────────────────────────────────────────────────────┘
  │
┌─ 2단계. 가구수 / 용량 ─────────────────────────────────────────────┐
│  (용량 권장치는 출처별 편차 큼 → 범위로만, "일반적/추정")           │
│                                                                   │
│  1인        → 약 200~350L   (일반형/소형 또는 슬림 양문형)
│  2인        → 약 200~500L   (일반형 또는 양문형)
│  3~4인      → 약 400~700L   (양문형 또는 4도어)
│  5인 이상   → 약 600~900L+  (대용량 4도어)
│
│  보정 룰오브섬: 한국 가이드 ≈ 1인당 100~150L + 여유
│  ※ 김치냉장고 별도 보유 시 메인 한 단계 작게 가능
│  ※ 표기용량의 실사용은 일반냉장고 약 60~70% 수준(🔸)
└───────────────────────────────────────────────────────────────────┘
  │
┌─ 3단계. 사용 패턴 (형태 최종 결정) ───────────────────────────────┐
│                                                                   │
│  Q5. 냉장 vs 냉동 어느 쪽을 더 자주 꺼내나?
│       ├─ 냉장 위주(신선식품 매일) → 상냉장 하냉동(4도어/일반형):
│       │     냉장이 눈높이, 넓은 식재료 수납 유리
│       └─ 냉동·즉석식품 비중 큼 → 양문형:
│             냉동도 서서 사용, 좌우 즉시 접근
│
│  Q6. 장보기 습관?
│       ├─ 대량·벌크 구매 → 한 단계 큰 용량 / 4도어
│       └─ 소량 자주 구매 → 표준~작은 용량
│
│  Q7. 문 여닫는 빈도(자녀·간식)?
│       ├─ 잦음 → 도어인도어/홈바로 냉기손실↓ (비용 감수 시)
│       └─ 보통 → 기본형
│
│  Q8. 얼음·정수 사용량?
│       ├─ 많음 → 디스펜서/제빙 (수납 -20~40L·필터 유지비·누수리스크 감수)
│       └─ 적음 → 미탑재(수동 제빙) — 가성비·고장포인트↓
└───────────────────────────────────────────────────────────────────┘
  │
[최종 후보 = 설치 가능 ∩ 적정 용량 ∩ 사용패턴 적합]
```

## 6. 도어 방향 & 키친핏 (보강 표)

### 6-1. 도어 방향 / 좌우 힌지

| 항목 | 내용 | 신뢰도 |
|---|---|---|
| 기본 원칙(LG 공식) | 도어는 기본적으로 변경 불가하게 설계, **일부 모델(상냉장 하냉동·컨버터블·일부 시리즈)만 좌우 변경 가능** | ✅ |
| 사용 중 변경 | 초기 설치가 아닌 사용 중 변경은 **서비스센터 유상**, 힌지 부품 분실 시 부품비 추가 | ✅ |
| 삼성 양문형 | 초기 설치 시 방향 변경 추가비 없음, 단 페어(2대) 설치는 조합에 따라 제한 | ✅ |
| 셀프 시공 | 보증 무효화 가능·문이 무거워 2인 작업 권장 | 🔸 |

### 6-2. 페어/벽 이격(삼성 공식, ✅)

| 상황 | 필요 간격 |
|---|---|
| 측면-벽 | 최소 12mm |
| 뒷면-벽 | 최소 약 50mm |
| 페어 설치(도어 서로 반대로 열림) | 7mm |
| 페어 설치(도어 같은 방향) | 15mm |

> ⚠️ 7mm/15mm 수치는 특정 1도어 변온/와인 페어링 유닛 기준으로 문서화 — 대상 양문형 페어에 동일 적용되는지 모델 확인 권장.
