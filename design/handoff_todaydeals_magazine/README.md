# Handoff: 오늘의딜 매거진 — 중립 쇼핑 가이드 웹진

## Overview
오늘의딜(todaydeals.co.kr)의 **별도 콘텐츠 섹션** — 광고·제휴 없이 **구매 판단의 기준만** 주는 중립 쇼핑 가이드 매거진. 딜 사이트가 "지금 싸요, 사세요"라면, 매거진은 **"이건 이렇게 고르세요"**. 슬로건: **"팔지 않습니다, 방향만 드립니다."**

URL 계획: `todaydeals.co.kr/magazine` (또는 `/guide`) — 기존 Next.js 앱에 **서브디렉터리 섹션**으로 추가(SEO 유리). 한글 서브도메인 비추(punycode).

## 디자인 파일
- `오늘의딜 매거진 - 디벨롭 시안.dc.html` — **메인 구현 대상.** 3개 프레임: ① 코너 시스템 + 정직성 배지(비주얼 아이덴티티 정의) · ② 매거진 메인(홈) · ③ 가이드 상세(끝장비교 — 모든 시그니처 컴포넌트 시연).
- `오늘의딜 - 쇼핑꿀팁 웹진 (3시안).dc.html` — 초기 3방향 탐색(A 클린카드 / B 랭킹리스트 / C 볼드블록). 참고용. 최종 방향은 **A 기반 클린·신뢰**로 확정됨.
- `support.js` — .dc.html 미리보기 런타임(구현 시 무시). `.dc.html`은 브라우저로 바로 열어 시안 확인.

`*.dc.html`은 **HTML 디자인 레퍼런스(시안)**다. 그대로 배포하지 말고 타깃 스택(Next.js + 선호 스타일링)으로 재현하라.

## Fidelity
**High-fidelity.** 색·타이포·간격·컴포넌트 구조를 정확히 재현. 썸네일/제품 이미지는 회색 플레이스홀더 → 실제 이미지(또는 CI 썸네일 템플릿)로 교체.

## 절대 원칙 (디자인·구현 모두 지킬 것)
1. **중립** — 특정 제품 추천/비방 안 함. 결론은 항상 **조건부**("A 상황이면 X, B면 Y").
2. **무광고·무제휴** — 본문에 구매/제휴 링크 **0**. "어디서 싸게"는 다루지 않음(그건 딜 사이트). 이 분리가 신뢰의 해자.
3. **통계 날조 금지** — "73%가 지적" 류 데이터 없는 숫자 금지(표시광고법·명예훼손·SEO 페널티). 정성적 표현("후기에서 반복되는 불만은…")으로.
4. **계산형 콘텐츠는 안전** — 제조사 공개 스펙으로 유지비 계산 = 정직·강력. 항상 **"추정치 · 실사용에 따라 다름"** 면책 표기.
5. **리뷰 대량 스크래핑 금지** — 후기 긁기는 약관 위반·차단.

## 핵심 컴포넌트 (재사용 템플릿 — 컴포넌트화 필수)

1. **정직성 배지** (`HonestyBadge`) — 모든 글 상단 고정. 원형 체크(오렌지 아웃라인) + "이 글은 끝까지 읽어도 특정 제품을 추천하지 않습니다 / 소비자의 바른 선택, 그 방향만 제시합니다." + `AD-FREE` 태그. 담백하게, 눈에 띄되 광고처럼 보이지 않게.
2. **코너 칩** (`CornerChip`) — 코너명 + 컬러. 글 카드·상세 상단·내비에 일관 사용.
3. **Decision Tree** (`DecisionTree`, 스마트가이드) — 질문 노드(오렌지/잉크) + YES/NO 분기 + 결과 칩(코너/분야 컬러). YES/NO 인포그래픽.
4. **끝장비교표** (`CompareTable`) — 잉크 헤더 행 + 행 줄무늬(`#FAF8F5`). 구분/항목 × 옵션 N. 종결형, 한눈에.
5. **유지비 매트릭스** (`CostMatrix`, 끝장비교) — 옵션별 3년 총비용을 **스택 가로 바**(전기 + 소모품)로. 총비용 Mono. 상단 **면책 문구 필수**("공개 스펙 기준 추정치"). 가장 큰 차별화 무기.
6. **체크리스트 카드** (`Checklist`) — 구매 전 체크 항목 + 오렌지 체크 박스.
7. **정직한 마무리** (`ClosingNote`) — 잉크 카드 + "선택은 당신의 몫입니다". 밀어붙이지 않는 마감.
8. **썸네일 템플릿** — CI 기반, 제목 자동 얹기 + 코너 라벨 노출(자동 생성).

## 페이지

### 매거진 메인 (`/magazine`)
헤더(로고+MAGAZINE / 코너 5 내비) → 슬로건 마스트헤드(+"무광고·무제휴") → 피처드 가이드(코너칩+분야태그+제목+읽기시간) → 최신 가이드 그리드(코너칩 색상 구분) → 푸터("광고·제휴 없이 운영됩니다").

### 가이드 상세 (`/magazine/[slug]`)
헤더 → 코너칩+분야 → H1+부제+바이라인(**편집국**, 개인명 X) → **정직성 배지** → 공감 도입 → (코너에 맞는 시그니처 컴포넌트) → 소제목+본문 반복 → **정직한 마무리** → 푸터. 읽기 컬럼 `max-width:720px`. **본문 내 구매·제휴 링크 없음.**

## 문체 (구현 시 카피 작성 규칙 — §5)
통일 존댓말(~합니다), 공감 도입 → 객관 기준 → 수치·표 → **판단은 독자**. "사세요/최고/사지마세요" 금지, 조건부 결론. 롤모델: 노써치(nosearch.com)식 중립 가이드.

## 인터랙션 / 반응형
- 카드 호버 `translateY(-3px)` + 그림자.
- 코너 내비 = 카테고리 라우팅. 코너칩/분야칩 = 필터.
- 반응형: ≤820px 그리드 1열, 비교표 가로 스크롤, Decision Tree 세로 스택, 유지비 바 유지. H1 `clamp`.

## 데이터 / 기술 (참고)
- 저장: Supabase 테이블(분야·코너 분류) + 글 페이지(메타·**Article JSON-LD**·sitemap).
- 생성 엔진: 클로드 기반(오늘의딜 API 키 재사용). AI 80% 취합·포맷팅 + 사람 20% 기획·**팩트체크**. 발행 전 사람 검증 필수(특히 수치·비교).
- 운영: 하루 1편, 에버그린 80%/트렌드 20%. 양치기(스케일드 콘텐츠) 금지.
