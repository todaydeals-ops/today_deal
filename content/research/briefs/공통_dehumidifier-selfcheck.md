# 제습기 자가점검: 물이 안 차임·제습 안 됨·냄새·고장 진단

## 개요

제습기가 제대로 작동하지 않는다고 느낄 때, 즉시 AS를 신청하기 전에 확인할 사항들을 조사했습니다. 대부분의 경우 사용자가 직접 해결할 수 있는 점검 항목들입니다.

## 물이 안 차는 경우

### 1단계: 습도 설정 확인
위닉스·LG 등 주요 제조사의 자가진단 가이드에 따르면, 제습이 안 되고 물이 안 차는 경우 **첫 번째 점검 항목은 습도 설정**입니다.
- **70% 이상으로 설정하면 자동 제습이 작동하지 않습니다.** 35~60% 범위로 조절하면 됩니다.
- 자동 제습 기능 자체가 꺼져 있지 않은지 확인하세요.

### 2단계: 물통 정위치 확인
- 물통을 빼서 정위치에 제대로 재조립했는지 확인합니다.
- 특히 위닉스 제습기는 물통 내 **부레(플로트)의 방향이 중요**합니다. 청소 후 재조립할 때 자석이 있는 부분이 센서 쪽으로 향하도록 주의해야 합니다.

### 3단계: 실내 습도 확인
- 봄, 가을, 겨울처럼 건조한 계절이라면 실제 실내 습도가 충분하지 않을 수 있습니다.
- 습도 범위가 이미 설정값 이하라면 물이 생기지 않는 것이 정상입니다.
- 필요하면 연속 제습 모드로 전환하면 됩니다.

### 4단계: 습도 센서 확인
습도 센서가 고장나면 제습기는 작동하지 않습니다. 위의 항목들을 확인했는데도 물이 안 차면 센서 고장을 의심하고 AS를 신청하세요.

## "물 가득 찼음" 표시가 계속 나오는 경우

### 증상
물통을 비워도 만수 표시나 "물통 비움" 알림이 계속 나타나는 경우입니다.

### 원인
물통 내부의 **수위 센서(플로트)가 고정되어 움직이지 않을 때** 이 증상이 발생합니다.

### 해결 방법
1. 물통을 제거합니다.
2. 물통을 위아래로 **가볍게 흔들어** 센서의 움직임을 테스트합니다.
3. 물통을 다시 장착합니다.
4. 3분 정도 기다리면 자동으로 재운전됩니다.

이 방법이 작동하지 않으면 서비스센터 점검이 필요합니다.

## 얼음이 얼어붙는 경우 (압축식 저온 결빙)

### 증상
압축식 제습기를 사용할 때 필터나 내부에 얼음이 보이는 경우입니다.

### 원인 — 고장이 아닙니다
18°C 이하의 저온 환경에서 압축식 제습기의 증발기에 응축수가 맺혔을 때, 그 수분이 얼어붙는 것은 **정상적인 물리 현상**입니다. 이는 고장이 아니라 저온 환경에서의 자연스러운 특성입니다.

### 제상(除霜) 운전
제습기가 결빙을 방지하기 위해 **제상 운전**을 자동으로 실행합니다. 이때 제습이 일시적으로 멈추는데, 이는 증발기를 녹이는 정상 작동입니다. 제상 운전이 끝나면 다시 제습합니다.

### 대처 방법
- 실내 온도를 올리면 성능이 정상 수준으로 회복됩니다.
- 저온 환경이 지속되는 지하·반지하·다다미실 등에서는 **데시칸트식 제습기**가 더 나을 수 있습니다(저온에서도 안정적).

## 냄새와 곰팡이

### 필터 청소
- 극세필터는 **월 1회 청소**를 권장합니다.
- 진공 청소기나 부드러운 솔로 표면 먼지를 제거합니다.
- 오염이 심할 때는 **중성세제 + 미지근한 물(40°C 이하)**로 세척합니다.
- 완전히 건조한 뒤 재장착하세요. 덜 마른 상태로 장착하면 곰팡이 원인이 됩니다.

### 물통 관리
물통을 주기적으로 세척하고 **완전히 건조**한 뒤 사용해야 냄새를 예방할 수 있습니다.

### 냉각핀(열교환기) 곰팡이
필터와 물통을 청소했는데도 냄새가 계속 나면, 원인은 습한 공기가 직접 닿는 **냉각핀의 곰팡이**입니다. 냉각핀은 자가 청소가 어려우므로 **서비스센터 점검**을 요청하세요.

### 자동 건조 기능 활용 (LG 제습기)
LG 제습기에는 자동 건조 기능(CO 표시)이 있습니다.
- 모드 버튼을 3초 이상 길게 눌러 활성화합니다.
- 운전 종료 후 팬을 계속 돌려 열교환기의 잔류 수분을 제거하여 곰팡이 발생을 예방합니다.

## 자동 정지 조건

### 습도 설정에 도달했을 때
설정한 습도에 도달하면 압축기만 정지합니다. 실내 습도가 다시 올라가면 자동으로 재운전됩니다.

### 물통이 가득 찼을 때
물통이 가득 차면 제습기는 자동으로 작동을 멈추며, 만수 표시등이 켜지고 경보음이 3회 울립니다. 물통을 비운 후 다시 사용하면 됩니다.

**주의**: 만수 용량은 물통 넘침을 방지하기 위해 실제 물통 용량보다 5~20% 적게 설정됩니다.

## 소음과 진동

### 정상 작동 소음
- **강풍**: 약 50dB 이상
- **약풍**: 약 40dB 정도
- 압축기가 울리는 **'우우웅~' 소음과 진동이 발생하는 것은 정상**입니다.

### 소음이 증가하는 원인
1. 필터에 먼지가 쌓여 팬에 부하가 걸린 경우 → **필터 청소**
2. 제습기를 직접 바닥에 두어 진동이 울려 증폭되는 경우 → **진동 방지 매트 사용**

### 소음 감소 방법
- 바닥 진동 방지 매트 사용
- 벽과 최소 **20cm 이상의 간격** 유지
- 필터를 주기적으로 청소

### 압축식 vs 데시칸트식
- **압축식**: 압축기 소음이 크므로 거실용 추천
- **데시칸트식**: 히터만 사용하여 소음이 적으므로 침실·조용한 공간에 추천

## 정리

| 증상 | 첫 점검 | 대처 |
|---|---|---|
| 물이 안 참 | 습도 설정, 부레 위치 | 설정값 조절, 부레 위치 확인 |
| 계속 "물 가득" | 센서 고정 | 물통 위아래로 흔들기 |
| 얼음 생김 (저온) | 온도 확인 | 고장 아님, 온도 올리면 회복 |
| 냄새 지속 | 필터·물통 청소 | 여전하면 냉각핀, 서비스센터 |
| 소음 증가 | 필터, 진동 | 청소 + 진동 방지 매트 |
