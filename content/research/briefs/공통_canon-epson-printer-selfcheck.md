## 3. 주요 발견사항

### 3.1 캐논 (Canon)

#### 에러코드 체계
- **형식:** 2~3자리 숫자(E5, P10 등) 또는 글자+숫자 조합
- **출처:** Canon 공식 사용자 안내서(USRMA-0806)
- **대응:** 사용자 안내서의 에러 코드 테이블에서 찾아 조치

#### 무한잉크 G시리즈 리셋
- **대상 모델:** G1000, G2000, G3000, G4000, G6000 계열 (2015~)
- **잉크 잔량 카운트 초기화:** 드라이버에서 각 색깔별 "Reset" 선택 후 탱크를 상한선까지 충전
- **강제 펌웨어 리셋:** 전원 버튼 누른 상태에서 중지/리셋을 2회 누름
- **주의:** 펌웨어 리셋은 모든 설정을 초기화하므로 필요한 경우만 진행

#### 무선 연결 자가점검
- **Wi-Fi 표시등 확인:** 표시등이 꺼져 있으면 Wi-Fi 버튼 활성화
- **네트워크 일치 확인:** 프린터와 기기가 동일 SSID에 연결
- **2.4GHz 지원 필수:** 대부분의 캐논 무선 프린터는 2.4GHz만 지원
- **WPS 활용:** WPS 버튼을 누르면 2분 내 자동 연결 가능 (보안 취약하므로 가정용만)

### 3.2 엡손 (Epson)

#### 폐잉크 패드 경고 (★ 중요 항목)
- **증상:** "프린터가 서비스가 필요합니다" 또는 "잉크 패드가 거의 가득 찼습니다" 메시지 → 인쇄 중단
- **원인:** 헤드 청소 및 인쇄 과정의 폐잉크가 패드 용량 도달
- **사용자 직접 교체 불가:** 공식 서비스센터 방문 필수
- **비용:** 보증기간 내에도 유상 서비스 (교체 비용 약 5만~8만원대)
- **연락처:** 한국엡손 1566-3515 또는 웹사이트 서비스센터 찾기
- **예상 주기:** 인쇄량과 헤드 청소 빈도에 따라 1~2년 간격

#### 프린트 헤드 막힘 자가점검
- **노즐 검사:** 제어판 또는 Windows 프린터 설정 > 유지보수 > 노즐 검사 → A4 컬러 패턴 인쇄
- **판별:** 패턴에서 청록, 진홍, 노랑, 검정 중 한 색이라도 끊어져 있으면 노즐 막힘 확인
- **헤드 청소:** 노즐 검사 > [프린트 헤드 청소] 메뉴 선택
- **반복 시도:** 청소 후 다시 노즐 검사하여 패턴 확인 (1~3회 반복 권장)
- **12시간 대기:** 3회 청소 후에도 개선 없으면 최소 12시간 기다린 후 재시도
- **주의:** 청소 시마다 잉크 소모 및 폐잉크 패드 수명 단축 → 최대 3회 제한

#### 무한잉크 L시리즈 잉크 레벨 초기화
- **방법:** EPSON Adjustment Program 다운로드 후 실행 > Check로 현재 잉크 패드 상태 확인 > 필요시 각 색깔 잉크 레벨 초기화
- **대상 모델:** L110, L121, L210, L220, L300, L310, L350, L360, L365, L385, L4160, L6270 등
- **위험 요소:** 프로그램이 모델별 상이하므로 정확한 모델명으로 검색 필수 (잘못된 버전 사용 시 인식 오류 발생)
- **tier:** 확인실패 (공식 조정 프로그램 존재는 확인되나 세부 링크·절차 미확보)

#### 인쇄 불량(흐림·줄) 해결
1. **노즐 검사:** 먼저 패턴에서 끊김 여부 확인
2. **끊김이 없는 경우:** 드라이버 설정에서 인쇄 품질을 "고품질" 또는 "최고 품질"로 올림
3. **끊김이 있는 경우:** 프린트 헤드 청소 진행 (위 3.2 참조)
4. **개선 없을 때:** 최소 12시간 대기 후 재시도

### 3.3 공통 항목

#### 스캔 작동 안 함
- **드라이버 재설치 (캐논·엡손 공통)**
  1. Windows 장치 관리자 > 이미징 장치 확인
  2. 모델명이 제대로 표시 안 되거나 느낌표 표시 시 우클릭 > 장치 제거
  3. 제조사 공식 웹사이트에서 최신 스캐너 드라이버 다운로드 및 설치
  4. Windows 재부팅 필수

- **연결 확인 (캐논·엡손 공통)**
  - USB: 케이블이 양쪽에 제대로 연결되었는지 확인
  - 무선: 프린터와 컴퓨터가 같은 네트워크에 연결, 방화벽 차단 여부 확인
  - USB 허브 사용 시 컴퓨터에 직접 연결하여 테스트

## 7. 결론 및 다음 단계

### 팩트 산출
- **신규 팩트:** 12건 (캐논 4건, 엡손 5건, 공통 3건)
- **재사용 팩트:** 6건 (HP 공통 원리)
- **거부 팩트:** 4건 (HP 고유 기능)

### 글 작성 시 주의사항
1. **엡손 폐잉크 패드는 필수 정보:** 사용자가 당황할 수 있는 대표 항목이므로 상단에 배치
2. **모델별 차이 명시:** G시리즈/L시리즈 등 무한잉크 타입별로 절차가 다름
3. **순차 대안 톤 유지:** "먼저 노즐 검사 > 끊김 있으면 헤드 청소 > 12시간 대기 후 재시도"
4. **서비스센터 안내:** 폐잉크 패드, 드라이버 재설치 후에도 안 되는 문제는 공식 채널로
5. **2.4GHz 주의:** 캐논 무선 프린터 대부분 2.4GHz만 지원하는 점 강조
