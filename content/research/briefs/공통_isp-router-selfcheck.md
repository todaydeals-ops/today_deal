# 통신사 공유기(KT·SK브로드밴드·LG유플러스) 인터넷 안 됨 자가점검 조사

## 조사 목표
통신사 공유기 사용자가 인터넷 연결 문제를 자가 진단하고 해결할 수 있도록 필요한 팩트 수집.
- 기본 점검 절차 (모뎀·공유기 재부팅)
- 통신사별 관리자 페이지 접속 방법
- LED 표시등 의미 (통신사별)
- 와이파이 연결되는데 인터넷 안 될 때 진단
- 특정 기기만 안 될 때
- 속도 저하·채널 간섭
- 고장 신고·원격 진단

## 조사 결과

### 1. KT 공유기

**관리자 페이지 접속**
- 주소: 172.30.1.254 또는 ktrouter.com
- 구형·독립형 모델: 192.168.1.1
- 계정: ktuser (아이디), 비밀번호는 모델마다 다름

**기본 비밀번호 (모델별)**
- 홈허브/홈허브 스페셜: ktuser / megaap
- 올레 기가와이파이 홈: ktuser / homehub
- 공유기 하단 스티커 확인 필수

**고장 신고**
- 마이케이티 앱: 고객지원 → 셀프진단 및 A/S 신청
- 전화: 100번 (KT 전화, 무료) 또는 080-000-9999
- 원격 진단 가능 (본인 명의 상품만)

### 2. SK브로드밴드 공유기

**관리자 페이지 접속**
- 공식 표준 주소가 명확하지 않음
- 공유기 하단 스티커 또는 설정서 확인 필수
- 일반적으로 192.168.1.1 또는 모델별 고유 주소

**기본 비밀번호**
- 공유기 하단/뒷면 스티커에 기재
- 모델마다 다름

**고객센터**
- ARS 106번 (무료, 연중무휴 24시간)
- ARS → 1번 선택 → 기술지원 상담사
- 온라인 접수: B world 홈페이지 (bworld.co.kr)

### 3. LG유플러스 공유기

**관리자 페이지 접속**
- 주소: http://192.168.219.1
- 표준화된 주소 (전 라인)

**기본 비밀번호**
- 공유기 하단/뒷면 스티커에 기재
- 모델마다 다름
- 분실 시: 고객센터(101번) 또는 공장 초기화

**고객센터**
- ARS 101번 또는 1544-0101 (무료)
- U+ 앱 챗봇 상담 가능
- 영상통화 원격 지원 제공

### 4. 인터넷 안 됨 기본 점검 (공통)

**1단계: 재부팅 순서**
1. 모뎀 전원 케이블 뽑음 → 3초 대기 → 재연결
2. 공유기 전원 케이블 뽑음 → 3초 대기 → 재연결
   - 반드시 모뎀을 먼저 끌 것
   - 동시에 뽑지 말 것

**2단계: LED 표시등 확인**
- 흰색 또는 초록색: 정상
- 빨간색: 공유기 문제
- 꺼져 있음: 전원 또는 연결 문제

**3단계: 인터넷 포트 확인**
- 공유기 뒷면에 모뎀에서 오는 랜선이 제대로 연결되었는지 확인
- 분리했다가 재연결해볼 것

### 5. 와이파이는 잡히는데 인터넷 안 됨

**진단 단계**
1. LED 확인 (위 참조)
2. 랜선 재연결 (위 참조)
3. Windows ipconfig /all 실행 → DNS 정보 확인
   - DNS가 0.0.0.0 또는 비어 있으면 공유기·모뎁·통신사 신호 장애
   - macOS: System Preferences → Network에서 확인

**해결책**
- LED와 랜선이 정상인데도 DNS가 없으면 고객센터 문의 필수

### 6. 와이파이 속도 저하·채널 간섭

**2.4GHz vs 5GHz**
- 2.4GHz: 범위 넓음, 간섭 많음 (전자레인지·무선마우스와 간섭)
- 5GHz: 범위 좁음, 빠름, 간섭 적음
- 듀얼밴드 공유기: 기기별로 대역 선택 가능

**간섭 해결**
- 공유기 관리자 페이지에서 채널 수동 변경 또는 자동 선택 사용
- 이웃 공유기와 같은 채널을 쓰면 간섭 발생

### 7. 자가점검 vs 기술 A/S

**자가점검으로 보통 해결되는 것**
- 재부팅 (모뎀 먼저, 3초 대기)
- LED·랜선 확인
- 와이파이 비밀번호 재입력
- 기기별 와이파이 설정 초기화

**기술 A/S가 필요한 경우**
- 위의 모든 것을 해도 인터넷 안 됨
- 모뎀이 너무 뜨거움 (만지기 힘들 정도)
- 통신사 신호 자체가 문제 (LED가 계속 빨강)

## 제한사항 및 미확인 사항

### 확인 실패
- SK브로드밴드 공유기의 표준 관리자 페이지 주소: 공식 문서에 명확하지 않음. 모델마다 다를 수 있으므로 사용자는 공유기 스티커 또는 고객센터 문의 필요.

### 추가 조사 불필요 (다음 글에서 해결)
- 특정 기기만 와이파이 연결 안 될 때: 이는 기기의 무선 설정 문제일 가능성이 높아, 공유기 자가점검이 아닌 기기별 조사 필요
- 공유기 채널 변경 방법: 각 통신사의 관리자 페이지 메뉴 구조가 다르므로, 개별 글에서 스크린샷과 함께 상세 설명 권장
