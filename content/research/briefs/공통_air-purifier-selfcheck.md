# 공기청정기 냄새·필터 교체·풍량 저하 자가점검 — 조사 전문

## 개요

공기청정기는 필터 구성과 교체 주기, 센서 청소 방법, 알림 리셋이 브랜드·모델별로 상이하다. 이 조사는 삼성·LG·위닉스의 공식 문서 기준으로 공통 원리를 정리하고 브랜드별 차이를 명확히 한다.

## 필터 구성: 세척 vs 교체

### 세척 필터 (재사용 가능)
- 프리필터, 먼지거름필터, 전기집진필터 등
- 미지근한 물(40℃ 이하)에 중성세제를 풀어 세척 가능
- **반드시 완전히 건조 후 사용** (최소 24시간)
- 월 1회 이상 정기 세척 권장

### 교체 필터 (일회용)
- 헤파필터, 탈취필터
- **물세척 불가** (재질 손상)
- 진공청소기나 부드러운 솔로만 먼지 제거
- 수명 종료 시 새로 교체해야 함

## 필터 교체 주기

**기준**: 1일 24시간 사용 기준 → 6개월 ~ 1년

**변수**:
- 실내 공기 오염도 (높을수록 단축)
- 일일 평균 사용 시간
- 계절 영향 (겨울 난방 시 필터 부하 증가)

**실제 교체 신호**:
1. 필터 교체 램프 점등 (제일 중요)
2. 냄새 발생 (탈취필터 수명 종료)
3. 바람세기 급감 (프리필터 오염)
4. 센서 표시 지속 (센서 오염)

## 냄새의 세 가지 원인

### 1. 필터 수명 종료
- 탈취필터의 흡착 능력 소진
- 프리필터에 쌓인 먼지
- **대응**: 필터 교체

### 2. 습도 (곰팡이)
- 내부 습도 70% 이상
- 특히 비오는 계절에 발생
- **대응**: 프리필터 세척 후 완전 건조, 내부 환기

### 3. 센서 렌즈 오염
- 센서가 냄새를 감지하지 못해 표시 오류
- **대응**: 면봉으로 렌즈 청소

## 풍량 저하의 원인

### 1. 필터 오염 (가장 흔함)
- 프리필터 먼지 축적
- 헤파필터 밀도 오염
- **대응**: 필터 세척 or 교체

### 2. 알림 미리셋
- 필터 교체 후 알림을 리셋하지 않음
- 제품이 저풍량 모드로 자동 제한
- **대응**: 모델별 리셋 버튼 조합 실행

### 3. 필터 오염도 센서 반응
- 센서가 필터를 교체해도 먼지가 많다고 판단
- 원인: 센서 렌즈 오염
- **대응**: 센서 렌즈 청소

## 오염도 센서 관리

### 센서 렌즈 청소
- **빈도**: 월 1회 (먼지 많은 환경은 주 1회)
- **방법**: 면봉에 물을 살짝 묻혀 렌즈 닦기 → 마른 면봉으로 수분 제거
- **중요**: 센서에 먼지가 끼면 필터를 교체해도 계속 높은 오염도 표시

### 센서 정상 작동 테스트
**LG**: 향수/세제를 티슈에 뿌려 센서에 대면 색상 변화 → '[냄새]' 표시
**LG**: 종이를 찢어 먼지 입자 발생 → 디스플레이 수치 증가 확인

### 신기술
LG 신규 모델(25년 1월~)에는 **레이저 센서** 탑재 → 자동 청소 기능으로 수동 청소 불필요

## 소음 증가의 원인

### 1. 필터 오염으로 인한 팬 부하
- 차단된 공기 흐름 → 팬 과부하
- 공기 통과 시 소음 증가
- **대응**: 필터 세척/교제

### 2. 모터 베어링 마모
- 회전축 안정성 저하
- 진동 및 '웅웅' 소음
- 내부 먼지 유입 → 마찰 증가
- **대응**: 전문가 점검

## 필터 정품 vs 비정품

### LG의 정품 인식 센서 (25년 1월 이후)
- **비정품 사용 시**: 경고 표시 + 기능 제한
- **정품 구입 경로**: LG전자 서비스센터 or 케어용품 소모품샵
- **비정품의 위험**: 절반 이하 가격이지만 성능 미보증, 살생물 물질 검출 사례

### 삼성·위닉스
- 명시된 정품 인식 기능 없음
- **대응**: 공식 채널(서비스센터, 공식 온라인샵)에서만 구매

## 브랜드별 필터 교체 알림 리셋

### 삼성
- **블루스카이**: 조명/필터 리셋 버튼 3초 이상
- **큐브**: 모드 버튼 10초 이상

### LG
- 모델별 상이 (버튼 조합이 다름)
- 1단: [청정세기 + 먼지상태] 3초 이상
- 2단 상부: 동일
- 2단 하부: [먼지상태 + 취침예약] 3초 이상
- **반드시 제품 설명서 확인**

### 위닉스
- 조사 범위 내 명시 정보 없음
- **권고**: 고객센터(1544-5081) 문의

## 자가진단 기능

### 삼성
- **스마트씽스 앱**: 필터 수명 상태 확인 가능
- **블루스카이**: 필터 수명 게이지 → 실제 필터 상태 기반 (단순 시간 경과 X)

### LG
- 스마트씽스 비수록 확인되지 않음
- 필터 교체 알림으로만 판단
