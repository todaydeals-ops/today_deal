# 삼성 공기청정기 필터 교체·자가점검 — 조사 전문

## 조사 범위

삼성 공기청정기의 필터 교체, 필터 교체 알림 리셋, 센서 자가진단에 관한 공식 문서 조사.

## 필터 구성 및 교체 방법

삼성 공기청정기는 **세척 필터**(프리필터, 먼지거름필터, 전기집진필터)와 **교체 필터**(헤파필터, 탈취필터)로 구성된다.

### 프리필터 세척
- 40℃ 이하 미지근한 물 + 중성세제
- 완전히 건조 후 사용 (최소 24시간)
- 월 1회 이상 권장

### 헤파필터 관리
- 물세척 불가 (재질 손상)
- 진공청소기 또는 부드러운 솔로만 청소
- 필터 오염이 심하면 교제 권장

## 필터 교제 주기

**기준**: 1일 24시간 사용 기준 → 약 1년 (범위: 6개월 ~ 1년)

필터 교체 알림 램프가 점등되면 반드시 교체해야 한다. 알림 전이라도 냄새가 나거나 바람세기가 약해지면 조기 교체 권장.

## 필터 교체 알림 리셋

### 블루스카이 시리즈
**방법**: 조명/필터 리셋 버튼을 3초 이상 누름

### 큐브 시리즈
**방법**: 모드 버튼을 10초 이상 누름

### 주의
- **모델별로 리셋 방법이 다르다**
- 제품 설명서를 반드시 확인해야 함
- 잘못된 버튼 조합 시도는 피할 것

## 자가진단 기능

### 스마트씽스 앱 연동
- 필터 수명 상태를 앱에서 실시간 확인 가능
- 자가진단 모드로 필터 수명 확인

### 블루스카이의 고급 기능
- **필터 수명 게이지**: 에어내비게이터에 표시
- **특징**: 단순 시간 경과가 아닌 **실제 필터 오염 상태를 기반**으로 표시
- 공기 오염도가 낮은 환경에서는 표시된 시간보다 더 오래 사용 가능

## 냄새 원인 진단

1. **프리필터 먼지**: 먼저 프리필터 세척 시도
2. **필터 수명 종료**: 필터 교체 알림 확인 → 교체
3. **센서 오염**: 면봉으로 센서 렌즈 청소

## 풍량 저하 원인

1. **프리필터 오염**: 진공청소기로 청소 후 물세척
2. **헤파필터 오염**: 진공청소기로 가볍게 청소
3. **알림 미리셋**: 새 필터 장착 후 버튼 조합으로 리셋

## 소음 증가 원인

### 필터 오염
- 필터 오염으로 공기 흐름 차단 → 팬 과부하 → 소음 증가
- **대응**: 필터 청소 or 교제

### 모터음
- 베어링 마모로 인한 진동음
- 내부 먼지로 인한 마찰음
- **대응**: 전문가 점검
