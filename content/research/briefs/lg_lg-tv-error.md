# LG TV 화면 안 나옴·신호없음·전원 문제 자가점검 — 조사 전문

**조사 대상:** LG TV 화면 안 나옴, 신호없음, 전원 문제의 원인과 자가 점검 절차  
**기간:** 2026-07-21  
**1차 출처:** LG전자 공식 고객지원(스스로해결)  
**조사 범위:** webOS 현행 TV · 국내 판매 모델

---

## 조사 결과 개요

LG TV 화면·신호 문제는 **외부기기 연결(셋톱박스) 불량·HDMI 케이블 접촉 불량·입력 소스 오류**가 90% 이상의 원인으로, TV 자체 하드웨어 문제는 드물다. LG는 webOS 기반 자가진단 기능을 제공하며, 2019년 이후 모델은 HDMI 케이블 테스트 기능을 포함한다.

---

## 1. TV 화면·소리 모두 안 나올 때 기본 점검

### 첫 확인 — TV 자체 정상 여부
리모컨 볼륨 버튼을 눌렀을 때 **음량 조절 표시가 화면에 나타나는지**로 판별한다.
- **표시가 나타남** → TV 정상, 외부기기·신호 문제
- **표시 없음** → TV 전원 또는 연결 문제

### 단계별 조치 순서
1. **외부기기 확인**: 셋톱박스·공유기 전원이 켜져 있는지 확인
2. **HDMI 케이블 재연결**
   - TV와 셋톱박스의 전원을 완전히 끈 후
   - HDMI 케이블을 양쪽에서 모두 분리
   - 케이블을 완전히 밀어 다시 연결(꺾여 있지 않은지 확인)
3. **입력 소스 선택**
   - 유선방송/안테나: "현재 방송" 선택
   - 셋톱박스 등 외부기기: "HDMI" 선택
4. **셋톱박스 재부팅**: 후면 전원 선을 뽑았다가 다시 연결

### 주의
HDMI 케이블 문제가 반복되면 정품 고속 HDMI 케이블로 교체해야 한다.

---

## 2. '신호없음' 메시지 표시 시 조치

### 원인 분석
신호없음 메시지는:
- 외부입력 선택 오류
- HDMI 연결 불량
- 셋톱박스 전원 문제
- 방송 신호 문제

를 나타낸다.

### 기본 조치 (위의 1단계와 동일)
- TV·셋톱박스 전원 확인
- HDMI 케이블 양쪽 재연결
- 입력 소스 재설정

### 유선방송/안테나 사용 시 추가 조치
신호없음 후 채널이 검색되지 않으면 자동 채널 설정을 실행해야 한다.

**webOS 6.0 이후:**
설정 → 전체 설정 → 일반 → 방송 채널 → 채널 찾기 → 자동 채널

(TV 모델·webOS 버전에 따라 메뉴 경로가 다를 수 있으므로 설정을 직접 탐색해야 함)

---

## 3. LG 매직리모컨 페어링

### 초기 페어링
1. TV를 켜고 약 20초 대기
2. 리모컨의 **휠(확인) 버튼**을 누름
3. 화면에 "등록 완료" 메시지가 나타남

### 페어링 실패 시 재등록
포인터가 화면에 나타나지 않거나 리모컨이 반응하지 않으면:
1. **이전 버튼**과 **설정 버튼**을 동시에 5초 이상 길게 누름
2. 리모컨 LED가 깜박이며 등록 해제 알림이 TV에 나타남
3. **휠 버튼**을 누름
4. "재등록 완료" 메시지 확인

### 주의
재등록 완료 메시지가 반드시 화면에 나타나야 한다.

---

## 4. 전원 관련 문제

### 전원 표시등(LED)이 꺼진 경우
- TV 전원 공급 단절 또는 설정 메뉴의 전원 대기 등 기능이 꺼진 상태
- 전원 코드 연결 상태를 확인해야 함

### 일시적 먹통/재시작 필요 시 (소프트웨어 오류)
1. 전원 코드를 완전히 분리
2. 충분히 대기 (30초 이상 권장)
3. 다시 연결하여 TV 자동 부팅 완료 대기
4. **과정 중 TV 전원 버튼을 누르면 안 됨**

---

## 5. LG webOS 자가진단 기능

### 접근 경로
설정 버튼 → 전체 설정 → 고객지원 → 퀵 헬프 또는 상태 진단

(메뉴명과 위치는 TV 모델과 webOS 버전에 따라 다름)

### 자가진단 항목 (2019년 이후 webOS)
- **HDMI/RF 점검**: HDMI 케이블 연결 상태 자동 테스트
- **상태 진단**: TV 각 부분의 정상 여부 확인

### HDMI 테스트 결과 활용
진단 결과가 "케이블 불량" 또는 "연결 불량"으로 나오면:
1. 먼저 같은 위치에 HDMI 케이블을 다시 연결해본다
2. TV의 다른 HDMI 포트로 연결해본다
3. 반복해도 실패하면 정품 고속 HDMI 케이블로 교체

---

## 6. HDMI 케이블 손상 진단

### 케이블 상태 확인 (육안)
- 커넥터의 금속 핀이 휘었는지 확인
- 케이블이 심하게 꺾여 있지 않은지 확인
- 이물질(먼지, 이슬)이 없는지 확인

### 손상 판별 기준
- 외관상 손상이 보이거나
- LG 자가진단에서 "케이블 불량" 진단을 받으면
- 정품 고속 HDMI 케이블로 교체 필요

---

## 7. TV 초기화(공장 초기화)

### 사용 목적
리모컨 오류, TV 소프트웨어 오류, 설치된 앱 오류 등을 해결할 수 있으나, 전원·화면·음성 하드웨어 문제는 해결되지 않는다.

### 절차
설정 → 전체 설정 → 고객지원 → 초기화 또는 리셋

### 주의사항
- 초기화 완료까지 전원 코드를 분리하면 안 됨
- 모든 설정과 앱이 제거되고 원래 상태로 복구됨

### 1차 출처 (공식)
LG전자 고객지원 "스스로해결" 페이지:
