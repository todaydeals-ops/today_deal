# 프로젝터 자가점검 조사 기록

## 조사 범위
- LG 시네빔 (국내 보급 빔프로젝터)
- 삼성 더프리스타일 (휴대용 프로젝터)
- 업계 표준 기술 (HDMI, 투사 거리, 키스톤 등)

## 조사 항목별 정리

### 1. 화면 안 나옴 및 신호 없음 (HDMI 입력)

**LG 공식 자가점검 가이드** ([LG 빔프로젝터 영상 미표시](https://www.lge.co.kr/support/solutions-1779994))
- 첫 확인: HDMI 케이블 양쪽 끝 완전 재연결
- 케이블 손상 확인 (꺾임, 단선)
- 입력 소스 메뉴 확인 (HDMI 1/2, 색상, 영상 등)
- 외부기기(셋톱박스, 컴퓨터) 전원 및 신호 확인
- 중요: RGB to HDMI 젠더 사용 금지 (HDMI to RGB 젠더 또는 직접 HDMI 연결 필수)
- 케이블만 제거해도 신호가 안 나오면 제품 불량 → LG서비스 문의(1544-7777)

**특이사항:** 프로젝터는 TV처럼 리모컨의 '외부입력' 버튼으로 소스를 선택하지 않고, 프로젝터 자체 메뉴에서 입력 소스를 선택해야 함. 따라서 셋톱박스 팩트는 재사용 불가.

### 2. 화면 흐림, 초점 문제

**삼성 더프리스타일 공식 가이드** ([자동 키스톤과 자동초점 해결](https://www.samsungsvc.co.kr/solution/1716526))

**자동 초점 오동작 원인:**
- 렌즈 보호 필름이 아직 붙어있음 → 반드시 제거
- 렌즈에 먼지나 이물질 → 부드러운 천으로 가볍게 닦음
- 렌즈 상단 센서에 먼지 → 면봉으로 부드럽게 청소
- 프로젝터 설치 방향 확인 (삼성 로고가 아래로)
- 외부 조도, 거리, 투사 면의 영향

**수동 초점 조정:**
- 경로: 홈 → 메뉴 → 설정 → 전체 설정 → 일반 및 개인정보 → 프로젝터 설정 → '초점' 항목
- 수동 초점도 동작하지 않으면 제품 내부 점검 필요 → 서비스센터 방문

### 3. 화면 기울어짐, 사다리꼴 왜곡 (키스톤 보정)

**삼성 더프리스타일 공식 가이드**

**자동 키스톤:**
- 더프리스타일은 기기 이동 시 자동으로 화면을 직사각형 모양으로 보정
- 자동 키스톤이 동작하지 않으면 렌즈 보호 필름 제거 확인
- 렌즈/센서 청소 필요
- 외부 조도, 거리, 투사 면이 영향을 줄 수 있음

**자동 재조정 비활성화:**
- 자동 조정이 빈번하면 설정에서 '기기 이동 후 자동 키스톤 조정' 해제 가능
- 해제 후에는 프로젝터 이동 시 수동으로 다시 조정해야 함

**수동 키스톤 조정:**
- 경로: 홈 → 메뉴 → 설정 → 전체 설정 → 일반 및 개인정보 → 프로젝터 설정 → '키스톤' 항목
- 기울어진 화면을 사각형으로 보정
- 응급 조치일 뿐, 프로젝터의 각도/높이를 정확히 조정하는 것이 더 나은 이미지 제공

### 4. 렌즈 직접 보기의 위험성

**제조사 공식 경고 (Optoma, Panasonic)**

**Optoma DLP 프로젝터 사용 설명서:**
> "프로젝터가 작동하고 있을 때 프로젝터 렌즈를 똑바로 들여다보지 마십시오. 밝은 빛이 눈을 손상시킬 수 있습니다."
> "프로젝터를 켤 때 투사 범위 내의 사람이 렌즈를 들여다 보지 못하도록 하십시오."

**Panasonic LCD 프로젝터 사용 설명서:**
> "위험 거리 내에서 (RG3 범위 내) 투사된 광원을 쳐다보지 마십시오. 직접 조사로 인해 눈이 손상될 수 있습니다."
> "프로젝터 사용 중 렌즈에서 나오는 빛에 피부를 노출시키거나 빛을 직접 보지 마십시오."

**위험 수준:**
- 고출력 광이 안구로 직접 조사되는 경우 안구 손상 초래
- 극단적인 경우 실명 가능
- 특히 아이들의 경우 시력 발달이 미숙하므로 주의 필수

**안전 조치:**
- 렌즈 청소 시에도 프로젝터 작동 중지 필수
- 아이들의 렌즈 접근 감시 필수
- 렌즈 점검 시 강한 빛이 피부에 직접 닿지 않도록 주의

### 5. 팬 소음, 발열, 통풍

**LG 공식 가이드** ([LG 빔프로젝터 소음](https://www.lge.co.kr/support/solutions-20151952633740))

**팬 소음 원인:**
- 프로젝터는 밝은 빛 투사로 내부 온도 상승
- 냉각을 위해 팬이 회전하며, 이는 정상 작동
- 주변 온도 높음 → 팬이 더 강하게 동작 → 소음 증가
- 환기 부족 → 내부 온도 상승 → 팬 소음 증가

**통풍 개선:**
- 프로젝터 주변 장애물 제거
- 통풍이 잘되는 곳으로 이전
- 선반/케이스 내부 설치 시 앞뒤좌우 최소 10cm 이상 공간 필요
- 통풍구가 완전히 막히지 않도록 확인

**비정상 소음:**
- 지속적인 큰 소음 → 팬 고장 또는 내부 이물질
- 서비스센터 문의 필요

### 6. 과열 보호 및 자동 종료

**LG 공식 안내**

**자동 종료 조건:**
- 내부 온도 40°C 이상 상승 시 제품 손상/화재 예방을 위해 자동 전원 차단
- 정상 보호 기능

**대응 방법:**
- 통풍 개선 및 실내 온도 낮춤
- 충분히 냉각될 때까지 대기 (2~3분)
- 재부팅

**주의:**
- 무리하게 계속 사용하면 내부 부품 손상 위험
- 자동 종료는 제품 자체를 지키기 위한 정상 기능

### 8. 화면 크기와 투사 거리

**업계 표준 (Optoma, Sony, Digital Projection)**

**투사 비율(Throw Ratio, TR) 개념:**
- TR = 스크린 가로 길이 ÷ 투사 거리
- TR이 작을수록 (예: 0.5:1) 가까운 거리에서 큰 화면 가능
- TR이 클수록 (예: 1.5:1) 먼 거리 필요

**예시:**
- 100인치 화면 목표
  - TR 0.5:1 프로젝터: 약 1.3m 거리
  - TR 1.5:1 프로젝터: 약 4m 거리

**결정 기준:**
- 설치 공간에 따라 프로젝터 모델 선택
- 각 프로젝터의 스펙에서 투사 비율 확인
- Optoma/Sony 투사 계산기 활용 가능

**온라인 계산 도구:**
- [Optoma 투사 계산기](https://www.optoma.com/projectioncalculator/?region=Korea)
- [Sony 투사 시뮬레이션](https://pro.sony/ko_KR/products/professional-projectors/projection-simulation-software-installers)

## 재사용 팩트 검토

**검토 대상:**
- `settop-hdmi-cable-reconnection` (셋톱박스 HDMI 재연결)
- `settop-input-source-selection` (셋톱박스 입력 소스)

**거부 사유:**
- 셋톱박스: HDMI를 TV와 단순 연결
- 프로젝터: HDMI를 컴퓨터/태블릿/셋톱박스 등 다양한 기기와 연결
- 셋톱박스: 입력 소스를 TV 리모컨의 '외부입력' 버튼으로 선택
- 프로젝터: 입력 소스를 프로젝터 자체 메뉴에서 선택
- **모델·연식·절차가 안 맞음 → 안 쓰는 것이 정답**
