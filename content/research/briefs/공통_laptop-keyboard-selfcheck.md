# 노트북 내장 키보드 자가점검 조사 전문

## 조사 범위

- 브랜드 공통 (LG 그램, 삼성 갤럭시북, 일반 Windows 노트북 등)
- **내장 키보드만 다룸** (무선 외장 키보드·터치패드는 별도 팩트)
- 집중 대상: 필터키/고정키 접근성 설정, 한/영 전환, 고스트 입력, 액체 피해

## 주요 발견사항

### 1. 접근성 설정이 원인인 경우 (높은 빈도)

**필터키(Filter Keys)** — 가장 흔한 오진 사례
- 켜져 있으면: 짧은 키 입력 무시, 길게 눌러야만 입력됨
- 자동 활성화: 오른쪽 Shift 8초 누르기 (의도 없이 발생 가능)
- 해제: 설정 > 접근성 > 키보드 > 필터 키 OFF
- **안내**: 이 현상은 "하드웨어 고장"으로 오인하는 독자가 많으므로, 먼저 접근성 설정부터 확인하도록 안내 필요

**고정키(Sticky Keys)** — 게이머나 프로그래머가 자주 겪음
- 켜져 있으면: Shift 5회 누르기로 자동 활성화, Ctrl+Alt 등 조합 키 순차 입력 가능
- 언제 문제: 게이밍 중 갑자기 최소화·포커스 상실
- 해제: 설정 > 접근성 > 키보드 > 고정 키 OFF

### 2. Num Lock 숫자/문자 혼동

**증상**: U, I, O, J, K, L 등 특정 문자 키가 갑자기 숫자로 입력됨
- 원인: Num Lock 켜짐 (노트북은 숫자 패드 대신 문자 키 일부를 전환)
- 해제: Fn + Num Lock (또는 Fn + F11, 모델별 다름)
- **주의**: 제조사별로 단축키가 상이하므로, 정확한 키는 매뉴얼 필수

**제조사별 차이:**
- 삼성: Fn + F11 (자물쇠 모양)
- LG/HP/Lenovo: Fn + Num Lock
- 정확한 키는 기종 매뉴얼 참조 필요

### 3. 한/영 전환 불가

**주요 원인 3가지** (중요!)
1. **하드웨어 키보드 레이아웃** 설정이 '101' (Alt 기반)으로 되어 있는 경우
   - 해결: 설정 > 언어 > 한국어 옵션 > 하드웨어 레이아웃을 '103/106: 한/영 키 포함'으로 변경
2. **입력 언어 단축키** 설정 변경됨
   - 확인: 설정 > 언어 & 지역 > 고급 > 입력 방법 에서 한/영 단축키 재설정
3. **IME 프로세스** (ctfmon.exe) 미실행
   - 해결: 작업 관리자에서 새 작업으로 ctfmon.exe 실행

### 4. 고스트 입력 (Ghost Keypress)

**증상**: 누르지 않은 키가 입력되거나, 키가 무한 반복 입력됨

**원인 3가지**:
1. **필터키 켜짐** ← 먼저 확인할 것!
2. **드라이버 손상**: 장치 관리자 > 표준 PS/2 키보드 > 제거 > 재부팅
3. **물리적 손상**: 액체 침투, 먼지 축적

**해결 순서**:
1. 필터키 해제
2. 드라이버 재설치
3. 외장 키보드로 테스트 (내장/외장 문제 구분)
4. 서비스센터 방문

### 5. 액체 쏟음 (안전 항목) ⚠️

**즉시 조치 (생명 문제)**:
- 🚨 **즉시 전원 차단**
- 🚨 **배터리 분리** (분리형 배터리)
- 🚨 **절대 켜지 말 것** (합선 → 메인보드 손상)
- 🚨 **드라이기 금지** (열 → 배터리 팽창·폭발 위험)

**올바른 대처**:
- 전원 OFF → 배터리 뺌 → 자연 건조 (2~3일) → 서비스센터 방문
- 건조에 시간 걸리는 이유: 내부 회로에 스며든 액체 완전 제거

**왜 서비스센터**: 일반인이 분해 청소하면 조립 오류로 더 망가질 수 있음

### 6. 키캡 청소

**올바른 방법**:
1. 전원 OFF & 배터리 분리
2. 부드러운 솔/에어 블로워로 먼지 제거
3. 깊은 청소: 70% 이소프로필알코올 + 면봉
4. 액체가 키보드 내부로 들어가지 않도록 주의

**금지 사항**:
- ❌ 드라이기 (열 변형)
- ❌ 물 또는 세제 직접 분사
- ❌ 알코올을 과량 묻혀 흘리기

**유지보수 주기**: 월 1회 솔로 털기면 충분

### 7. 외장 키보드 테스트 (진단 도구)

**용도**: 내장 키보드의 물리적/소프트웨어 문제 구분

| 외장 키보드 상태 | 판단 |
|---|---|
| 정상 | 내장 키보드 물리적 손상 (수리 필요) |
| 같은 증상 | 소프트웨어 문제 (드라이버, IME, 설정) |

**의미**:
- 외장이 정상 → 내부 기계식 스위치 고장이나 커넥터 손상
- 외장도 같음 → Windows 설정, 드라이버, 언어 팩 재설치로 해결 가능

## 조사 출처 정리

**공식 문서** (1차 소스):
- Dell 대한민국: 필터키 증상/해제
- Lenovo: Num Lock, 필터키
- ASUS: 고스트 입력, 키보드 청소
- 삼성전자서비스: Num Lock, 고스트 입력, 외장 키보드 테스트
- LG 스스로 해결: Num Lock, 고스트 입력
- Microsoft Q&A: 고정키, 고스트 입력
- CORSAIR: 청소 방법, 고정키

**사용자 커뮤니티** (참고용):
- 나무위키: 한/영 키 정의
- AJD, Brunch: 한/영 전환 가이드
- WorkingUS, iFixit: 액체 피해 대처

## 글쓰기 전략

### 각 증상별 제시 순서 (추천)

**노트북 키보드 먹통/입력 안 됨** → 먼저 확인할 3가지
1. 필터키 켜져 있나? (설정 > 접근성)
2. Num Lock 켜져 있나? (Fn + 확인)
3. 한/영 전환만 안 되나? (하드웨어 레이아웃 확인)

### 액체 쏟음은 절대 "첫 시도" 단계가 아님
- **경고 박스**로 분리 제시
- "해보고 안 되면 다음" 톤이 아니라 **"절대 하지 말 것" 단정**
- 그 뒤에만 "서비스센터" 안내

### 외장 키보드 테스트는 "원인 판별 도구"로
- 고장 확정 전에 반드시 제시
- "이 방법이 통하면 내부 기계식 문제, 외장도 같으면 소프트웨어"

- **총 팩트**: 13개
- **tier 분포**: 확실 13 / 논쟁 0 / 확인실패 0
- **evidence 분포**: 공식 12 / 후기 0 / 실측 0

- **기각한 팩트**: 0개 (LG/삼성 기존 파일은 터치패드만 다루므로 겹침 없음)
