# 스팀다리미·핸디스티머 자가점검 조사

## 조사 범위

스팀다리미(steam iron) 자가점검 항목:
- 스팀 안 나옴 (물 부족, 예열 부족, 온도 설정, 석회질)
- 물 샘·뚝뚝 떨어짐 (예열 부족, 물 과다, 온도)
- 옷에 물자국·갈색 얼룩
- 열판 변색·하얀 자국
- 예열 불량
- 자동 꺼짐
- 석회질 제거·셀프클린
- 정수 vs 수돗물 논란
- 핸디 vs 스탠드 차이
- 화상 주의

## 주요 발견

### 1. 스팀이 안 나올 때 진단 순서

필립스 공식 가이드에 따르면 우선순위:

1. **물탱크 확인** (가장 흔한 원인)
   - 물이 MIN 표시선 이상으로 있는지 확인
   - 출처: 필립스 기술 지원

2. **예열 완료 대기**
   - 스탠드형: 최대 2분
   - 핸디형: 20초~1분 미만
   - 표시등이 깜박임을 멈추고 켜진 상태 유지까지 대기
   - 출처: 필립스 기술 지원

3. **온도 설정 확인**
   - 점 1개(합성섬유) 모드에서는 스팀 불가
   - 점 2개 또는 3개(리넨) 위치로 설정 필요
   - 출처: 필립스 기술 지원

4. **석회질 제거** (장기 사용 제품)
   - 정기 디스케일링으로 예방 가능
   - 출처: 필립스 기술 지원

### 2. 물이 떨어지거나 옷에 물자국이 생길 때

**주요 원인들:**

1. **예열 부족**
   - 스팀 분사 전 충분한 예열 필수
   - 물이 완전히 가열되지 않으면 물방울 배출
   - 출처: 필립스 기술 지원

2. **과도한 물 주입**
   - 물탱크는 2/3 정도만 채울 것
   - 물을 너무 많이 넣으면 응축수 증가
   - 출처: 필립스 사용 설명서

3. **강력 스팀 버튼 과다 사용**
   - 연속 누르기 피할 것
   - 사용 후 수 초 간격 필요
   - 출처: 필립스 기술 지원

4. **호스 응결** (스탠드형)
   - 첫 사용 또는 장기 미사용 후 호스 내 응축수 배출
   - 사전 스팀 분사로 응축수 제거 후 사용
   - 출처: 사용자 가이드

**옷 자국 제거:** 식초로 제거 가능 (출처: 소비자 가이드)

### 3. 스팀이 물로 떨어지는 현상

**이는 정상 현상이다.**

- 배출 온도가 낮으면 증기가 빠르게 응축되어 물로 떨어짐
- 열역학적으로 정상적인 현상
- 해결: 예열을 충분히 한 후 사용
- 출처: 필립스 기술 지원

### 4. 물의 종류: 정수 vs 수돗물

**권장 순서:**
1. **증류수/탈염수** (가장 좋음 - 미네랄 0)
2. **정수기물** (중간 - 일부 미네랄 제거)
3. **수돗물** (가능하지만 석회질 주의)

**수돗물 사용 시:**
- 경수 지역은 미네랄 축적 빠름
- 경수 지역: 수돗물 50% + 증류수 50% 혼합 권장
- 정기 디스케일링 필수
- 출처: 필립스 기술 지원

**차이점:**
- 가습기와 달리 스팀다리미는 가열로 세균 문제 없음
- 순수 경도(미네랄) 이슈만 해당
- 석회질 축적이 주 문제

### 5. 석회질 제거

**권장 주기:**
- 일반: 1~2개월에 1회
- 경수 지역 또는 매일 사용: 3~4주마다
- 스팀 약화 시 즉시 수행

**세척 방법:**
- **식초:** 물과 식초를 3:1 비율로 끓인 후 20분 대기, 헹굼 반복
- **구연산:** 물에 구연산 2스푼(10~20g) 넣고 5~10분 대기, 헹굼 반복
- 출처: 필립스 공식 문서 + 사용자 가이드

**세척 후 주의:**
- 깨끗한 물로 충분히 헹굴 것
- 식초/구연산 냄새 잔여 제거 필요
- 헹굼 후 물을 여러 번 끓여 배출

### 6. 핸디형 vs 스탠드형 차이

| 항목 | 핸디형 | 스탠드형 |
|---|---|---|
| **예열 시간** | 20초~1분 | 1분 이상 |
| **스팀 성능** | 30g 미만 | 30g 이상 |
| **물탱크** | 소용량(200~400ml) | 대용량(1L 이상) |
| **휴대성** | 우수 | 불가 |
| **사용 용도** | 간편 다림질, 이동 | 대량 다림질 |
| **주름 제거력** | 작은 주름 | 큰 주름 |

**선택 기준:**
- 가족 단위 대량: 스탠드형
- 개별 간편용·이동용: 핸디형

출처: AJD 스팀다리미 추천 가이드, 노써치

### 7. 화상 위험 및 안전

**⚠️ 주의 사항:**

1. **스팀 화상**
   - 100도 이상 고온 증기
   - 옷을 들고 다릴 때 손이 스팀에 노출되면 2도 이상 화상
   - 출처: 필립스·테팔 안전 지침

2. **보호장갑**
   - 보호장갑 착용 권장
   - 없으면 공기 순환 가능한 주방 장갑 사용
   - 출처: 핸디형 사용 가이드

3. **물 샘으로 인한 감전·화재**
   - 물이 새면 감전/화재 위험
   - 즉시 사용 중단 및 서비스센터 점검
   - 출처: 필립스·테팔 사용 설명서

4. **어린이·반려동물**
   - 사용 중 반드시 어른이 감시
   - 사용 후 뜨거운 상태로 방치하지 말 것
