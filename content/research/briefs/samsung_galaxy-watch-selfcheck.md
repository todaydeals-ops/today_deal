# 갤럭시워치 자가점검 — 연결 끊김·충전 불량·배터리 소모·센서 정확도

## 조사 범위

이 브리프는 갤럭시워치 사용 중 발생하는 다음 4가지 상황에서의 자가점검을 다룬다:
1. 폰과의 블루투스 연결 끊김 또는 연결 안 됨
2. 무선 충전 불량
3. 배터리가 비정상적으로 빨리 소모됨
4. 심박·수면 측정이 부정확하거나 센서 인식 불량

## 1. 연결 문제 — 폰과 블루투스 끊김

### 원인 분석
- **블루투스 간섭**: 갤럭시워치와 폰의 거리가 10m를 넘거나 벽·금속 물체가 장애물이 되면 신호가 약해진다.
- **앱 캐시 오류**: Galaxy Wearable 앱의 캐시가 손상되면 연결 정보가 제대로 로드되지 않는다.
- **페어링 정보 손상**: 반복된 연결 끊김이 발생하면 페어링 데이터가 깨져 있을 수 있다.

### 해결 순서 (공식)
1. **기본 확인** (공식):
   - 갤럭시워치 전원 켜짐 확인
   - 스마트폰 블루투스 켜짐 확인
   - 두 기기 거리 10m 이내, 장애물 없음

2. **Galaxy Wearable 앱 재연결** (공식):
   - Galaxy Wearable 앱 실행 → '시작' 선택
   - 워치 기기 선택 → 일치하는 숫자 확인 → '확인' 선택
   - 권한 동의 및 앱 설정 완료

3. **앱 캐시 초기화** (공식 — 삼성 공식 자료에 안내):
   - 스마트폰 설정 → 앱 → Galaxy Wearable → 저장공간 → 캐시 삭제

4. **새 폰에 연결할 때** (공식):
   - 워치의 설정 → 일반 → '새 폰에 워치 연결' 선택
   - 새 폰의 Galaxy Wearable 앱에서 시작 진행

**주의**: 위 방법들이 모두 데이터를 손실하지 않으며, 여러 번 시도해도 안전하다.

## 2. 충전 문제 — 무선 충전 안 됨

### 원인 분석
갤럭시워치는 자기 유도식 무선 충전을 사용한다. 충전이 안 되는 이유는:
- **접촉 불량**: 워치와 충전 패드 사이 정렬이 어긋나거나, 먼지·습기가 개입됨
- **액세서리 간섭**: 워치 보호 커버나 밴드의 두께 때문에 코일 거리가 멀어짐
- **충전 패드 불량**: 정품이 아니거나 어댑터 연결이 빠짐
- **워치 온도 이상**: 내부 온도 센서가 고온을 감지하면 안전상 충전 차단

### 해결 순서 (공식)
1. **물리적 확인** (공식):
   - 워치와 충전 패드 사이에 물, 땀, 먼지 없음 확인
   - 손전등으로 충전 패드 접점 상태 시각적 확인

2. **정렬 조정** (공식):
   - 보호 커버 제거 후 재시도
   - 워치를 천천히 움직여 충전 표시가 나타나는 위치 찾기
   - 한 번 붙으면 고정된 상태 유지

3. **정품 충전기 확인** (공식):
   - 삼성 공식 갤럭시워치 무선 충전 패드 사용
   - 어댑터 연결 상태 재확인 (분리했다가 다시 연결)
   - 다른 USB 포트나 어댑터로 시도

4. **배터리 상태 확인** (공식):
   - 워치가 과열된 경우 실온에서 30분 이상 방치 후 재시도
   - 배터리 충전이 0%에 가까우면 약 5분 후 화면이 켜질 때까지 대기

**주의**: 공식이 아닌 Qi 충전기로는 호환성이 보장되지 않음. 무선 충전 패드 자체가 방수가 아니므로 물에 직접 닿으면 안 됨.

## 3. 배터리 소모 — 비정상적으로 빨리 닳음

### 주요 소비 기능 (공식)

| 기능 | 소모 영향 | 권장 설정 |
|---|---|---|
| **항상 켜기(AOD)** | 매우 높음 | 가능하면 해제. 필요 시 야간만 활성화 |
| **심박 측정** | 중간 | "휴식 중만"으로 설정(권장), "항상 켜기" 피할 것 |
| **수면 추적** | 중간 | 밤에만 활성화하면 배터리 절약 |
| **GPS** | 높음 | 운동 중만 켜기, 평상시 해제 |
| **블루투스** | 낮음~중간 | 항상 켜짐(폰과의 동기화 필요) |
| **WiFi 항상 켜기** | 중간 | 필요 없으면 해제 |
| **워치페이스** | 낮음 | 애니메이션 많은 페이스는 더 많이 소모 |

### 배터리 소모 진단 순서
1. **설정 확인** (공식):
   - 설정 → 디스플레이 → '항상 켜기' 상태 확인
   - 설정 → 건강 → 심박수 → 측정 주기 확인 (최저값 권장)
   - 설정 → 앱 → GPS/WiFi 사용 앱 검토

2. **배터리 사용 현황 확인** (공식):
   - 설정 → 배터리 → 앱별 소비량 확인
   - 평상시 사용하지 않는 앱이 높은 소비를 보이면 비활성화

3. **착용 불량 확인** (공식):
   - 워치가 손목에서 자주 뜬다 → 착용감지가 반복되어 배터리 소모
   - 손목뼈를 피해 팔꿈치 쪽으로 옮겨 단단히 착용

4. **소프트웨어 업데이트** (공식):
   - 설정 → 정보 → 소프트웨어 업데이트 확인
   - Galaxy Wearable 앱에서도 업데이트 확인 (폰 쪽)

**주의**: "항상 켜기"는 편의 기능이지만, 배터리 소모가 2배 이상 증가할 수 있다. 배터리 수명이 다한 경우(구형 워치, 2년 이상 사용)는 교체만이 해결책.

## 4. 센서 정확도 — 심박·수면 측정 부정확

### 정확한 측정을 위한 착용법 (공식)

**착용 위치:**
- 손목뼈 위쪽 3~4cm, 팔꿈치 방향으로 착용
- 손목뼈 관절을 피할 것 (혈관 활동이 약한 구간)

**착용 강도:**
- 워치와 피부가 밀착되어야 함
- 너무 느슨하면 센서가 피부와 떨어져 신호 손실
- 움직임이 적고 일정한 상태에서 측정이 정확함

**측정 환경:**
- 심박수: 운동 직후, 움직임이 큰 상황에서는 부정확할 수 있음
- 수면: 워치는 손목 움직임과 심박 데이터로 수면 단계를 추정하며, 정확도는 "추정"이 아니라 "경향" 수준

### 부정확 원인과 대응
1. **착용이 느슨함**:
   - 매번 착용 후 한 손가락이 들어갈 정도의 간격만 남기기
   - 착용 후 10초 누른 상태 유지하면 신호 안정화

2. **착용 위치 잘못됨**:
   - 손목 정중앙(뼈 부분)에서 팔꿈치 방향으로 2~3cm 올리기
   - 커뮤니티 보고: 올바른 위치에서만 정확도 50% 이상 개선

3. **신체 조건 영향 (불가제어)**:
   - 손목 털이 많으면 신호 약화
   - 타투가 있으면 광학 센서 오류 가능
   - 냉한 환경에서 혈류 감소 시 측정 불가

4. **센서 청소** (공식):
   - 워치 뒷면 센서 부분을 부드러운 천으로 닦기
   - 직접 물에 담그지 말 것 (방수는 되지만 센서 성능 저하 위험)

## 5. 기기 관리 — 재부팅·초기화·소프트웨어 업데이트

### 강제 재부팅 (공식)
응답하지 않거나 화면이 멈췄을 때:
- **방법 1**: 아래쪽 버튼 7초 이상 누르면 자동 재부팅
- **방법 2**: 측면 전원 버튼 + 아래쪽 버튼 동시에 3초 → "재시작" 선택

재부팅 후 데이터 손실 없음.

### 초기화 (공식)
연결 문제, 소프트웨어 오류가 계속될 때:

**정상 부팅 시**:
1. 빠른 설정(상단에서 아래로 스와이프)
2. 톱니 아이콘 → 계정 및 백업 → 초기화 선택
3. 확인 후 진행 (모든 데이터 삭제)

**부팅 안 될 때**:
1. 전원 버튼 + 아래 버튼 동시에 7초 누르기
2. 삼성/갤럭시 로고 나타나면 놓기
3. 화면 지침에 따라 초기화 선택

**주의**: 초기화는 모든 설정, 앱, 데이터를 삭제한다. 꼭 필요할 때만 수행. 초기화 전 배터리가 50% 이상이어야 함.

### 소프트웨어 업데이트 (공식)
워치의 펌웨어와 Galaxy Wearable 앱이 최신 버전이어야 연결, 배터리, 센서 기능이 최적화된다.

**워치 펌웨어 업데이트**:
- 설정 → 정보 → 소프트웨어 업데이트 확인
- WiFi + 충전 중 진행하는 것이 권장됨

**Galaxy Wearable 앱 업데이트**:
- 스마트폰의 Galaxy Store 또는 Google Play에서 업데이트
- 폰 보안 업데이트도 함께 진행할 것

## 6. 자가진단 — Galaxy Wearable 앱

갤럭시워치 일부 모델(워치 4, 워치 5 이상)은 Samsung Members 앱을 통해 자가진단을 실행할 수 있다.

**진행 방법**:
1. Samsung Members 앱 실행
2. 왼쪽 메뉴 → 진단 선택
3. 갤럭시워치 선택 → 블루투스, 배터리, 센서 등 진단 항목 선택
4. 결과 확인 (정상/이상 표시)

**주의**: 모든 갤럭시워치 모델이 자가진단을 지원하는 것은 아니다. 최신 모델일수록 지원 가능.

## 배터리 팽창 — 즉시 중단

만약 워치 배터리가 부풀어 오르거나 과열되거나 타는 냄새가 난다면, 즉시 사용과 충전을 중단하고 공식 서비스센터로 문의해야 한다. 이는 일반 자가점검이 아닌 안전 응급 상황이다.

## 정리 — 우선순위 순서

| 증상 | 첫 번째 시도 | 다음 단계 | 최후 수단 |
|---|---|---|---|
| **연결 안 됨** | Galaxy Wearable 앱 재실행 | 앱 캐시 삭제 → 재연결 | 초기화 후 재연결 |
| **충전 안 됨** | 정렬 조정 + 먼지 제거 | 커버 제거 + 어댑터 재확인 | 서비스센터(충전기 지참) |
| **배터리 빨리 소모** | 항상 켜기 끄기 + 심박 주기 조정 | 배터리 사용 앱 검토 + 착용 확인 | 소프트웨어 업데이트 |
| **센서 부정확** | 착용 위치·강도 확인 | 센서 청소 | 센서 재정렬이 필요하면 서비스센터 |
