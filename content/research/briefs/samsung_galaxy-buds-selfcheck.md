# 갤럭시 버즈 자가점검 조사 전문

**주제:** 갤럭시 버즈 한쪽 소리 안 들림·연결 문제·초기화 자가점검  
**조사일:** 2026-07-21  
**기준:** 삼성전자서비스 공식 문서(1차 소스)·사용자 커뮤니티 후기

---

## 1. 한쪽 소리 안 들릴 때

### 1.1 기본 확인사항 (공식)
삼성전자서비스 공식 문서(solution/164819)에서 안내하는 순서:
- **이어팁 미착용 또는 불안정한 착용** — 귀에 안정적으로 삽입했는지 확인
- **이물질 확인** — 귀지, 먼지가 스피커 부분에 있는지 시각 점검
- **좌우 이어팁 크기 불일치** — 같은 사이즈로 맞춰야 음량이 균등함

### 1.2 스피커·메시 그릴 청소 (공식)
삼성 공식 청소 가이드(solution/4216322, 4634544):
1. 부드러운 솔을 클렌징 워터에 적심
2. 마이크 홀·스피커·공기 덕트 각각 약 15초씩 닦음
3. 종이 타월로 수분 제거
4. 증류수로 다시 헹굼
5. 완전히 건조(24시간 이상 권장)

**주의:** 알코올, 아세톤 같은 화학용제 금지. 뾰족한 물체로 무리하게 긁으면 센서·스피커 손상 위험

### 1.3 음량 균형 조정 (공식)
설정 > 접근성 > 청각 보조 > 연결된 오디오에서 좌우 음량 바로 조정 가능
- **제약:** 모든 모델에서 지원되지 않음 (예: 버즈 SM-R170/R175 미지원)
- **한계:** 통화 중에는 작동하지 않음

## 2. 페어링·연결 문제

### 2.1 한쪽만 연결되지 않을 때 (공식)
**가장 효과적인 해결책:** 기존 페어링 제거 후 양쪽을 모두 다시 페어링  
(solution/337194에서 명시)

### 2.2 수동 페어링 방법 (공식, 모델별)
삼성서비스 solution/337194에서 제시:
- **버즈 4/4 프로:** 케이스 뒤 연결 버튼 3초 이상
- **버즈 3/3 프로:** 케이스 하단 버튼 3초 이상
- **버즈 2/2 프로:** 양쪽 이어버드 동시에 3초 이상
- **구형 모델:** 양쪽 터치패드 동시에 3초 이상

**타임아웃:** 3분 내에 완료되지 않으면 자동 취소. 케이스를 닫았다가 다시 열어 재시도

### 2.3 페어링 팝업 안 나타날 때 (공식)
- 위치 권한 비활성화가 원인일 수 있음
- 갤럭시 웨어러블 앱에서 위치 정보, 근처 기기 검색, 알림 권한을 **모두** 허용해야 함
- 이전에 다른 기기와 연결된 상태면 그 기기와의 연결을 먼저 끊을 것

## 3. 초기화 방법

### 3.1 앱을 통한 초기화 (권장, 공식)
갤럭시 웨어러블 앱 > 이어버드 설정 > 초기화 > 화면 안내 따라 진행  
(solution/25700 참조)

### 3.2 터치패드를 통한 초기화 (공식, 앱 연결 안 될 때)
1. 충전 케이스에 양쪽 이어버드 삽입
2. 양쪽 터치패드를 **동시에 7초 이상** 누르기
3. 배터리 표시등 녹색 깜빡임 → 꺼짐 = 완료

**모델별 차이:** 정확한 터치패드 위치와 누르는 시간은 모델에 따라 다를 수 있으므로 매뉴얼 확인 필요

**주의:** 초기화 후에는 재페어링이 필수

---

## 4. 충전 접점 청소

### 4.1 시각 확인 (공식)
손전등으로 충전 포트 내부를 비춰 금속 단자 부분의 먼지·이물질 확인

### 4.2 공기 제거 (공식)
포트 안으로 바람을 불어 먼지와 이물질 제거

### 4.3 상세 청소 (공식, solution/4216322)
1. 면봉을 증류수에 적심
2. 충전 케이스 내부 충전 단자 부분을 부드럽게 닦음
3. 마른 면봉으로 모든 수분 제거
4. 완전히 건조시킨 후 충전 시도

**중요:** 알코올 금지, 습기 완전 제거 필수

### 4.4 이어버드 스템 단자 청소 (공식, solution/4216322)
이어버드 스템의 충전 단자에도 땀·습기가 묻을 수 있으므로 깨끗한 천으로 닦음

---

## 5. 배터리 소모 편차

### 5.1 정상 범위 (사용자 경험 + 기술)
좌우 배터리 용량 차이가 **15% 이내**는 정상. 원인:
- 충전 회로 완벽 동기화 불가능
- 마스터-슬레이브 구조 (한쪽이 주가 되어 작동)

### 5.2 사용 습관에 따른 편차 (후기)
- 통화할 때 늘 같은 쪽만 사용 → 그쪽 배터리 더 빨리 노화
- 특정 쪽에서만 음악·노이즈 캔슬링 → 에너지 소모 편차 심화
- **해결:** 좌우를 번갈아 사용하고 충전 접점을 깨끗이 유지

### 5.3 15% 초과 편차 또는 급격한 방전 (기술)
배터리 성능 저하 또는 충전 불량 문제일 수 있음. 서비스센터 점검 필요

## 6. 자동 전환(오토 스위칭)

### 6.1 기능 활성화 (공식, solution/337190)
**요구사항:**
- One UI 3.1 이상
- 같은 삼성 계정으로 로그인한 2개 이상 기기

**활성화 방법:**
1. 스마트폰 설정 > 연결 > 블루투스 > 연결된 버즈 상세 설정
2. '다른 기기로 자동 전환' 토글 켜기
3. 갤럭시 웨어러블 앱 > 이어버드 설정 > '원활한 이어버드 전환' 활성화

### 6.2 작동 원리 (공식)
예: 스마트폰에서 음악 재생 중 → 타블렛에서 유튜브 시작 → 오디오가 자동으로 타블렛으로 전환

### 6.3 알려진 문제점 (후기, 커뮤니티)
- 갤럭시 북과의 자동 전환이 작동하지 않는 사용자 다수 보고
- 펌웨어 업데이트 후에도 해결되지 않는 사례
- **원인:** 미해결. 서비스센터 점검 권장

**문의:** 고객센터 1588-3366

## 7. 갤럭시 웨어러블 앱 진단 기능

### 7.1 자가 진단 지원 (공식, 커뮤니티)
- **지원 모델:** 갤럭시 버즈 2 프로 (현재)
- **접근:** Samsung Members 앱 > "도움받기" > "연결된 기기 진단" > 진단 기기 선택

### 7.2 요구사항
- 갤럭시 버즈·워치 + Samsung Members 앱 + Galaxy Wearable 앱 모두 **최신 버전** 필수

### 7.3 제약
모든 갤럭시 버즈 모델에서 지원되지 않음. 최신 모델도 시간 경과 후 지원될 가능성

## 8. 종합 조치 순서

갤럭시 버즈 문제 해결 시 권장 순서:

1. **청소 (손해 없음)**
   - 스피커 메시 그릴: 부드러운 솔 + 클렌징 워터
   - 충전 접점: 바람 → 증류수 면봉 → 건조

2. **소프트 초기화**
   - 충전 케이스 닫았다 열기
   - 터치패드 3초 누르기

3. **재페어링**
   - 기존 페어링 제거
   - 양쪽 모두 다시 페어링

4. **설정 점검**
   - 음량 균형 조정
   - 위치 권한 확인
   - 자동 전환 토글 확인

5. **서비스센터**
   - 위의 모든 방법 후에도 해결 안 될 때
   - 배터리 급격한 편차
   - 자동 전환 미작동 (갤럭시 북 연결)
