---

## 조사 개요

경동나비엔 가스보일러(콘덴싱)의 에러코드와 온수·난방 문제별 자가진단 절차를 조사했습니다.

### 정보 출처의 특성

- **공식 문서**: 경동나비엔 공식 홈페이지의 FAQ, 제품 매뉴얼 PDF는 현재 404 오류로 접근 불가
- **주요 정보원**: 사용자 커뮤니티(클리앙, 당근), 블로그, YouTube 자가진단 영상, AS 정보 공유 사이트
- **신뢰도**: 다수 사용자가 보고한 방법으로 재현성이 있으나, 원본 공식 문서 확인 불가

## 주요 에러코드

### E01 / E001 (열교환기 과열)

**의미**: 보일러 열교환기가 과열되거나 비등 현상이 발생한 상태

**원인**:
- 난방 회로의 물 순환 부족
- 각방 분배기가 닫혀 있어 순환이 원활하지 않음

**자가점검**:
1. 보일러 하단의 각방 분배기 밸브가 완전히 열려 있는지 확인
2. 밸브가 열려 있는데도 에러가 계속되면 내부 센서 이상 가능성 → 전문가 점검 필요

**한계**: 공식 문서 미확인. 사용자 보고 기반.

---

### E02 / E302 (저수위 / 물 부족)

**의미**: 난방 회로 내 물이 부족한 상태

**원인**:
- 수도 단수 또는 차단
- 보일러 냉수 공급 밸브가 닫혀 있음
- 배관 누수로 인한 물 손실

**자가점검**:
1. 집으로 들어오는 수도 차단기가 열려 있는지 확인
2. 보일러에 연결된 냉수 공급 밸브(직수 밸브)가 완전히 열려 있는지 확인
3. 밸브를 열어도 물이 안 나오면 수도 단수 여부 확인

**수동 급수 모델**: 
- 손잡이를 좌우로 움직여 물을 채울 수 있는 모델도 있음
- 설명서 확인 필요

---

### E03 / E003 (불착화 / 점화 불량)

**의미**: 보일러의 점화(착화) 실패. 가스가 연소되지 않음

**원인**:
- 가스 공급 중단 또는 부족
- 가스 밸브가 닫혀 있음
- 배기 이상 (응축수 결빙 등)

**자가점검**:
1. **가스 공급 확인**: 다른 가전(가스렌지 등)에서 가스가 나오는지 확인
2. **가스 밸브**: 보일러와 연결된 가스 중간 밸브가 완전히 열려 있는지 확인
3. **겨울철 배기 결빙**: 응축수 배출관이 얼어 있을 수 있음
   - 배기 호스 외부 온도가 0℃ 이하인지 확인
   - 따뜻한 물로 해동 시도 (50℃ 미만 상온수)
4. **전원 리셋**: 보일러 전원을 껐다가 5분 후 다시 켜보기

**⚠️ 안전 주의**:
- 가스 냄새가 나면 **즉시 사용 중단**
- 창문을 열어 환기하고 가스공급사(도시가스 121 또는 LPG 업체)에 신고
- 전기 장비 스위치를 만지지 말 것 (스파크 위험)

---

### E04 / E004 (거짓 화염 / 센서 오류)

**의미**: 불꽃이 감지되었으나 실제 점화는 완료되지 않은 상태

**원인**:
- 불꽃 센서 오염 또는 오류
- 내부 점화 시스템 이상

**자가점검**:
1. 전원을 껐다 켜기 (소프트 리셋)
2. 반복되면 전문가 진단 필요

**한계**: 센서 관련 에러로 대부분 전문가 개입 필요. 자가 해결법 제한적.

---

### E10 / E010 (공기압 이상)

**의미**: 보일러 내부 공기압 또는 순환계 압력 이상

**원인**:
- **겨울철**: 응축수 배출 호스가 결빙되어 역압 발생
- 순환 펌프 이상
- 내부 센서 오류

**자가점검**:
1. **겨울철 응축수 결빙 확인**:
   - 보일러 후면의 응축수 배출 호스(보통 회색/흰색 사각형 파이프)를 만져봄
   - 딱딱한 얼음으로 느껴지면 결빙
   - 따뜻한 물(50℃ 미만)을 부어 해동 시도
   
2. **전원 리셋**:
   - 보일러 전원을 뺐다가 15초 후 다시 꽂기
   - 일부 사용자 보고: 이 방법으로 작동 복구된 경우 있음

3. 반복되면 펌프 이상 가능 → 전문가 점검

---

### E16 (바이메탈 과열)

**의미**: 보일러의 과열 방지 장치(바이메탈)가 트립되어 자동 차단된 상태

**원인**:
- E01과 유사: 난방수 순환 부족
- 내부 온도 감지기 이상

**자가점검**:
1. 각방 분배기 밸브가 열려 있는지 확인
2. 밸브가 열려 있는데도 반복되면 엔지니어 점검 필요

## 온수가 안 나올 때

### 점검 순서

1. **전원 확인**
   - 보일러 전원 플러그가 연결되어 있는가
   - 전원 스위치가 켜져 있는가

2. **수도 공급 확인**
   - 집으로 들어오는 메인 수도 밸브가 열려 있는가
   - 겨울철에 동파 방지용으로 의도적으로 닫아두는 경우 확인

3. **보일러 냉수 직수 밸브 확인**
   - 보일러 하단에 있는 파란색/검은색 손잡이 밸브 (직수 밸브)
   - 완전히 열려 있는지 확인
   - 반시계 방향으로 끝까지 돌려 개방

4. **보일러 가동 모드 확인**
   - 온수 공급 모드로 설정되어 있는가
   - 난방 모드로만 설정되면 온수가 안 나옴

5. **겨울철 동파 확인**
   - 직수 배관 외부를 만져봄 (보일러 입수관이 차갑거나 얼음이 있는가)
   - 동파된 경우: 따뜻한 타월로 감싸거나 미온수로 녹임

6. **가스 공급 확인**
   - 다른 가전에서 가스가 나오는지 확인
   - 가스 밸브가 열려 있는지 확인
   - 온수 기능은 가스로 물을 데우므로, 가스가 없으면 온수가 안 나옴

7. **수위 에러 확인**
   - 보일러 디스플레이에 E02 또는 저수위 표시가 있는지 확인

### 해결되지 않으면

대부분의 경우 위 점검만으로 해결됩니다. 그래도 안 되면 고객센터 연락.

---

## 난방이 안 될 때

### 점검 순서

1. **난방 모드 확인**
   - 보일러 또는 실내온도조절기가 난방 모드로 설정되어 있는가
   - 설정 온도가 현재 실내 온도보다 높은가

2. **각방 분배기 밸브 확인**
   - 보일러 하단에 있는 배관 중 각 방으로 가는 밸브들
   - 난방이 필요한 방의 밸브가 완전히 열려 있는가
   - 일부 방만 안 데워지면 그 방의 밸브만 닫혀 있을 가능성

3. **난방 회로 프라이밍**
   - 여름철 오래 사용하지 않은 후 겨울철에 처음 켤 때
   - 배관 내 공기 제거 필요
   - 보일러의 공기 빼기 밸브(airway purge valve)를 천천히 열어 물이 나올 때까지 흘려보냄
   - 그 후 밸브를 다시 닫기

4. **보충수/압력 확인** (별도 섹션 참조)

5. **배관 막힘**
   - 장기간 난방을 사용하지 않았을 때 스케일이나 녹슬음으로 막힐 수 있음
   - 특히 구형 시스템에서 발생 가능
   - 자가 해결 어려움 → 전문가 점검

6. **실내온도조절기 배터리**
   - 무선 온도조절기를 사용하는 경우 배터리 확인
   - 배터리 부족하면 신호가 안 갈 수 있음

## 보충수 및 압력 관리

### 압력계 읽기

- 보일러 전면에 압력계(game gauge, 바늘 게이지)가 있음
- 정상 범위: **1.0 ~ 1.5 bar** (경우에 따라 0.8 ~ 2.0 bar)
- 압력이 너무 낮거나 높으면 문제 발생

### 압력이 낮을 때 (0.5 bar 이하)

**자가 보충수**:
1. 보일러 하단의 **보충수 밸브** 찾기 (파란색 또는 검은색 손잡이)
2. 밸브를 천천히 반시계 방향으로 돌려 열기
3. 물이 흐르면서 압력계의 바늘이 올라가는 소리가 들림
4. 압력계가 **1.2 ~ 1.3 bar** 정도가 되면 밸브를 다시 시계 방향으로 돌려 잠금
5. 밸브를 꽉 조이지 말 것 (수도꼭지처럼 부드럽게)

**주의**:
- 물이 나오지 않으면 수도가 차단되었을 가능성 (메인 수도 밸브 확인)
- 보충수를 여러 번 해도 압력이 올라가지 않으면 누수 가능 → 전문가 점검

### 압력이 높을 때 (2.0 bar 이상)

1. 보일러 상단의 **공기 빼기 밸브**(airway purge valve, 보통 빨간색)를 찾기
2. 작은 컵을 아래에 놓고 나선형 돌기를 반시계 방향으로 천천히 돌리기
3. 물이 나올 때까지 돌렸다가, 물이 나오기 시작하면 멈추기
4. 압력이 1.2 ~ 1.3 bar로 내려가면 밸브를 다시 시계 방향으로 닫기

**주의**:
- 공기 빼기 밸브에서 물이 계속 새어 나오면 밸브 교체 필요

## 겨울철 동파 방지 (외출 시)

### 외출 전 설정

1. **보일러 전원은 켜두기**
   - 계속 저온 운전 모드에서 물을 데우면서 동파 방지
   - 또는 "겨울 외출" 모드가 있으면 설정

2. **실내온도조절기 온도**
   - 최소 5 ~ 10℃ 정도로 설정해두기
   - 또는 제조사 지정 "동파 방지 온도"(보통 5℃) 설정

3. **수도 밸브는 열어두기**
   - 절대 닫으면 안 됨 (물이 고이면서 얼 수 있음)

4. **배관 노출 부분 보온**
   - 실외 배관, 베란다 배관 등은 보온재로 감싸기

### 동파된 경우

- 따뜻한 물(50℃ 미만 상온수)을 배관에 부어 해동
- 가열 패드로 감싸기
- 급할 때: 드라이어 따뜻한 바람으로 배관 녹이기
- 배기 호스(응축수)가 얼었으면 해동 후 배출 확인

## 안전 주의사항

### ⚠️ 가스 냄새

**즉시 조치**:
1. **보일러 사용 중단** (차단기 또는 전원 OFF)
2. **환기 실시** (창문, 환기구 열기)
3. **가스 공급사 신고**:
   - 도시가스: 121 (전국 통일)
   - LPG: 해당 충전소 또는 1544-9119
4. **전기 스위치 조작 금지** (스파크로 인한 화재 위험)

**원인**:
- 배관 누수
- 점화 실패 후 미점화 가스 축적
- 불완전 연소

### ⚠️ 배기 이상 / 일산화탄소

**위험 신호**:
- 배기 배출구에서 물이 흘러내림 (배출구 얼음, 결빙)
- 배기 배출구 주변에 검은 매연이나 그을음
- 실내에서 특이한 냄새 (매운 냄새, 타는 냄새)
- CO 경보기가 울림

**조치**:
1. **환기** (창문 전부 개방)
2. **보일러 중단**
3. **배출구 점검** (얼음, 막힘, 새 둥지 등)
4. **전문가 점검 신청**

**예방**:
- 배기 배출구가 외부에 완전히 노출되어 있는지 확인
- 배출구 주변 30cm 이내에 물건 없게 하기
- 보일러실은 24시간 환기 유지
- CO 경보기 설치 (2020년 8월 이후 신상품에 포함)

### ⚠️ 겨울철 배출구 얼음

**E10 에러의 주요 원인**:
- 응축수 배출관(보일러 후면 흰색/회색 사각파이프)이 얼음
- 콘덴싱 보일러는 폐열 회수로 인해 배출가스 온도가 낮아 응축수가 생김
- 영하에서는 이 물이 얼어 역압 발생

**대처**:
- 배출관 외부 온도 확인 (손으로 만져보기)
- 얼음이 있으면 따뜻한 물(50℃ 미만)로 해동
- 배출구 주변을 보온재로 감싸서 미리 예방
