## 1. 에러코드 및 에러 표시

### 삼성 인덕션

#### dE 에러 (데모 모드)
- **원인:** 청소 중 버튼 누름, 외부 물체 접촉, 일시적 오작동
- **해결:** 
  - 기본: 전원 차단 후 재연결, 또는 전용 차단기 OFF→ON
  - 특정 모델(NZ63A8708X, NZ63T8708XT, NZ63T8708XS): 시계 버튼+MAX 버튼 동시 3초(비프) → 시계 버튼 3초(비프) → MAX 버튼 3초(비프) → 전원 OFF/ON
- **출처:** [삼성전자서비스 solution/1258925](https://www.samsungsvc.co.kr/solution/1258925)

#### L 표시 (차일드락)
- **의미:** 조작부 잠금 상태, 버튼 작동 불가
- **해결:** 자물쇠 모양 버튼을 3초간 길게 누르면 해제
- **출처:** [삼성전자서비스 solution/1024670](https://www.samsungsvc.co.kr/solution/1024670)

#### 기타 에러·이상 표시
- **원인:** 조작부 오작동 또는 센서 이상
- **해결:** 
  1. 전원 플러그 제거 → 2~3분 대기
  2. 다시 연결하여 정상 작동 확인
  3. 증상 지속 시 서비스센터 문의
- **출처:** [삼성전자서비스 solution/1025772](https://www.samsungsvc.co.kr/solution/1025772)

공식 웹사이트에서 인덕션 전용 에러코드 가이드를 찾지 못했습니다. LG 고객센터(1544-7777)로 문의 권장.

## 2. 가열이 약할 때 확인 순서

### LG 공식 진단 절차
**우선순위:** 용기 문제(90%) → 버너 위치 → 설정값

#### 1) 용기 재질 확인
- **사용 가능:** IH 마크 있음, 철제, 법랑, 스테인리스(니켈 함량 주의)
- **사용 불가:** 알루미늄 양은, 유리, 도자기, 뚝배기, 구리
- **확인 방법:** 냉장고 자석을 냄비 **바닥 중앙**에 붙여보기 (자석이 달라붙으면 사용 가능)
- **IH 마크:** 코일 모양 또는 "IH" 문자 표시

#### 2) 용기 크기 확인
- **기준:** 버너 크기의 90% 이상
- **권장:** 지름 19~22cm
- **문제:** 버너보다 작은 냄비 사용 시 노출된 가열면에서 열이 빠져나가 화력 감소

#### 3) 용기 가열지수 확인
LG 인덕션에서 제공하는 가열지수(0~10):
- **0~4:** 사용 비권장
- **5~8:** 저효율 (가열 느림)
- **9~10:** 고효율 (권장)

**확인 방법:**
1. 냄비에 물 또는 음식물을 담기
2. 불판에 올리고 열 단계를 9로 설정
3. 가열 시작 후 Lock + Burner Select 버튼 동시 3초 누르기
4. 타이머 윈도우에 지수 표시 (2~3초)

#### 4) 용기 위치 및 상태 확인
- **위치:** 버너의 십자 표시(중심)에 용기 중심을 정렬
- **바닥 상태:** 평평한가? 다리나 요철이 없는가?
- **문제 금지:** 내용물 없이 공냄비 가열하지 말 것

## 3. 팬 소음

### 정상 소음인가?
**Yes.** 삼성 공식 안내: "인덕션은 조리 시 열이 발생하므로 팬 소리가 나는 것은 정상"

### 냉각팬 작동 원리
- **목적:** 제품 내부 온도 제어
- **작동 온도:** 35~82°C 범위
- **소음 종류:** 붕~, 웅~, 쉿쉣, 윙윙 등
- **자동 멈춤:** 내부 온도가 작동 기준 이하로 떨어지면 정지

### 소음 증가 원인 (정상 범위 내)
1. **냄비 두께:** 너무 얇거나 가벼우면 진동 증가 → 소음 증폭
2. **냄비 바닥 상태:** 요철이 있거나 비뚤어진 냄비
3. **냄비 크기:** 버너보다 작은 냄비 사용
4. **냄비 재질:** 합금 소재나 니켈 함량 높은 스테인리스
5. **냄비 내용물:** 조리량 많으면 더 강한 가열 필요 → 팬 강하게 작동

**권장:** 평평한 바닥의 중두께 냄비 사용으로 소음 감소

## 4. 상판 청소 및 유지보수

### 상판이 뜨거운가?
**인덕션 vs 하이라이트 차이:**
- **인덕션:** 냄비 바닥만 가열 → 상판은 미지근함 → 조리 직후 청소 가능
- **하이라이트:** 상판 자체가 발열 → 식을 때까지 기다려야 함

### 청소 절차

#### 안전 확인
- 전원 완전 차단 또는 해당 버너만 OFF
- H(가열) 표시가 사라질 때까지 대기 (인덕션은 빠름)

#### 청소 방법 (삼성·LG 공통)
1. **세제 선택:** 중성 주방용 세제 또는 세라믹 글래스 전용 세정제
2. **부드러운 도구만:** 소프트 스펀지, 극세사 천 (금속 수세미·광택제 금지)
3. **담금:** 세제를 5~10분간 불려 오염을 부드럽게 함
4. **스크래핑:** 플라스틱 스크래퍼를 **30~45도 각도**로 일정 방향만 긁기 (왕복 금지)
5. **마무리:** 종이타올이나 행주로 깨끗이 닦기

### 눌어붙음 예방
- **조리 직후 즉시 닦기:** 국물이나 소스가 굳기 전 제거
- **주 1회 전체 청소:** 누적된 오염 제거
- **매 조리 후 간단 청소:** 더운 국자로 빠르게 닦기

## 7. 과열 보호 (자동 전원 차단)

### 정상 작동
- 제품 내부 온도가 설정 기준을 넘으면 자동 차단
- 냉각팬 작동하며 온도 내림
- 온도 정상화되면 다시 사용 가능

### 자주 발생하는 경우
- **냉각팬 통풍구 막힘:** 먼지/유증기 청소
- **공냄비 가열:** 내용물 없이 오래 사용
- **환기 부족:** 주방 환기 개선
- 이런 경우 지속되면 서비스센터 점검 권장
