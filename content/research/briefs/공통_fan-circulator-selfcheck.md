# 스탠드형 선풍기·서큘레이터 자가점검 조사 전문

## 조사 개요

스탠드형 선풍기와 서큘레이터의 자가점검 및 자가수리에 관한 조사. 전원이 안 켜짐, 날개 회전 불량, 소음, 모터 과열·화재 위험 등의 증상과 해결 방법을 중점 조사. 안전 경고(화재, 손상 위험) 및 분해 청소 방법을 포함.

---

## 0단계: 기존 팩트 검토

### 휴대용 선풍기 트렌드(공통_portable-fan-trend.json)
- **재사용 불가**: 휴대용 선풍기는 배터리 탑재, 스탠드형은 AC 전원 사용 → 배터리 관련 팩트 모두 declined
- 모터 타입: 휴대용(DC/BLDC) vs 스탠드형(주로 AC 모터) → 비교 대상 계열 상이
- 결론: declined_facts에 2건 기록

### 공기청정기 자가점검(공통_air-purifier-selfcheck.json)
- **재사용 불가**: 공기청정기는 필터·센서 중심의 점검, 스탠드 선풍기는 모터·회전축 중심
- 구조 차이로 인해 직접 인용 불가
- 결론: 재사용하지 않음

## 1단계: 웹 검색 결과 정리 (6회 이내)

### 검색 1: 삼성 스탠드 선풍기 자가점검
**출처**: 삼성전자서비스 공식
- **리모컨 배터리**: CR 2032, 양극(+) 위로, 동전으로 열고 닫기
- **버튼 작동 불량**: 푸시 버튼(이물질 청소) vs 터치 버튼(손가락 앞부분만)
- **리셋 방법**: 전원 코드 1분 뽑았다 다시 연결
- **주의**: 리모컨 LED가 보이는지 스마트폰 카메라로 확인(수신부 이상 판별)

### 검색 2: LG 선풍기 자가점검
**출처**: LG 고객지원(에어컨 중심)
- LG는 선풍기 전용 문서 미확인
- 에어컨 전원 문제 해결과 원리 동일(차단기, 배터리, 버튼 확인)
- **결론**: 공식 문서 부재, 삼성 정보로 통합

### 검색 3-4: 선풍기 날개 회전 불량 & 타는 냄새 화재 위험
**출처**: 소비자 후기 + 소방청 안내
- **회전 불량 원인**: 윤활유 부족(웅 소리 신호), 먼지 축적
- **해결법**: WD-40 윤활제(모터 축만), 그리스(베어링 부분)
- **화재 위험**: 타는 냄새 = 즉시 전원 코드 뽑기
- **과열 원인**: 윤활유 산화 또는 먼지로 인한 열 방출 차단
- **예방**: 타이머 활용(3시간 후 20~30분 쉬기)

### 검색 5: DC 모터 vs AC 모터
**출처**: 선풍기 구매가이드 + 제품 비교 분석
- **AC 모터**: 약 50W, 강한 바람, 저가, 소음 있음(스탠드형 표준)
- **BLDC (DC) 모터**: 약 30W, 소음 적음, 내구성 우수, 고가, 약한 바람
- **주의**: BLDC는 초미풍일 때만 조용, 중풍 이상에서는 오히려 시끄러울 수 있음

### 검색 6: 스윙(좌우 회전) 작동 불량
**출처**: 커뮤니티 + 자가수리 가이드
- **정상 작동**: 회전 버튼 누르면 회전 시작, 다시 누르면 해당 각도에서 고정
- **불량 원인**: 베어링 윤활제 고착, 버튼 내부 연결 불량, 콘덴서 고장
- **해결법**: 전원 뺀 상태에서 버튼 반복 조작, 윤활제 도포
- **주의**: 손으로 억지로 돌리면 내부 기어 손상

## 2단계: 안전 경고 사항 별도 점검

### 화재 위험 (tier=확실, evidence=공식)
1. **타는 냄새 = 즉시 전원 차단**
   - 모터 윤활유 산화로 인한 과열
   - 먼지로 인한 열 방출 차단
   - 전류 쇼트 → 불꽃 → 기름때 도화선
   - 조치: 즉시 콘센트에서 전원 코드 뽑기

2. **화재 예방**
   - 타이머 활용: 3시간 가동 후 20~30분 OFF
   - 모터 커버 주변 먼지 정기 청소
   - 24시간 연속 가동 금지

### 손상 위험 (tier=확실, evidence=공식)
1. **날개 청소 시 전원 코드 필수 분리**
   - 콘센트에 꽂혀 있으면 갑자기 작동하여 손가락 끼임
   - 심각한 상해 가능
   - 자동 시작 기능 있을 수 있으므로 가족에게도 알림

2. **분해 시 주의**
   - 모터 축 억지로 회전시키면 내부 기어 손상
   - 전자석 코일에 기름 절대 금지
   - 확신 없으면 전문 수리소 이용

## 3단계: 계층 구분 (scope 정밀화)

### 대상 범위 상이 (declined_facts 처리)
- **휴대용 선풍기 배터리 팩트**: 스탠드형은 배터리 미탑재 → scope 불일치
- **휴대용 BLDC/DC 비교**: 스탠드형은 AC 모터 표준 → 비교 대상 모터 계열 다름

### 제품 라인별 차이
- 터치 버튼 vs 푸시 버튼: 모델별 상이 → caution에 기록
- 고정 각도 범위: 서큘레이터마다 상이 → caution에 기록
- 콘덴서 고장: BLDC 모터 제품에서 더 흔한 경향
