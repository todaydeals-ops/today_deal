# LG 코드제로 무선청소기 흡입력 저하·작동 불량 자가점검 — 조사 브리프

**주제:** LG 코드제로(A9, A9S 등) 흡입력 약할 때·작동 불량 자가진단·배터리 관리·필터 청소·올인원타워 운영
**조사일:** 2026-07-20
**브랜드:** LG
**카테고리:** 가전 / 무선청소기

---

## 조사 개요

LG 코드제로는 2017년 출시 무선청소기로, A9S(2020년~), A9, A9 air 등 다양한 라인이 있다. 올인원타워 충전대와 자동 먼지 비움 기능은 동급 무선청소기 중 특이 사항으로, 필터·배터리·먼지비움 시스템 운영이 별도 조사 대상이다.

다이슨 무선청소기 팩트(dyson_vacuum-selfcheck.json)와의 재사용성 검토 결과, 필터 미장착 시 모터 손상이라는 **공통 물리 원리** 1건은 재사용 가능했으나, 신호 방식(진동·표시등·알림), 필터 청소 절차, 배터리 신호 표시 등은 모두 브랜드별로 다르므로 거부했다.

## 주요 조사 결과

### 1. 모델 구분 (scope 근거)
- **모델명 규칙:** AU/AO(오브제), AT/AS(일반) = A9S / 기타 = A9 air
- **연식 구분:** 24년형까지 "A9S"·"A9 air" 명칭 / 25년형부터 숫자 체계로 변경
- **출시 역사:** A9S는 2020년 출시, 구형 A9 대비 130g 경량화 + 배터리 140AW→200AW 상향
- **기능 차이:** A9S는 ThinQ 앱 연동(자가진단·청소 통계), 먼지통 레버 추가

### 2. 흡입력 저하 — 원인별 점검 순서
공식 문서에서 제시한 흡입력 저하의 주요 원인과 점검 방법:

1. **배터리 부족** → LED 표시창 1칸만 점등 확인 / 충전 필요
2. **이물질 막힘** → "우-웅 우-웅" 소리 + 표시등 "이물 막힘 알림" / 배터리 분리 후 흡입구 버튼으로 분리·청소
3. **먼지통 가득참** → 흡입력 급감 / 큰 먼지 분리 장치를 월 1회 이상 물 세척
4. **필터 오염** → 배기 필터(HEPA)·프리필터 먼지 / 월 1회 이상 물 세척 권장
5. **부품 미장착** → 흡입구, 배기 필터 미끼우기 / 배기 필터는 본체 흰색 원 기준 시계 방향 회전체결
6. **배터리 수명 다 함** → 충분히 충전해도 작동시간 지나치게 짧음 / 배터리 교체 필요

**주의:** 먼지통 내부와 회전솔은 물로 세척하면 안 됨 (손상 위험)

### 3. 필터 종류별 청소 (중요 구분점)
LG 코드제로는 3가지 필터 구성:

| 필터 | 역할 | 청소 방법 |
|------|------|----------|
| **큰 먼지 분리 장치** | 1차 필터링(대형 입자) | 청소솔로 표면 먼지 제거 후 물세척 |
| **프리필터(마이크로)** | 중형 입자 필터링 | 물세척 가능, 완전 건조 필수 |
| **배기필터(HEPA)** | 미세 입자(0.3㎛) 99.97% 포착 | 물세척 가능(일부 모델 제외), 조립 시 내부 실러 접김 확인 필수 |

**핵심 주의:** 프리필터를 제거하고 사용하면 모터와 배기필터가 손상된다(다이슨과 동일한 공통 원리이나, LG는 이를 "구조적 필수"로 명시).

### 4. 필터 건조 시간 및 조건
- **필터 세척 후 건조 시간:** 통풍이 잘 되는 **그늘에서 최소 24시간 이상 완전히 건조**
- **금지 사항:** 헤어드라이어, 전자레인지, 직접 열원 사용 금지
- **물기 제거:** 세척 후 "손으로 탈탈 털어 물기를 제거"한 후 건조 시작
- **위험:** 물기가 남으면 냄새 + 모터 손상 위험

**다이슨과의 차이:** 다이슨은 "선풍기 앞이나 열린 창가" 권장하지만, LG는 "그늘" 명시. 두 표현 모두 통풍+바람+자연 건조를 의미하나, "그늘"은 직사광선 피함을 강조.

### 5. 배터리 충전 불가 — 점검 순서
1. **충전 단자 청소** → 마른 수건이나 부드러운 솔로 이물질 제거 (무리한 힘 금지)
2. **충전 단자 재장착** → 빼냈다 다시 끼우기
3. **충전대 어댑터 확인** → 충전대 뒷면 어댑터 끼워짐 확인 + 전원 플러그 확인
4. **온도 환경** → 5°C 이하에서는 충전 불가 → 실내 상온(15°C 이상)에서 충전
5. **LED 표시 확인** → 충전 시작 시 LED 깜박임 → 완료 시 전체 점등 (깜박임 없으면 재접촉)
6. **배터리 수명 확인** → 완전 충전 후에도 사용 시간 지나치게 짧으면 배터리 교체 필요

**온도 민감성:** 배터리는 화학 장치로, 겨울철 베란다나 야외에서의 충전은 피하고, 실내 따뜻한 환경에서만 충전할 것.

### 6. 올인원타워 먼지 비우기 시스템 (A9S 특화 기능)

#### 기본 운영 모드
- **수동비움(Manual, 기본 설정):** 청소기를 충전대에 꽂은 후, 원할 때 시작/정지 버튼으로 수동 비움
- **자동비움(Auto):** 시작/정지 버튼을 3초 누르면 전환 → 청소 후 충전대 꽂으면 자동 비움

#### 작동 절차
1. 간편 비움 시스템 레버를 **아래로 눌러 먼지 압축**
2. 먼지통 덮개 분리 버튼을 눌러 **압축된 먼지 배출**
3. 올인원타워에서 먼지를 비우기 전에도 레버를 여러 번 눌러 먼지통을 정리할 것

#### 상태 표시등
| 표시등 | 상태 | 의미 |
|--------|------|------|
| 먼지비움 아이콘 | 빨간색 깜박임 | 먼지 봉투 교체 필요 |
| 먼지비움 아이콘 | 켜짐 | 비움 중 |
| 먼지비움 아이콘 | 꺼짐 | 비움 완료 |
| 보조배터리 아이콘 | 깜박임 | 충전 중 |
| 보조배터리 아이콘 | 켜짐 | 충전 완료 |

#### 먼지 비우기 불량 — 원인과 해결
- **원인 1:** 간편 비움 시스템 레버 상단·하단에 먼지 끼임 → 청소
- **원인 2:** 먼지통과 큰 먼지 분리망 사이 단단한 이물 끼임 → 레버가 내려가지 않음 → 이물 제거
- **예방:** 먼지비움 전 레버를 여러 번 눌러 먼지 압축

**봉투 교체 주기:** 빨간색 깜박임 표시 시 교체. 봉투가 없거나 가득 차면 비움 작동 불가.

### 7. 작동 중 갑자기 꺼짐 (이물 막힘 발생)
- **증상:** "우-웅 우-웅" 소리 + 표시등 "이물 막힘 알림" → 보호를 위해 자동 종료
- **조치 순서:**
  1. 먼지통·큰 먼지 분리 장치 분리·청소
  2. 배기필터·프리필터 청소
  3. 긴 나무 막대로 먼지통 안쪽 이물질 제거
  4. 길이 조절 연장관 내부 이물 제거
  5. 흡입구 이물 청소

### 8. 배터리 작동 시간 저하
- **원인 1:** 필터 오염 → 필터 세척으로 해결
- **원인 2:** 필터 미장착 → 필터 확인·체결
- **원인 3:** 배터리 수명 다 함(충분히 충전해도 지나치게 짧음) → 교체 필요
- **배터리 교체:** LG전자 홈페이지 소모품샵 또는 서비스센터

### 9. 고객 지원
- **통합 고객센터:** 1544-7777 (모든 AS·상담·점검 문의)
- **접근성:** 공식 자료는 모두 LG 스스로해결(support.lge.co.kr)에 수록

## 조사의 한계 및 미확인 사항

1. **배터리 리셋 방법:** "배터리 분리 후 15초 대기" 같은 소프트 리셋이 다이슨에는 있지만, LG 공식 문서에서는 명시되지 않음. 후기·커뮤니티에서 보고되는 방법이 있을 수 있으나, 공식 문서 부재 상태라 "확인실패"로 남김.

2. **필터 교체 주기:** LG 공식에서는 "월 1회 이상 세척"만 명시했으나, "교체" 주기는 명시되지 않음. 다이슨과 달리 LG는 필터 재사용식(세척·건조 후 재사용) 설계로 추정되나, 교체 시점은 미확인.

3. **에러코드 체계:** 다이슨처럼 LG도 "이물 막힘 알림", "배터리 저전 알림" 등이 표시되지만, 전체 에러코드 체계(정의, 상황별 구분, 해제 방법)는 공식 문서에 수록되지 않음.

4. **올인원타워 봉투 규격 및 교체 주기:** "빨간색 깜박임 시 교체"만 명시되어 있고, 봉투의 용량·규격·가격 등은 조사 범위 외.

## 글 집필을 위한 권장사항

1. **모델 명시:** A9, A9S 모두 동일한 점검 방법이지만, 올인원타워는 "A9S 주요 기능"임을 명시
2. **순차 대안 톤:** 흡입력 저하 시 6가지 점검을 "배터리 → 이물 → 먼지통 → 필터 → 부품 → 배터리 수명" 순으로 권고. 각 단계마다 "이 방법으로 안 되면 다음으로"
3. **필터 건조 강조:** "24시간" + "그늘" + "통풍" + "자연 건조"를 명확히 하고, 열원 금지를 안전 경고로 처리
4. **올인원타워 전용 섹션:** 기기 특화 기능이므로 별도 절차 영역 필요
5. **온도 환경:** 겨울철 충전 실패 사례가 많을 것으로 예상되므로, "5°C 이하 충전 불가" 명시
