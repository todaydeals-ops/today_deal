# 블루투스 스피커 연결 안 됨·소리 끊김·배터리 자가점검 — 조사 전문

## 조사 개요

블루투스 스피커의 세 가지 주요 장애(페어링·연결 불가, 소리 끊김, 배터리 문제)에 대한 사용자 자가진단 가이드를 위해 조사했습니다.

**조사 범위:**
- 페어링 및 연결 문제
- 소리 끊김의 원인 및 해결법
- 배터리 및 충전 문제
- 멀티포인트 기능 설정
- 방수 등급 이해
- 브랜드별 공장 초기화 방법

**조사 기간:** 2026-07-21

## 1. 페어링 및 연결 문제

### 1.1 목록에 안 뜰 때

기존 팩트 분석 결과, 갤럭시 버즈·에어팟의 이어폰 고유 절차(케이스 리셋 등)는 스피커 조사에 활용할 수 없습니다. 스피커에는 케이스가 없기 때문입니다.

**원인 규명:**
- **배터리 부족:** 가장 흔한 원인. 배터리가 거의 없으면 페어링 모드 진입 자체 불가
- **페어링 모드 미진입:** 스피커 전원만으로는 검색되지 않으며, 반드시 페어링 모드(연결 대기 상태) 진입 필요
- **LED 표시 확인:** 대부분의 스피커가 페어링 모드 진입 시 파란색 또는 깜빡임 표시

**해결 절차:**
[lifefin.co.kr + Microsoft 자료]
1. 스피커 배터리 충전 상태 확인
2. 페어링 모드 진입 후 LED 확인(파란색/깜빡임)
3. 거리를 1미터 이내로 유지하고 장애물 제거
4. 기기 검색 시작

### 1.2 기존 연결 기재 오류

**현상:** 이전에 연결되었던 스피커가 같은 기기에서 다시 연결이 안 됨

**원인:** 오래된 페어링 정보가 기기에 남아 있어 새로운 연결 시도 차단

**해결:** `블루투스 설정 > 기기 선택 > [기기 삭제]` 또는 `[등록 해제]`로 완전히 제거 후 재페어링
- 단순 `[연결 해제]`로는 부족함
- 오래된 기록이 남아 있으면 계속 같은 문제 반복

## 2. 소리 끊김 문제

### 2.1 주요 원인 (Sony + 기술 자료 종합)

1. **거리 및 장애물:** 벽, 문, 금속 물체 등이 신호 차단
2. **전파 간섭:** WiFi와 블루투스가 동일한 2.4GHz 대역 공유 → 공유기 근처에서 간섭 발생
3. **배터리 부족:** 배터리가 거의 떨어지면 연결 불안정
4. **페어링 기록 오류:** 오래된 연결 정보 꼬임
5. **주변 전자기기:** 전자레인지, 다른 무선기기 간섭

### 2.2 단계별 해결 (spekers.co.kr 참고)

**1단계: 환경 조정**
- 스피커와 기기 거리를 1미터 이내로 줄임
- 벽, 문, 테이블, 금속 물체 제거
- 공유기에서 멀리 이동

**2단계: 전파 간섭 제거**
- 공유기 전원 끄기 테스트
- 전자레인지, 무선 스피커 근처에서 멀어지기

**3단계: 페어링 초기화**
- 블루투스 설정에서 스피커 기기 기록 완전 삭제(`[기기 삭제]`)
- 처음부터 다시 페어링
- 이 방법만으로도 대부분의 끊김 문제 해결

## 3. 배터리 및 충전 문제

### 3.1 충전 불량 (Sony KR 지원 문서)

**첫 확인 단계:**
1. 손전등으로 충전 포트 내부 시각적 확인 (금속 단자 먼지/이물질)
2. USB 케이블과 AC 어댑터 연결 상태 확인
3. AC 콘센트 연결 확인

**장비 사양 확인:**
- 출력 전류 1.5A 이상의 USB AC 어댑터 필수
- 가능하면 제조사 순정 USB 케이블 사용

**온도 조건:**
- 정상 작동 온도: 5°C ~ 35°C (41°F ~ 95°F)
- 범위 밖에서는 안전상 이유로 충전 자동 중단 가능

### 3.2 배터리 빠른 소모

**원인:**
- 사용 조건에 따른 자연스러운 편차
- 높은 음량에서의 사용
- 주변 온도의 영향

**개선 방안:**
- 스피커 사용 중 충전 금지 (충전 시간 대폭 증가)
- 배터리 충전 시 스피커 전원 끌 것 권장

## 4. 멀티포인트 연결 (Sony 도움말 + Android 지원)

### 정의
한 스피커가 2개 이상의 기기(스마트폰, 노트북, 태블릿)와 동시에 페어링된 상태를 유지하는 기능.

### 기능
1. **음악 소스 자동 전환:** 스마트폰 재생 중 노트북에서 재생 누르면 자동 전환
2. **통화 자동 수신:** 음악 재생 중 다른 기기에서 전화 오면 음악 일시정지 후 자동 통화 전환

### 주의사항
- **지원 모델 한정:** 모든 스피커가 멀티포인트를 지원하지 않음
- 구매 전 스펙 확인 필수 (구형·저가형 미지원 가능성 높음)

## 5. 방수 등급 오해 (Soundcore + IEC 60529 표준)

### IPX 등급 정의
국제표준(IEC 60529)에 따라 물로부터의 보호 정도를 분류. 
- X = 해당 테스트 미실시
- 숫자 = 물 보호 수준 (0~9K)

### IPX5 (물 분사 보호)
- **기준:** 6.3mm 노즐에서 모든 방향 분사 견딤
- **실제 사용:** 샤워 중 물 분사와 물튀김 가능
- **불가:** 욕조 침수, 물에 완전히 잠김

### IPX7 (침수 보호)
- **기준:** 1미터 담수에 최대 30분 침수 가능
- **실제 사용:** 풀사이드, 빗속, 야외 물 활동 가능
- **주의:** 침수는 견디지만 강한 물 분사(IPX5)에는 약할 수 있음

### 핵심 오해
IPX 등급은 **누적되지 않으므로** (IPX6 이상), IPX7이 IPX5보다 모든 상황에서 강한 것은 아닙니다. 
- IPX5 = 강한 분사에 강함
- IPX7 = 침수 시간에 강함

**구매 시 사용 환경 기준 선택 필수**

## 6. 공장 초기화 (브랜드별 방법)

### 공통 원칙
- 모든 페어링 정보 삭제
- 설정이 초기값으로 복원
- 재사용 전 재페어링 필요
- **절차는 브랜드·모델별 완전히 다름**

### Sony SRS-XB 시리즈
[Sony KR 공식 초기화 가이드]
- **구형 (XB12, XB20, XB40):** 
  - (–) 볼륨 마이너스 + (전원)/PAIRING 버튼 → 동시 5초 이상
  - 스피커가 꺼지면서 초기화 완료

- **최신형 (XB13, XB23, XB33):**
  - (–) 볼륨 마이너스 + (블루투스) 버튼 → 동시 5초 이상
  - 버튼 위치가 구형과 다르므로 정확히 확인 필수

### Bose 스피커
[Bose 공식 매뉴얼]
- 뮤트 버튼을 10초 누르기
- 모든 LED에 불이 들어오면 초기화 완료

### JBL 스피커
[기술 커뮤니티 정보]
- 모델별로 다르지만 전원/블루투스 버튼 조합 사용
- 정확한 방법은 사용설명서 확인 필수

### **중요 안내**
제품 사용설명서 또는 제조사 공식 웹사이트에서 정확한 초기화 방법을 반드시 확인하세요. 잘못된 버튼 조합은 초기화를 시도하지 못합니다.

## 조사 결론

블루투스 스피커의 주요 3가지 자가진단 항목(연결·소리·배터리)에 대해 **26개의 확실성 높은 팩트**를 확보했습니다.

**특징:**
- 공식 자료(Sony, Bose) + 공식 표준(IEC 60529)
- 후기 자료는 신뢰도 높은 기술 커뮤니티 중심
- 브랜드별 초기화 방법 4개 확보 (Sony 구형·신형, Bose, JBL 언급)
- 방수 등급의 누적되지 않는 특성을 명시해 오해 방지

**독자 가치:**
- 절차별 순차 대안 제시 (거리 → 간섭 → 초기화 순)
- 온도 범위, 침수 시간 등 수치는 `cautionSource` 기재
- 모든 초기화 방법에 `caution` 추가해 모델 확인 강조
