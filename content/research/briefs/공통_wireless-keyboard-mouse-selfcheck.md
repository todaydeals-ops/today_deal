# 무선 키보드·마우스 연결 끊김·인식 안 됨 자가점검 조사 브리프

**작성일**: 2026-07-21  
**조사 범위**: USB RF 동글·블루투스 무선 입력기기 통합  
**핵심 발견**: USB 3.0과 2.4GHz 동글 간섭이 널리 알려지지 않은 주요 원인

---

## 1. 무선 키보드·마우스의 두 가지 기술 기반

### 1.1 USB RF 동글 (2.4GHz)
- **동작**: USB 리시버(동글)에서 2.4GHz RF 신호로 수신
- **장점**: 설정이 간단 (꽂으면 자동 인식), 단일 수신기로 여러 기기 연결 가능
- **단점**: 동글을 분실하면 마우스/키보드 사용 불가, USB 포트 필요
- **호환성**: USB 포트가 있는 모든 기기 (PC, 일부 태블릿)

### 1.2 블루투스
- **동작**: 기기 내장 칩에서 2.4GHz 블루투스 신호로 직접 연결
- **장점**: 여러 기기 동시 페어링, 동글 불필요, 증가하는 USB-C 대응 기기에 유리
- **단점**: 초기 페어링 절차 필요, 하나의 마우스/키보드는 보통 하나의 기기에만 자동 연결
- **호환성**: 블루투스 칩이 있는 모든 기기

**주목**: 두 방식 모두 **2.4GHz 대역을 사용**하므로, 같은 종류의 간섭 원인을 공유한다.

---

## 2. **USB 3.0과 2.4GHz 동글 간섭** — 잘 알려지지 않은 원인

### 2.1 문제의 기술적 원인 (ASUS 공식)
USB 3.x 장치가 신호를 송수신할 때 **2.4GHz 주파수 대역에 노이즈를 방출**한다. 이로 인해 같은 2.4GHz를 사용하는 무선 동글, Wi-Fi, 블루투스 신호가 간섭을 받는다.

### 2.2 증상
- **입력 지연**: 키보드 누르고 1~2초 뒤 입력 인식
- **마우스 끊김**: 마우스 커서가 갑자기 멈추거나 튀기 반복
- **간헐적 연결 두절**: 특정 시간대나 상황에서만 끊김

### 2.3 해결 방법 (우선순위 순)

#### ① USB 2.0 포트 사용 (가장 효과적)
USB 2.0은 저속 표준이라 2.4GHz 대역에 노이즈를 방출하지 않는다. 키보드·마우스는 고속 통신이 필요 없으므로 USB 2.0으로도 충분하다. **대부분의 PC에 USB 2.0 포트가 있다.**

> **PC 포트 확인**: 데스크탑 케이스 뒷면에 흰색(또는 검은색) 포트 = USB 2.0, 파란색 포트 = USB 3.0/3.1

#### ② USB 연장 케이블로 거리 확보
USB 2.0 포트가 없거나 찾기 어려우면, USB 3.0 포트를 사용하되 **USB 연장 케이블**로 동글을 포트에서 최소 20cm 이상 멀리 배치한다. 거리가 멀수록 간섭이 줄어든다.

- 연장 케이블로 가능한 멀리 배치 (30cm 이상 권장)
- 데스크탑 케이스 뒷면 포트에는 꽂되, 케이블로 전면으로 빼내기
- 모니터의 USB 허브 포트 활용 (케이스로부터 거리 확보)

#### ③ 주변 간섭원 정리
USB 3.0 포트 주변에서 고속 이동식 저장소(외장 SSD, 외장 하드) 제거. 이들도 간섭을 악화시킨다.

## 3. 인식 안 됨 자가점검 순서

### 3.1 배터리 확인
무선 기기가 인식되지 않으면 먼저 **배터리 용량**을 확인한다. 배터리 부족 경고가 있거나 전원 표시등이 없으면 교체한다.

### 3.2 물리적 연결 확인
- USB 동글이 포트에 제대로 삽입되었는지 육안 확인
- **포트를 바꿔본다**: 한 포트에서 안 되면 다른 포트 시도. PC의 여러 USB 포트가 독립적으로 작동하며, 특정 포트의 접촉 불량일 수 있다.

### 3.3 USB 절전 설정 확인
특히 노트북의 경우, Windows의 **USB 선택적 절전** 기능이 켜져 있으면 절전 복귀 후 USB 동글이 인식되지 않을 수 있다.

- **경로**: 제어판 > 전원 옵션 > 고급 전원 설정
- **설정**: USB 선택적 절전 모드 → **사용 안 함**
- **증상**: 점심시간 이후, 회의 후 처음 누르는 키보드가 반응 없음 (이후 자동 복구)

### 3.4 거리 확인
무선 동글의 작동 거리는 일반적으로 10~20m이지만, 벽·금속 물체가 많으면 5~10m로 줄어든다. **기기를 수신기로부터 최소 10m 이내에서 사용**한다.

## 4. 연결 끊김 원인별 대응

### 4.1 연간헐적 끊김 (간섭)
위의 **USB 3.0 간섭** 섹션 참조. USB 2.0 포트 또는 연장 케이블 사용이 1차 해결책이다.

### 4.2 같은 2.4GHz 대역 기기 간섭
무선 키보드·마우스(2.4GHz RF) ↔ 무선 마우스(2.4GHz RF), 블루투스 스피커, Wi-Fi, 전자레인지 등이 동시 사용 시 발생. 

- Wi-Fi를 5GHz로 변경 가능하면 변경
- 무선 동글의 채널을 변경 (제품 설정 도구에서 가능한 경우)
- 간섭원 기기와의 거리 확보

---

## 5. 마우스 커서 튐·부정확한 움직임

### 5.1 원인: 센서 먼지
광학 센서 부분에 먼지나 마우스 패드 갈린 먼지가 쌓이면 커서가 튄다.

### 5.2 센서 청소
- **도구**: 깨끗한 건조 천, 부드러운 이쑤시개, 압축 공기
- **방법**: 마우스 밑바닥의 LED(센서) 부분을 부드러운 천으로 닦기
- **주기**: 월 1회 정도 유지보수
- **주의**: **절대 알코올 사용 금지** (손 소독제, 알코올 와이프 등). 알코올은 외부 코팅을 부식시킨다.

### 5.3 표면 호환성 확인
광학 센서는 **투명 유리, 거울, 고광택 표면에서는 정상 작동하지 않는다.** 마우스 패드를 깔거나 표면을 덮어 사용한다.

---

## 6. 배터리 소모 빠르기 (배터리 관리)

### 6.1 LED 조명 관리
마우스·키보드에 LED가 있으면 밝기를 낮추거나 끈다. LED는 상당한 전력을 소비한다.

### 6.2 거리 유지
수신기로부터 10m 이내에 기기를 유지한다. 거리가 멀면 신호 강도를 높여야 하므로 전력 소비 증가.

### 6.3 전원 스위치 활용
마우스 밑면에 전원 스위치가 있으면, 사용하지 않을 때(야간, 휴무일) 꺼둔다.

---

## 7. 특정 키만 안 눘림 / 입력 먹통

### 7.1 원인 판별
- **일부 키 안 눌림**: 페어링 문제일 가능성 높음
- **전체 입력 안 됨**: USB 동글 인식 문제 (배터리, 포트, 절전 설정)

### 7.2 해결 절차 (순서대로)

#### ① 키보드 초기화
대부분의 무선 키보드는 **Fn + ESC를 약 3초 이상** 눌러 초기화한다. 초기화 후에는 페어링을 다시 해야 한다.

> **주의**: 제품마다 초기화 방법이 다를 수 있다. 정확한 방법은 제품 매뉴얼 참조.

#### ② 페어링 재설정
Windows: 설정 > 블루투스 및 기타 장치 > 기존 키보드 제거 → 키보드 페어링 모드 활성화 → 다시 추가

#### ③ 드라이버 업데이트
Windows 키 + X > 장치 관리자 > 키보드 확장 > 해당 키보드 우클릭 > 드라이버 업데이트

---

## 8. 절전 모드 복귀 지연 (블루투스)

블루투스 키보드에서 발생. 절전 모드에서 깨어날 때 첫 키 입력이 반응하지 않거나 여러 번 눌러야 깨어난다. 

- **원인**: 블루투스 연결 재확립 시간
- **대응**: 여러 번 눌러야 할 수 있음을 사용자에게 안내. Windows의 USB 절전 설정 비활성화도 도움.

---

## 9. 미확인 항목 (tier: 확인실패)

- **무선 동글 펌웨어 업데이트 가능성**: 제조사마다 다르므로, 제품 공식 웹사이트의 지원 페이지 확인 필요. 업데이트 가능하면 연결 안정성 개선될 가능성 있음.

## 글쓰기 시 주의점

1. **USB 3.0 간섭은 독립적 섹션**으로 강조. 많은 사용자가 모르는 원인이며, 이를 알리는 것이 큰 가치다.
2. **해결책은 우선순위 순**으로 제시 (USB 2.0 포트 사용이 가장 효과적).
3. **"해보세요"로 끝내되, 각 단계마다 "다음에 이걸 시도하세요"로 순차 대안 제시**.
4. **후기 기반 정보**(마우스 센서, 배터리, 절전 문제)에는 "사용자들이 보고한", "많은 경우" 같은 표현으로 신뢰도 명시.
5. **경고 사항** (알코올 사용 금지, USB 2.0 포트 찾기)은 명확히 강조.
