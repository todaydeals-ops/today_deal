# 아이폰 Face ID 인식 안 됨·카메라 흐림 자가점검

## 조사 개요

아이폰 Face ID가 인식되지 않거나 카메라가 흐릿해지는 현상을 사용자가 스스로 점검하고 대처할 수 있는 방법을 정리했다. 공식 Apple Support 자료와 기술 커뮤니티 자료를 바탕으로 소프트웨어·하드웨어 단계별 해결 프로세스를 구성했다.

---

## 1. Face ID 인식 실패 — 기본 점검 사항

### 기종 및 iOS 버전 확인

**Face ID는 iPhone X 이후에서만 지원된다.** 기종별 기능 차이가 있으므로 먼저 자신의 기종을 확인해야 한다.

- **마스크 착용 상태 인식**: iPhone 12 이후 + iOS 15.4 이상에서만 지원.
  - iPhone X, 11: 마스크 착용 시 Face ID 사용 불가. (홍채 부분을 반드시 노출해야 함)
  - iPhone 12~현재: 설정 > Face ID 및 암호 > '마스크로 Face ID 사용' 활성화 필요.

### 거리와 각도 최적화

Apple 공식 문서에 따르면 **Face ID는 25~50cm(약 10~20인치) 거리에서 최적으로 작동한다.**

- 기기가 정면에서 약 45도 각도까지는 인식되지만, 그 이상 기울어지면 인식률이 떨어진다.
- 장시간 같은 각도에서 사용하면 얼굴 스캔 데이터가 해당 각도에 편향되어 다른 각도에서는 인식이 어려워질 수 있다. 이 경우 다양한 각도에서 사용하거나 Face ID를 재설정해야 한다.

### 조명 환경 확인

**극도로 어두운 환경과 직사광선 모두 Face ID 인식을 방해한다.**

- Face ID는 적외선 카메라를 사용하므로 가시광선 조명이 완전히 없어도 작동하지만, 극도로 어두우면(예: 조명 없는 어두운 방) 센서가 얼굴 특징을 캡처하기 어렵다.
- 반대로 햇빛이 강한 야외에서는 적외선 카메라가 간섭받아 작동하지 않을 수 있다.

**해결**: 조명이 적당한 실내에서 테스트해볼 것.

## 2. TrueDepth 카메라 물리적 장애물 확인 — 고장으로 오해하는 대표 사례

### 케이스와 화면 보호필름 제거

**케이스나 화면 보호필름이 iPhone 상단의 노치(또는 다이나믹 아일랜드) 영역을 덮고 있으면 Face ID 인식이 완전히 불가능하다.**

Apple 공식 가이드에서는 Face ID 문제가 생기면 먼저 다음을 확인하도록 명시:
1. 케이스 제거
2. 화면 보호필름 제거
3. TrueDepth 카메라 렌즈 청소

이는 사용자들이 **고장이라고 생각해 AS 신청하는 가장 흔한 원인**이다.

### TrueDepth 카메라 청소

iPhone 상단 중앙의 노치/다이나믹 아일랜드 영역에 여러 센서가 들어있다:
- 적외선 플러드 조명(Infrared Flood Illuminator)
- 적외선 카메라(Infrared Camera)
- 도트 프로젝터(Dot Projector)

**먼지, 기름, 또는 물때가 이 영역을 가리면 인식이 불가능하다.**

**청소 방법:**
- 부드러운 마이크로파이버 천으로 가볍게 닦기
- 물이나 용매 사용 금지
- 센서를 누르거나 문지르지 말 것

---

## 3. Face ID 설정 메시지별 대처

### "얼굴을 인식할 수 없음" / "얼굴을 캡처할 수 없음"

**원인**: 설정 중 조명 부족, 얼굴이 작게 들어옴, 카메라 막힘

**대처**:
1. 케이스·필름 제거 후 재시도
2. 밝은 장소에서 정면을 마주하고 설정 재시도
3. 기기 강제 재시작 후 다시 설정
4. 계속 실패하면 Face ID 재설정 시도

### "Face ID를 사용할 수 없음" 또는 "Face ID를 활성화할 수 없습니다"

**이 메시지는 TrueDepth 카메라 시스템의 하드웨어 결함을 의미한다.** 소프트웨어로는 해결 불가능하다.

**원인**:
- TrueDepth 카메라 부품 손상
- 도트 프로젝터 또는 적외선 카메라 고장
- 센서 연결 불량

**대처**: Apple 정식 서비스센터 방문 필수. 자가진단으로 해결 불가.

---

## 4. Face ID 재설정

기본 점검으로도 인식이 안 되면 Face ID를 처음부터 다시 등록해본다.

**절차**:
1. 설정 > Face ID 및 암호 > '암호' 입력
2. 아래로 스크롤 > 'Face ID 재설정' 선택
3. 기존 얼굴 데이터 삭제 확인
4. 다시 설정하기 > 기기를 얼굴 가까이 두고 천천히 돌리며 스캔

**조건**:
- 밝은 조명에서 진행
- 안경 착용 여부를 설정 중 미리 선택
- 다양한 각도에서 얼굴을 스캔해야 정확도 상승

**대체 외모 추가**: 설정 후 '대체 외모' 옵션에서 안경을 쓴 상태, 수염이 있는 상태 등 일상의 다양한 모습을 추가 학습시킬 수 있다.

## 5. 카메라 흐림·초점 불일치 — 렌즈부터 점검

### 렌즈 청소

**카메라가 흐리거나 초점을 잡지 못하는 첫 번째 원인은 거의 항상 렌즈 먼지 또는 물때다.**

**청소 방법**:
- 부드러운 마이크로파이버 천으로 렌즈 표면을 가볍게 닦기
- 물, 알코올, 초음파 청소기 사용 금지
- 강한 압력으로 비비지 말 것

효과: 대부분의 "흐릿함" 현상이 렌즈 청소로 해결된다.

### 렌즈 청소 후에도 흐린 경우

**OIS(광학 손떨림 보정) 또는 AF(자동초점) 메커니즘 손상의 신호.**

#### OIS 손상 가능성

- **iPhone 6 Plus, 6s Plus, 7 이후 모델에 탑재**
- OIS 메커니즘이 손상되면 사진이 흔들려 보이고 초점이 맞아도 "떨린" 영상이 된다.
- **고진동 환경(오토바이 엔진 진동 등)에 노출되면 OIS가 손상될 수 있다는 것을 Apple이 공식 안내했다.**

#### 강한 진동으로 인한 OIS 손상 — Apple 공식 경고

2023년 Apple은 다음을 명시적으로 공지:
- 고출력 오토바이 엔진의 강한 진동에 장시간 노출되면 OIS와 AF 시스템이 손상되어 사진·영상 화질이 저하될 수 있다.
- OIS는 iPhone 6 Plus 이후 모든 기종에 있다.
- 폐루프 AF(자동초점 메커니즘)는 iPhone XS 이후에만 탑재되어 있다.

**예방법**: 오토바이에 아이폰을 고정해야 한다면 반드시 **진동 감쇠 마운트(vibration dampening mount)** 또는 **짐벌(gimbal)**을 사용할 것. 스쿠터나 전동 오토바이라도 마찬가지로 진동 대응 마운트 사용을 권장한다.

### 렌즈 응결(김서림)

**추운 환경 → 따뜻한 실내로 이동 또는 뜨거운 물의 증기에 노출되면 렌즈 내부에 응결이 발생할 수 있다.**

- 급격한 온도 변화가 주 원인
- 렌즈를 서서히 자연 건조시키면 대부분 해결
- 드라이어나 열로 빠르게 제거하려 하면 렌즈 코팅이 손상될 수 있음

## 6. 카메라 앱 검은 화면

### 소프트웨어 단계

1. **카메라 앱 강제 종료 후 재시작**
   - iOS 17 이후: 화면 상단에서 아래로 스와이프 > 앱 카드 길게 누른 후 제거
   - iOS 16 이하: 설정 > 일반 > 멀티태스킹 > 카메라 앱 찾아 비활성화

2. **저장공간 확인**
   - 설정 > 일반 > iPhone 저장공간
   - 저장공간이 부족(5GB 미만)하면 카메라가 검은 화면을 띄울 수 있다.
   - 불필요한 앱·사진·영상 삭제로 여유공간 확보

3. **iOS 업데이트 확인**
   - 설정 > 일반 > 소프트웨어 업데이트
   - iOS 버그일 수 있으므로 최신 버전이 있는지 확인

## 7. 카메라 플래시 작동 안 함

### 배터리 확인

- **배터리가 15% 이하이면 플래시는 자동으로 비활성화된다.** (배터리 최적화 모드 활성화 시에도)
- 충전 후 카메라 앱을 재시작하면 플래시가 복구되는 경우가 대부분.

### 카메라 앱 설정 확인

1. 카메라 앱 열기
2. 화면 상단 플래시 아이콘 확인
   - 번개 모양(활성) / 번개+금지(꺼짐) / 번개+자동
3. 플래시 설정을 '자동' 또는 '항상 켜짐'으로 변경

### 하드웨어 고장

위의 모든 방법을 시도했는데도 플래시가 나오지 않으면 LED 플래시 하드웨어 고장을 의심해야 한다. Apple 서비스센터 점검이 필요하다.

## 8. 카메라 렌즈 스크래치·보호 유리 손상

### 렌즈 스크래치의 영향

- **얕은 스크래치**: 사진 화질에 미치는 영향은 거의 없다. (렌즈 자체의 광학 특성은 유지)
- **깊은 흠집**: 광학 성능을 저하시킬 수 있으며, 내부 먼지 침투 위험 증대

### 렌즈 보호 유리·필름 손상 (iPhone 11 이후)

- **iPhone 11 이후 모델의 후면 카메라는 렌즈 보호 유리로 덮여 있다.**
- 이 보호 유리가 손상되면:
  1. 먼지가 카메라 모듈 내부로 침투 가능
  2. 수분이 내부로 들어갈 수 있음
  3. 사진 화질이 저하될 수 있음

**대처**: 보호 유리 손상이 보이면 Apple 정식 서비스센터에서 교체할 것을 권장.

## 9. 정리 — 자가점검 체크리스트

### Face ID 인식 불가

- [ ] 기종이 iPhone X 이상인가?
- [ ] 케이스와 화면 보호필름을 제거했는가?
- [ ] TrueDepth 카메라(상단 노치/다이나믹 아일랜드)를 마이크로파이버 천으로 닦았는가?
- [ ] 밝은 장소에서 정면을 마주하고 설정을 다시 시도했는가?
- [ ] 기기 강제 재시작을 했는가?
- [ ] Face ID를 완전히 재설정해보았는가?

**위의 모든 방법을 시도했는데도 "Face ID를 사용할 수 없음" 메시지가 나타나면 → Apple 서비스센터 방문**

### 카메라 흐림·초점 불가

- [ ] 마이크로파이버 천으로 렌즈를 가볍게 닦았는가?
- [ ] 렌즈 청소 후에도 여전히 흐린가?
  - 그렇다면 OIS/AF 메커니즘 손상 가능성. 특히 오토바이에 고정한 적이 있는가?
- [ ] 렌즈 응결은 없는가? (추운 환경에서 따뜻한 곳으로 급격히 이동했는가?)

**렌즈 청소로 해결 안 되면 → Apple 서비스센터 방문**

### 카메라 앱 검은 화면

- [ ] 카메라 앱을 강제 종료 후 다시 열었는가?
- [ ] 저장공간이 충분한가? (설정 > 일반 > iPhone 저장공간)
- [ ] iOS를 최신 버전으로 업데이트했는가?

**위의 모든 방법을 시도했는데도 안 되면 → Apple 서비스센터 방문 또는 원격 진단 신청**

### 카메라 플래시 작동 안 함

- [ ] 배터리를 충전했는가?
- [ ] 카메라 앱의 플래시 설정이 켜져 있는가?
- [ ] 배터리 최적화 모드를 끼야 하는가?

**위의 모든 방법을 시도했는데도 안 되면 → Apple 서비스센터 방문**

## 10. 서비스센터 방문 기준

Apple 정식 서비스센터 또는 Apple Authorized Service Provider(AASP) 방문이 필요한 경우:

1. **Face ID**
   - "Face ID를 사용할 수 없음" 메시지 표시
   - 모든 기본 점검 후에도 인식 안 됨

2. **카메라**
   - 렌즈 청소 후에도 계속 흐림
   - 카메라 앱 검은 화면이 지속
   - 플래시가 완전히 작동하지 않음
   - 렌즈 내부 응결이 자연 건조로 해결되지 않음

3. **물리적 손상**
   - 렌즈 보호 유리 깨짐
   - 노치/다이나믹 아일랜드 영역의 센서 손상
