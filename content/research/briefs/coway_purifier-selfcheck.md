# 코웨이 정수기 자가점검 조사 브리프

**조사 주제:** 코웨이 정수기 렌탈 제품의 자가점검 항목 및 범위, 필터 교체 주기, 렌탈과 자가구매의 서비스 경계선

**조사 범위:**
- 브랜드: 코웨이 (웅진코웨이)
- 제품: 정수기 (렌탈 기준)
- 조사 기간: 2026-07-20
- 지역: 한국(KR)

---

## 주요 발견

### 1. 렌탈 vs 자가구매 서비스 구조

**렌탈 제품 (하트서비스):**
- 정기 방문관리: 2~6개월 주기 점검
- 필터 교체: 방문 전문가 시행 (무상)
- 토탈케어 서비스: 렌탈 기간 중 1회 무상 (내외부 부품 교체·탱크 살균·세척)
- 점검 항목: 외부 점검, 탱크/수로 살균, 아이스메이커 트레이 확인

**자가구매 제품:**
- 자가교체 옵션: 정기배송 서비스로 필터 자가 교체
- 수도호스 교체 가능 (정수기만 해당)
- 멤버십 서비스: 월 일정 금액으로 점검 및 필터 교체 가능

**핵심 구분:** 렌탈은 "방문관리가 원칙", 자가구매는 "자가관리 또는 유상 서비스" 선택 가능

### 2. 물 안 나올 때 자가점검 순서

1. **원수공급밸브 확인** (가장 흔한 원인)
   - 반시계 방향으로 더 이상 돌아가지 않을 때까지 완전히 열기
   - 필터 교체/수도 공사 후 특히 확인 필요

2. **필터 교체 후 플러싱**
   - 새 필터 장착 후 반드시 추출 버튼으로 물 배출
   - 정수 모드로 3분 이상 물 배출 (필터 활성화)

3. **전원 및 수도 단수 확인**
   - 콘센트 연결 상태
   - 수도 단수 여부

### 3. 냉온수 이상 증상별 자가점검

**공통 기본 점검:**
- LED 버튼 확인: 냉수/온수 LED가 켜져 있는지 확인 (꺼져 있으면 ON으로 전환)
- 전원 리셋: 플러그를 빼고 5분 대기 후 재연결

**온수 특이 사항:**
- 온수 가열 시간: 약 5분(미지근)→30분(뜨거움)
- 온수 버튼은 켜도 즉시 뜨거운 물이 안 나옴 (정상)

**냉수만 안 나오는 경우:**
- 냉각 시스템 점검 (전원 리셋 후 30분 대기)

### 4. 누수 감지 에러 (E01)

**자가점검 단계:**
1. 정수기 하단 및 주변 바닥 물 고임 확인
2. 고인 물 완전히 닦아내기
3. 전원 플러그 뽑고 30분 대기
4. 전원 재연결 후 에러 지속 여부 확인

**원인:** 대부분 밸브 교체로 해결 (전문 서비스 필요)

**주의:** 원수 공급 호스는 기존 재사용 금지, 제대로 된 부품 사용

### 5. 필터 교체 주기 및 알림

**교체 주기:**
- 정기 점검: 2~6개월 주기
- 필터 교체: 일반적으로 6개월~1년
- 렌탈: 방문 전문가가 일정에 맞춰 교체
- 자가구매: 정기배송 또는 별도 신청

**알림 서비스:**
- 렌탈/자가구매 모두 교체 주기 사전 알림 제공
- 교체 후 확인 안내

**물 이상한 맛:**
- 필터 교체 후 플러싱 부족 (3분 이상 필요)
- 정기 점검 연체로 필터 수명 초과

### 6. 고객센터 및 서비스 신청

**주요 연락처:**
- 코웨이 고객센터: 1588-5200
- 지역별 렌탈샵 안내 가능
- 자가 교체 불가 시 유상 서비스 신청 가능

**서비스 범위 확인:**
- 렌탈 상품: 토탈케어 1회 무상 (기간 내)
- 자가구매: 멤버십으로 점검/교체 서비스 가능
- 긴급 수리: 고객센터 즉시 접수

## 자가점검 범위의 명확한 경계

### 자가점검 가능 (공식 안내)
✓ 냉수/온수 LED 확인  
✓ 전원 리셋 (플러그 재연결)  
✓ 원수공급밸브 개폐 확인  
✓ 수도 단수 여부 확인  
✓ 필터 교체 후 플러싱  
✓ 먼지필터 청소 (물로 헹굼)  
✓ 바닥 물 고임 확인 (누수 신호)

### 자가분해 금지 (렌탈 제품)
✗ 냉각/가열 부품 분해  
✗ 밸브 교체  
✗ 수로관 분해  
✗ 탱크 분리  
✗ 내부 회로 조정

**원칙:** 렌탈 제품은 방문관리 범위에서만 유지보수. 자가구매도 복잡한 분해는 권장하지 않음(문제 시 유상 서비스)
