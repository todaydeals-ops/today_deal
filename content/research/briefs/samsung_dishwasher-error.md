## 1. 에러코드 체계

삼성 식기세척기는 5가지 주요 에러코드를 표시한다. 각 코드는 특정 기능의 오류를 가리킨다.

### 1.1 누수 (E03, FE, LC)
- **의미:** 급수 시스템에서 물이 새고 있는 상태
- **첫 점검:** 급수 밸브와 연결부를 육안으로 확인
- **조치:** 물이 흐르면 즉시 급수 밸브를 잠금

### 1.2 급수 불량 (4C)
- **의미:** 기기가 물을 받지 못함
- **점검 순서:**
  1. 싱크대 수도꼭지에서 물이 나오는지 확인
  2. 식기세척기 전용 급수 밸브가 열려 있는지 확인
  3. 급수 호스가 얼어 있거나 막혀 있지 않은지 확인

### 1.3 배수 불량 (5C)
- **의미:** 기기가 물을 빼내지 못함
- **점검 순서:**
  1. 배수 필터의 음식물 잔여물 청소
  2. 배수 호스가 꺾여 있지 않은지 확인
  3. 강제 배수 테스트 실행 (모델별로 버튼 조합이 다름)

### 1.4 펌프 오작동 (3C)
- **의미:** 세척(순환) 펌프가 정상 작동하지 않음
- **첫 조치:** 전원 완전 off → 재시작
- **미해결 시:** 서비스 신청 필요

### 1.5 고온 발열 (HC)
- **의미:** 가열 시스템의 온도 상한 감지
- **첫 조치:** 세제를 넣고 빈 식기세척기로 한 사이클 운행
- **지속 시:** 전원을 끄고 서비스 신청

## 2. 배수 필터 청소

### 2.1 분리 방법
- **도구:** 손 (특별한 도구 불필요)
- **방향:** 시계 반대 방향으로 회전
- **분리 후:** 위쪽으로 들어올림
- **참고:** 모델에 따라 핸들 모양이 다를 수 있으나 회전 방향은 동일

### 2.2 청소 주기
- **기준:** 주기적으로 분리해 음식물 잔여물 확인
- **구체적 숫자:** 공식 문서에는 주기가 명시되지 않음. 사용 빈도와 음식물 종류에 따라 시각적으로 판단
- **미청소 시 결과:**
  - 세척 성능 저하
  - 배수 불량 유발
  - 5C 에러 발생 가능성 증대

### 2.3 정상 상태 vs 고장
- **배수 필터 아래 물 고임:** 정상. 하수 냄새 역류 방지 설계
- **바닥 전체에 물이 고임:** 배수 펌프 오작동 → 강제 배수 테스트 실행

## 3. 세척 불량

### 3.1 적재(로딩) 방법과 분사암
- **점검 항목:** 상단 분사암이 음식이나 그릇으로 막혀 회전하지 못하는지 확인
- **해결 방법:** 분사암이 자유롭게 회전하도록 식기 배치를 조정
- **공식 문서 한계:** 구체적인 적재 규칙(예: 상단 바구니는 언제 사용, 몇 cm 간격)은 이 조사에서 확보하지 못함

### 3.2 세제 종류
- 공식 문서에서 지정 세제 또는 세제 사용량 규칙은 이 조사 범위에서 확보하지 못함
- 향후 조사 필요 항목

## 4. 배수 불량의 자가진단

### 4.1 강제 배수 테스트
- **목적:** 배수 펌프가 정상 작동하는지 별도 확인
- **실행 방법:** 모델마다 다름 (공식 문서 참고)
  - 방법 1: 전원 → 시작 → 20~30초 후 일시정지/재개
  - 방법 2: 도어 닫고 시작 버튼 3초 누르기

### 4.2 진단 판정
- **정상 신호:** 강제 배수 후 필터 아래 물이 빠짐
- **이상 신호:** 바닥까지 물이 고여 있음 → 서비스 필요

## 5. 물 얼룩 해결

### 5.1 원인
세척 완료 후 건조 단계에서 물방울이 식기 표면에 맺혀 건조되지 않으면 물 자국이 남는다.

### 5.2 헹굼보조제(린스)의 역할
- **작용:** 물의 표면장력을 낮춤
- **효과:**
  - 물방울이 빠르게 흘러내림
  - 건조 성능 향상
  - 물 얼룩 감소
  - 그릇의 광택 향상

### 5.3 헹굼보조제 사용법
- **투입 위치:** 세제 투입구 옆 길쭉한 칸(헹굼 칸)
- **용량:** 'Fill' 표시선까지 채움
- **투입 시기:** 수동이 아닌 자동 (기계가 세척 중 자동 투입)

### 5.4 대체재 사용 금지
- **식초 대용:** 절대 금지
- **이유:** 헹굼 분배기 메커니즘 손상
- **대체재 불가능:** 반드시 공식 헹굼보조제 사용

## 6. 건조 불량

### 6.1 건조 원리
- **방식:** 가열된 물의 잠열을 이용한 물 기반 건조
- **특징:** 외부 가열 요소를 사용하지 않음
- **결과:** 전통적인 가열식 건조기보다 느릴 수 있음

### 6.2 세척 완료 후 내부 습기
- **현상:** 고장이 아닌 **정상**
- **이유:** 물 기반 건조 방식의 필연적 결과

### 6.3 건조 성능 향상 팁
1. **헹굼보조제 사용** (앞서 설명)
2. **도어 개방:** 세척 완료 후 1~2시간 도어를 살짝 열어 수증기 배출
3. **조합 효과:** 두 방법을 동시에 적용하면 더 효과적

## 7. 조사 한계 및 향후 과제

### 7.1 확보되지 않은 정보
- 배수 필터 청소 구체적 주기 (월 1회? 주 2회?)
- 식기 적재 규칙 (상단/하단 바구니 사용 기준, 최소 간격)
- 권장 세제 종류 및 사용량
- 비스포크/프리미엄 라인의 추가 기능
- 모델별 강제 배수 테스트 정확한 버튼 조합 (문서는 "모델에 따라 다름"만 명시)

### 7.2 근거 확보 상황
- 에러코드 5개: 모두 공식 문서에서 확보
- 배수/건조 자가점검: 공식 문서 기반
- 헹굼보조제: 공식 문서에서 상세히 설명

### 7.3 다음 조사자를 위한 메모
- 삼성 공식 사용설명서(매뉴얼) 업로드 페이지에는 모델별 상세 절차가 있을 것으로 예상
- 비스포크 라인의 경우 별도 에러코드나 기능이 있을 가능성
- 사용자 후기에서 실제 필터 청소 주기를 찾을 수 있을 것으로 예상
