# 삼성 비스포크 김치플러스 김치냉장고 에러코드·냉각 불량 자가점검 조사 보고

**조사 기간**: 2026-07-21  
**출처 기준**: 삼성전자서비스 공식 문서 우선  
**검색 횟수**: 웹 검색 2회 + 웹페치 4회 (총 6회)

## 주요 발견 사항

### 1. 에러코드 체계

삼성 김치냉장고의 에러코드는 주로 **온도 센서 불량** 관련 코드 위주다:

| 에러코드 | 의미 | 조치 |
|---|---|---|
| **HO** | 상단 온도 센서 이상 | 전문가 점검 필요 |
| **nO** | 중단 온도 센서 이상 | 전문가 점검 필요 |
| **LO, RO** | 좌우 센서 불량 (온도 감지 불가) | 냉기가 약해지거나 음식이 상함 → 전문 점검 필수 |
| **LL, RR (동시 점멸)** | 양쪽 모두 온도 도달 불가 | 심각한 냉각 이상 → 즉시 점검 |
| **H5, n5** | 센서/내부 온도 이상 감지 | 에러별로 원인 다름 |

**비교: 위니아 딤채와 다른 점**
- 위니아는 C, E/E1/EA/E2, F/rF 등 더 다양한 에러코드를 사용
- 삼성은 센서 관련(HO/nO/LO/RO) 코드 위주
- **제조사별 에러코드 체계가 완전히 다르므로 혼용 불가**

### 2. 냉각 불량 자가점검

**냉기가 약할 때 해결 순서** (공식 안내):

1. **온도 설정 조정** (가장 먼저 시도)
   - 현재 설정에서 한 단계 높게: 약냉 → 표준 → 강냉
   - 대부분의 경우 이것만으로 해결

2. **저장 식품 양 확인**
   - 과다 저장 시 냉각 효율이 저하됨
   - 적정 양으로 조절

3. **도어 밀폐 확인**
   - 도어가 완전히 닫혔는지 확인
   - 자주 여닫으면 냉기 손실 발생

4. **냉각 시스템 정상 여부 확인**
   - 냉장고 벽면에 **서리가 쌓여 있는가?**
   - 벽면 서리 + 식품 냉기 = 냉각 시스템 정상
   - 서리가 없으면 센서/압축기 불량 의심 → 전문 점검

### 3. 김치 보관 온도 설정

삼성 기준으로 소금 함량에 따라 3단계 온도 모드 제공:

**표준 소금 김치 (1.5% 이하)**
- 강냉: **-1.8°C** ± 0.5
- 표준: **-1.3°C** ± 0.5 (가장 많이 사용)
- 약냉: **-0.8°C** ± 0.5

**저염 김치 (0.7% 소금)**
- 강냉: **-1.0°C**
- 표준: **-0.5°C**
- 약냉: **0.0°C**

**주의**: 저염 김치는 1~2주 내 동결되지 않도록 모니터링 필수  
**얼음 결정 발생 시**: 더 따뜻한 모드로 즉시 조정 필요

### 4. 온도조절 방법

- **스탠드형**: 오른쪽 화살표 → 칸 선택(상/중/하) → OK 버튼 → 모드 선택(강냉/표준/약냉/저염 등) → OK로 저장
- **뚜껑형**: 모델별 디스플레이 타입 상이 (Aram/Mouton 등)
- **모델별로 조작 방법이 다르므로 제품 설명서 필독**

### 5. 전원 리셋

**일시적 오류 해결법**:
1. 전원 코드를 빼기
2. 충분히 대기
3. 전원 코드를 다시 연결
4. **약 16시간 동안 냉장고가 온도 재감지 → 에러 표시 여부 재확인**

**중요**: 리셋 후 에러가 사라졌어도 16시간 후 다시 나타나면 기능적 불량 확정

### 6. 회수하지 않은 정보

**웹 검색 범위 내에서 확인되지 않은 항목**:
- 정전 후 복구 시간 (위니아는 약 10분, 삼성은 명시 안 됨)
- 뚜껑형 vs 스탠드형 냉각 구조 차이 (삼성은 구조 차이를 공식 문서에서 설명하지 않음)
- 스마트싱스 자가진단 기능 (별도 조사 필요)
- 성에/결로 관련 상세 대책 (벽면 서리만 언급)
- 소음 관련 (언급 없음)
- 문 안 닫힘 문제 (별도 조사 필요)

### 7. 공식 고객센터
