# HP 프린터 인쇄 안 됨·오프라인·잉크 인식 오류 자가점검

조사 날짜: 2026-07-21  
대상: HP 프린터 전 라인 (잉크젯·레이저젯 겸용)  
범위: 한국(KR), 글로벌 겸용

---

## 1. "인쇄 불가" 오류의 단계별 진단

HP 프린터가 인쇄되지 않거나 오프라인으로 표시될 때 확인 순서는 다음과 같다:

### 1-1 연결 상태 확인 (첫 번째)
**USB 연결 프린터**
- 프린터 뒷면의 USB 케이블이 제대로 꽂혀 있는지 확인
- 케이블을 한 번 빼고 다시 꽂아 재접속 시도

**무선(Wi-Fi) 연결 프린터**
- 프린터 제어판의 Wi-Fi 표시등 확인 (켜져 있으면 정상)
- 컴퓨터와 프린터가 **동일한 네트워크(라우터)**에 연결되어 있는지 확인
- 다른 기기(스마트폰 등)에서도 같은 Wi-Fi에 접속되는지 확인

**참고**: 프린터와 컴퓨터가 다른 라우터나 핫스팟에 연결되면 인쇄 불가능하다.

### 1-2 프린터 오프라인 상태 해제 (Windows 필수)
**증상**: "프린터 오프라인" 표시가 떠 있음에도 실제로는 연결되어 있을 때

**해결 방법**:
1. Windows 제어판 열기
2. 장치 및 프린터 → 프린터 목록에서 해당 프린터 찾기
3. 프린터를 **우클릭** → "인쇄 작업 목록 보기" 클릭
4. 창 상단 메뉴 → **"프린터" 탭 → "프린터를 오프라인으로 사용" 체크 해제**
5. 프린터 재부팅 후 인쇄 재시도

**결과**: 체크를 해제하면 대기 중인 모든 문서가 자동으로 인쇄 대기열에 들어간다.

### 1-3 재부팅 (두 번째)
- **프린터**: 전원 끄기 → 30초 대기 → 다시 켜기
- **컴퓨터**: Windows 재부팅
- **순서**: 프린터 먼저, 그 다음 컴퓨터

### 1-4 프린터 "일시 중지" 상태 확인
일부 프린터는 사용자가 의도하지 않게 "일시 중지" 상태에 빠질 수 있다.
- 제어판의 메뉴에서 "일시 중지" 또는 "일시 정지" 상태를 찾아 해제

## 2. 무선(Wi-Fi) 연결 문제 해결

### 2-1 Wi-Fi 표시등 확인
프린터 제어판을 보고:
- **표시등 켜짐** = 네트워크 연결 정상
- **표시등 깜박임** = 연결 대기 중
- **표시등 꺼짐** = Wi-Fi 연결 끊김

### 2-2 Wi-Fi 다시 연결하기 (일반 방법)
1. 프린터 제어판에서 **설정/메뉴 → Wi-Fi 또는 네트워크 설정** 진입
2. 네트워크 목록에서 현재 연결된 라우터명 선택
3. 비밀번호 입력 → 재연결 시도
4. 연결 성공 메시지 확인 후 프린터 재부팅

### 2-3 WPS(Wi-Fi Protected Setup) 연결 (빠른 방법)
일부 HP 프린터와 라우터가 WPS를 지원할 때:

**절차**:
1. 프린터 제어판에서 **WPS 버튼** 누르기 (또는 메뉴에서 WPS 모드 선택)
2. **2분 이내**에 라우터의 WPS 버튼 누르기 (또는 컴퓨터에서 Wi-Fi 설정에서 WPS 연결 선택)
3. 프린터에 "연결 성공" 메시지 표시 → 자동 연결 완료

**장점**: 비밀번호를 입력하지 않아도 됨

### 2-4 네트워크가 자주 끊길 때
- 라우터와 프린터 사이 거리를 3미터 이내로 단축
- 라우터 재부팅 (전원 끄기 → 30초 대기 → 켜기)
- 같은 공간에 다른 무선 기기가 많으면 무선 채널 변경 (라우터 설정)
- 라우터 위치를 높은 곳으로 이동

## 3. 잉크 카트리지 인식 오류

### 3-1 오류 증상
- "카트리지 없음" 메시지
- "호환되지 않는 카트리지" 메시지
- 제어판에서 잉크 레벨 표시 불가

### 3-2 원인
**가장 흔한 원인**: 카트리지의 **금색 접촉부(칩)**에 먼지, 보호 필름 잔여, 또는 오염물 축적

### 3-3 접점 청소 방법

**준비물**:
- 증류수 (또는 깨끗한 물)
- 보풀 없는 부드러운 천 또는 면봉
- 마른 타올

**절차**:
1. 프린터 전원 끄기
2. 카트리지를 프린터에서 제거하기
3. **금색 접촉부**를 보풀 없는 천으로 부드럽게 닦기
   - 물에 살짝 적신 면봉 사용 가능
   - 원형으로 부드럽게 문질러 오염물 제거
4. 프린터 내부의 대응 접촉부도 같은 방식으로 청소
5. **완전히 말린 후** (최소 5분) 카트리지 재삽입
6. 프린터 전원 켜고 잉크 레벨 인식 확인

### 3-4 주의사항 ⚠️
- **절대 금지**: 잉크 노즐 부분 청소 (카트리지 끝 부분의 구멍)
- **절대 금지**: 손으로 구리판 직접 접촉 (정전기 손상 위험)
- **절대 금지**: 세제·알코올·화학약품 사용
- 처음 설치한 카트리지라면 보호 필름을 완전히 제거했는지 확인

### 3-5 문제 지속 시
- 다른 신규 카트리지로 테스트해 보기 (현재 카트리지 불량 가능)
- 카트리지를 정확한 위치에 끝까지 밀어 넣었는지 재확인
- HP 공식 카트리지 사용 권장 (호환 카트리지는 칩 호환성 문제 가능)

## 4. 인쇄 품질 불량 — 헤드 청소 & 정렬

### 4-1 인쇄 품질 문제 증상
- 줄무늬가 생김
- 색상이 옅거나 누락됨
- 텍스트가 흐릿함
- 특정 색만 잘 안 나옴

### 4-2 프린트 헤드 청소

**방법 1 — 제어판 사용 (권장)**:
1. 흰 A4 용지를 입력함에 몇 장 넣기
2. 프린터 제어판에서 **설정 → 도구 → 프린트 헤드 청소** 진입
3. 화면 지침을 따라 진행 (대개 1~2분 소요)
4. 자동으로 테스트 페이지 인쇄되어 결과 확인

**방법 2 — HP Smart 앱 사용**:
1. HP Smart 앱 실행 → 프린터 선택
2. **인쇄 품질 도구** 또는 **유지보수** 메뉴 진입
3. **"프린트 헤드 청소"** 또는 **"카트리지 청소"** 선택
4. 앱 화면 지침 따르기

### 4-3 청소 후 인쇄 품질이 나아질 때까지의 절차
1. 첫 번째 청소 → 인쇄 테스트
2. 결과가 여전히 안 좋으면 **30분 대기** (프린터 작동 중단)
3. 30분 후 **정렬(alignment) 작업 진행** (아래 참고)

### 4-4 프린트 헤드 정렬

**제어판 사용**:
1. 프린터 제어판 → **설정 → 도구 → 카트리지 정렬** 또는 **프린트 헤드 정렬** 선택
2. 화면 지침 따르기 (대개 2~3분 소요)
3. 정렬용 패턴 페이지 자동 인쇄

**결과**: 정렬 후 인쇄하면 색상이 정확히 겹쳐 나온다.

### 4-5 청소 횟수 제한 ⚠️
프린트 헤드 청소할 때마다 **잉크가 소비**된다.
- 권장: **불필요한 청소는 자제**
- 필요할 때만 실행 (인쇄 품질 저하가 확실할 때)
- 과도한 청소는 잉크 낭비 + 헤드 부품 수명 단축

## 5. 용지 걸림 제거

### 5-1 안전한 제거 방법
1. **프린터 전원 끄기** (중요!)
2. 프린터 문 또는 액세스 패널 열기
3. 걸린 용지를 **양손으로 잡고 천천히 당기기**
   - 수평으로 부드럽게 빼내기 (찢지 말 것)
   - 급하게 당기거나 빠르게 움직이면 부품 손상
4. 모든 용지 조각 제거 (파편이 내부에 남으면 다음 인쇄 시 문제)
5. 프린터 전원 다시 켜기

### 5-2 내부 깊숙한 용지
- 부드럽게 다양한 방향에서 당겨 보기
- 강제로 빼려고 하지 말 것 (롤러, 분리판 손상 위험)
- 쉽게 안 빠지면 고객센터 연락

### 5-3 용지 걸림 예방
- 입력함에 권장 용량 이상 용지 넣지 않기
- 새 용지를 추가할 때는 기존 용지를 모두 제거한 후 정돈하여 추가
- 구기거나 손상된 용지 사용 금지
- 습기가 많은 환경에서 용지보관 금지 (팽창하여 걸림)

## 6. 초기화 및 드라이버 문제

### 6-1 콜드 리셋 (전원 사이클링)

**증상**: 프린터가 응답하지 않거나 화면이 먹통일 때

**방법**:
1. 프린터 전원 플러그를 뽑기
2. **1~2분 기다리기** (프린터 내부 전자회로 방전)
3. 전원 플러그를 다시 연결
4. 프린터 전원 버튼으로 켜기
5. 부팅 완료 대기 (제조사 로고 표시 → 대기 화면)

**더 강한 리셋**:
1. 프린터 전원 끄기
2. 프린터 전원 버튼을 **10초 이상 길게 누르기**
3. 손 떼기
4. 프린터 전원 켜기

### 6-2 드라이버 재설치

**상황**: 인쇄는 되지만 프린터 매뉴얼 기능 사용 불가, HP Smart 앱 연결 안 됨

**완전 제거 → 재설치 절차**:

**Step 1: 기존 드라이버 완전 제거**
1. Windows 제어판 → **프로그램 및 기능** 진입
2. HP 프린터 관련 프로그램 모두 찾기:
   - "HP [모델명] 드라이버"
   - "HP Smart"
   - "HP Print and Scan Doctor"
3. 각 항목 선택 → **제거** 클릭
4. Windows 제어판 → **장치 및 프린터** 진입
5. 프린터를 우클릭 → **장치 제거**
6. 체크박스 "드라이버 소프트웨어도 삭제" 체크 → 제거
7. **컴퓨터 재부팅** (매우 중요!)

**Step 2: 새 드라이버 설치**
- **방법 A (공식 권장)**: HP 공식 웹사이트(support.hp.com/kr-ko) 접속 → 프린터 모델명 검색 → 최신 드라이버 다운로드 → 설치
- **방법 B (자동 설치)**: Windows 장치 관리자에서 프린터 우클릭 → "드라이버 업데이트" → "자동으로 검색"

**Step 3: 설치 후**
1. 설치 마법사 완료
2. 컴퓨터 **다시 재부팅**
3. HP Smart 앱 재설치 (필요시)
4. 인쇄 테스트

## 7. HP Smart 앱 & 진단 도구

### 7-1 HP Smart 앱 기능
- 프린터 모니터링 (잉크 잔량, 토너 상태)
- 원격 인쇄 (클라우드 프린팅)
- 인쇄 품질 도구 (헤드 청소, 정렬)
- 프린터 상태 진단
- 지원 요청 및 고객센터 연결

**지원 기기**: Windows, macOS, iOS, Android
**다운로드**: Microsoft Store, Apple App Store, Google Play 또는 support.hp.com

### 7-2 HP Print and Scan Doctor (Windows 전용 진단 도구)
무료 다운로드 프로그램으로 다음을 자동 진단한다:
- USB/Wi-Fi 연결 상태
- 드라이버 충돌
- 포트 설정 오류
- 네트워크 연결성

**사용 방법**:
1. HP 공식 웹사이트에서 다운로드
2. 실행 → "시작" 클릭
3. 프린터 선택
4. 자동 진단 → 결과 및 해결 방안 표시

## 8. 문제 해결 의사 결정 트리

```
인쇄 안 됨
├─ USB 연결 확인? → 케이블 재정렬
├─ Wi-Fi 표시등 확인? → Wi-Fi 재연결 또는 WPS 연결
├─ 오프라인 상태 해제? → Windows "프린터를 오프라인으로 사용" 해제
├─ 프린터 재부팅? → 전원 끄기 → 30초 → 켜기
├─ 드라이버 문제? → HP Print and Scan Doctor 실행
└─ 여전히 안 됨? → 고객센터 연락

잉크 인식 안 됨
├─ 카트리지 접점 청소 (증류수 + 면봉)
├─ 카트리지 재삽입 (완전히 밀어 넣기)
└─ 다른 카트리지로 테스트

인쇄 품질 나쁨
├─ 프린트 헤드 청소
├─ 30분 대기
└─ 프린트 헤드 정렬
```
