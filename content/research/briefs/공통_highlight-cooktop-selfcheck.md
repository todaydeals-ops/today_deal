# 하이라이트(전열식) 전기레인지 가열 불량·에러 표시·자가점검

## 조사 개요

하이라이트 전기레인지는 **전열식(전구 원리)** 가열 방식으로, 인덕션의 자기유도와는 완전히 다르다. 사용자들이 가장 헷갈리는 지점은 **"인덕션은 냄비만 뜨겁고 상판은 미지근하지만, 하이라이트는 상판 자체가 뜨겁다"**는 차이다.

조사 출처:
- **삼성전자서비스 공식** (전열식 가열 원리, 용기 요구사항)
- **LG 스스로해결 공식** (에러코드, 잔열 표시, 화력 약함 원인)
- **사용자 커뮤니티 및 상용 청소 가이드** (상판 청소, 당분류 위험, 화상 위험)

---

## 핵심 팩트

### 1. 하이라이트 가열 원리 — 인덕션과의 근본적 차이

**하이라이트 (전열식)**
- 전구와 같은 원리로 히터가 전기저항으로 **직접 열 발생**
- 열이 유리 상판을 통해 용기에 전달 (전도열)
- **상판 자체가 뜨거움** → 조리 직후 식을 때까지 기다려야 함
- 화력 낮음 (설계상 특성)
- **모든 용기 재질 사용 가능** (철·스테인리스·알루미늄·유리·도자기·세라믹 모두 OK)

**인덕션 (자기유도식)**
- 코일에서 생긴 **자기장이 냄비 바닥을 유도 가열**
- 냄비만 뜨겁고 상판은 미지근함 → 조리 직후 즉시 청소 가능
- 화력 강함
- **자성 재질만 사용 가능** (철제·법랑·일부 스테인리스, IH 마크 필수)
- 용기 크기·위치·자성 강도가 중요

**결론**: 인덕션 팩트 중 "용기 재질 제약", "자석 테스트", "IH 마크", "가열지수", "냄비 두께·소음"은 하이라이트에 적용 불가. 반대로 "상판 균열 위험", "조리 후 열 관리"는 공통.

---

### 2. 잔열 표시등 (H / h)

**[H] 표시**
- 의미: 상판이 **뜨거운 상태**
- 해제: 전원 버튼을 1초간 누르면 사라짐
- 주의: 표시가 사라져도 **실제로는 뜨거울 수 있음**

**[h] 표시 (소문자)**
- 의미: 상판이 **따뜻한 상태**
- 해제: 마찬가지로 전원 버튼 1초
- 주의: 마찬가지로 실제 온도가 남아있을 수 있음

출처: LG 스스로해결 공식 문서

---

### 3. 에러코드 해석

**[E] 또는 [E1]** (버튼 오작동)
- 원인: 버튼을 9초 가까이 눌렀거나, 버튼에 음식물·기름·물이 묻음
- 해결: 전원을 끄고 해당 버튼을 마른 천으로 닦기
- 예방: 조리 시 기름·물 튀김 주의, 정기적 버튼 청소

**[E6]** (냉각팬 과열)
- 원인: 냉각팬 센서가 과열됨
- 해결: 전원을 끄고 충분히 식힌 후 재시작
- 예방: 통풍구 정기적으로 청소

**[P]** (일시 정지)
- 의미: 일시 정지 기능이 활성화됨
- 해결: 일시 정지 버튼 다시 누르기

**[U]** (용기 감지 불가)
- 의미: 상판에 용기가 없거나 위치가 잘못됨
- 해결: 적절한 용기를 중앙에 올려놓기
- **주의**: LG 공식 문서에서 "IH 마크 필수"라고 언급했는데, 이는 인덕션 설명일 가능성 높음. 하이라이트의 U 표시 기능 여부는 모델별로 다를 수 있으므로 개별 매뉴얼 확인 필요.

---

### 4. 용기 요구사항

**하이라이트 용기의 2가지 필수 조건**

1. **바닥이 평평할 것** (매우 중요)
   - 상판과의 접촉 면적이 클수록 열 전달 효율 증가
   - 바닥이 휘거나 볼록하면 가열 불량
   - 버너 크기에 맞을 것: 작은 용기 사용 시 노출된 상판으로 열이 손실됨

2. **재질은 무관** (인덕션과의 최대 차이)
   - 철제, 스테인리스, 알루미늄, 유리, 도자기, 세라믹 모두 사용 가능
   - IH 마크 불필요
   - 자석 테스트 불필요

---

### 5. 상판 청소와 위험

**청소 절차**
1. 조리 후 전원 버튼 1초로 전원 차단
2. **완전히 식을 때까지 대기** ← 인덕션과의 가장 큰 차이
3. 전용 세정제 또는 Schott 인증 세제 사용
4. 유리 전용 수세미만 사용
5. 마른 천으로 마무리

**위험 1: 당분류 눌어붙음 (균열 위험)**
- 설탕, 물엿, 캐러멜, 꿀이 상판에 흘렀을 때 **굳기 전에 즉시 닦기**
- 굳어버린 당분이 냄비 바닥에 붙으면 균열 위험
- **응급 처리**: 레인지를 약 1~2분 작동해 당을 녹인 후 자연 분리 (억지로 떼면 균열 생김)

**위험 2: 화상 위험**
- 전원 버튼을 눌렀다고 해서 상판이 즉시 식지 않음
- 특히 어린이, 반려동물 주의
- 외관상 불이 꺼져 있어도 뜨거움

**위험 3: 상판 균열 (감전 위험)**
- 상판에 균열이 생기면 수분과 음식물이 내부 전자부품으로 침투
- 감전 및 합선 위험 발생
- **즉시 전원 차단 후 서비스센터 상판 교체 의뢰**

---

### 6. 가열 불량 / 약함 자가점검

**하이라이트의 화력이 약한 이유**
- 설계상 특성 (상판 온도 유지 방식)
- 인덕션보다 낮은 효율은 **정상**이며 고장이 아님

**체크 사항**
1. 용기 바닥이 평평한가? → 휜 냄비 사용 중이면 교체
2. 버너 크기에 맞는 용기인가? → 너무 작은 용기 사용 시 열 손실
3. 버너가 켜졌다 꺼졌다를 반복하는가? → 상판 온도 유지 정상 작동
4. 전원이 제대로 공급되는가? → 전원 콘센트·차단기 확인

**일반 에러 리셋**
- 전원 플러그 제거 후 2~3분 경과
- 다시 연결해 정상 작동 확인

---

### 7. 차일드락 (공식 문서 미발견)

**현황**: 삼성전자서비스·LG 스스로해결 공식 문서에서 하이라이트 전용 차일드락 해제 절차가 명시되지 않음.

**대응**:
- 모델별 매뉴얼에서 확인 필요
- 고객센터 문의 (삼성 1588-3366 / LG 1544-7777)
- 인덕션 차일드락은 자물쇠 버튼 3초 (하이라이트와 다를 수 있음)

## 거부 팩트 (declined_facts)

**인덕션 전용 팩트 9건** — 가열 원리 완전 다름
- 용기 재질 (철·법랑·스테인리스만) → 하이라이트는 모두 가능
- 자석 테스트 → 하이라이트는 자성 무관
- IH 마크 → 하이라이트에 필요 없음
- 가열지수 (LG만) → 하이라이트는 개념 자체 없음
- 버너 중심 정렬 → 인덕션의 자기장 특성, 하이라이트는 덜 중요
- 공냄비 금지 → 인덕션은 냄비 손상 위험, 하이라이트는 다른 문제
- 냄비 두께·소음 → 인덕션의 고주파 특성
- 가열지수 확인 버튼 조합 → 인덕션만

---

## 결론

하이라이트 조사는 **"인덕션과의 차이"**에 집중했다. 사용자가 겪는 혼란은 대부분:
1. "하이라이트도 냉한 용기만 쓸 수 있나?" → 아니다, 모든 재질 OK
2. "왜 화력이 약하나?" → 설계상 특성, 정상
3. "조리 직후 청소할 수 있나?" → 아니다, 반드시 식혀야 함

이 세 가지를 명확히 하면 자가점검 글의 기본 골격이 세워진다.
