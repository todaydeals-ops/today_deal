# LG 냉장고 에러 표시·냉각 불량 자가점검 — 조사 브리프

**조사 일시:** 2026-07-20  
**출처:** LG전자 공식 고객지원  
**대상:** LG 냉장고 전 모델(디오스 등)

---

## 요약

LG 냉장고의 에러 표시 16가지, 냉각 불량 시 자가점검 항목 7가지, 스마트진단(씽큐) 사용법, 설치·정전 후 냉각 도달 시간을 LG 공식 고객지원 페이지에서 확인했습니다. 삼성 냉장고 팩트와는 달리, LG 에러코드는 팬 모터·센서·통신 계층으로 명확히 분류되며, 재상할 수 있는 소재와 전문 점검이 필요한 것이 구분되어 있습니다.

---

## 에러코드 체계

LG 냉장고는 **16가지 주요 에러코드**를 공식으로 안내합니다. 이는 삼성의 계층별 혼동(dC 국내=도어감지 vs 해외=탈수불균형)과 달리, 부위별로 명확하게 분류됩니다.

### 팬 모터 에러
- **IF**: 제빙실 팬모터 이상
- **FF**: 냉동실 팬모터 이상 (성에 제거로 일시 완화 가능)
- **rF**: 냉장실 팬모터 이상 (성에 제거로 일시 완화 가능)
- **CF**: 기계실 팬모터 이상

### 센서 에러
- **FS/rS**: 냉동/냉장 센서 이상
- **dS**: 제상 센서 이상
- **HS**: 습도 센서 이상
- **IS**: 제빙 센서 이상
- **SS**: 팬트리 센서 이상
- **rt**: 외기 센서 이상

### 통신 에러
- **CO**: 메인보드와 디스플레이 PCB 간 통신 이상
- **Od**: WiFi 모뎀 통신 이상

### 제상·기타
- **dH**: 제상 불량 (온도퓨즈/제상히터/PCB 단선·불량)
- **gF**: 유량 모터/물입력 이상
- **It**: 제빙기 이상
- **OFF**: 제품 판매 전 특수 기능 모드

### 점검 코드
- **CH/CL**: 설치 24시간 후 냉각 약함 감지(전문 점검 권장)

**주의:** 대부분 전문 서비스가 필요하지만 FF·rF·dH는 성에/냉기배출구 막힘이 원인이면 자가 대처 후 개선될 수 있습니다.

## 냉각 불량 시 자가점검 항목 (우선순위 순)

LG는 냉장고 냉동·냉장이 약할 때 **7가지 확인 항목**을 공식으로 제시합니다. 글에서는 다음 순서를 권장합니다.

### 1. 온도 설정 확인
- **냉동실:** 기본은 영하 -16℃~-17℃(약함) → 불충분하면 -22℃~-24℃ 조정
- **냉장실:** 기본은 3℃ → 여름철 문 개폐 증가 시 한두 단계 낮춤

### 2. 도어 패킹(고무 패킹) 상태
- 문이 닫히지 않으면 바깥 공기 유입 → 냉기 약화
- **청소 방법:** 4면을 깨끗이 닦고, 따뜻한 수건으로 2-3회 반복 닦기
- **밀착 확인:** 청소 후 최소 2시간 동안 문을 열지 않고 대기
- **교체 판단:** 부풀어 오르거나 탄성이 없으면 교체 필요

### 3. 환기 간격 확인 (설치 환경)
- **뒷면:** 벽으로부터 10cm 이상 간격
- **옆면:** 벽으로부터 5cm 이상 간격
- **기계실 덮개:** 먼지 제거 필수
- **설치 위치:** 온도 5℃~43℃ 범위(베란다 등 외부 설치 피할 것)

**함정:** 환기 부족 → 압축기 과열 → 냉각 약화. 뒷면 먼지만 해도 개선될 수 있습니다.

### 4. 냉기 배출구 확인
- 냉동실 안쪽 벽면에 **얼음(성에) 누적** 확인
- 성에가 출구를 막으면 냉기 순환 방해
- **제거 방법:** 도어 열어 최소 8시간 이상 방치 → 수건으로 물기 제거

### 5. 적재량 확인
- 냉장실: **60% 정도만 채우기**
- 과도한 식품 → 냉기 순환 방해 + 전력 소비 증가

### 6. 문 닫힘 확인
- 선반 위치, 과도한 수납물 확인
- 물이 반복적으로 들어갈 수 있으므로 2~3회 점검

### 7. 실내 온도 확인
- 여름철 고온 환경이거나 외부 설치 시 냉각력 저하
- 베란다·현관 설치는 피할 것

## 설치 및 정전 후 냉각 회복 시간

### 신규 설치/재설치 후
- **냉기 시작:** 1~2시간 후 약한 냉기 방출 시작
- **완전 냉각:** 2~3일 소요(계절·환경에 따라 변동)
- **확인 방법:** 2시간 경과 후 물을 넣어 살얼음 형성 확인 → 2~3일 후 냉기 정상 여부 재확인
- **주의:** 초기 냉기 약함은 정상. 식품은 충분한 시간 후 넣기

### 정전/전원 차단 후 복구
1. **성에 제거:** 전원 분리 → 도어 완전히 열기 → 최소 24시간 방치
2. **물기 제거:** 마른 수건으로 내부 물기 닦기
3. **전원 재연결:** 재연결 후 냉기 정상화까지 24~48시간 소요

**중요:** 성에가 발생한 상태에서 바로 전원을 켜면 FF/rF 에러가 나올 수 있습니다.

## LG ThinQ 스마트진단(씽큐)

### 지원 범위
- **스마트 진단 로고가 있는 냉장고만** Wi-Fi 진단 가능
- 지원하지 않는 기기는 **소리 기반 진단**(리모콘 신호음 패턴 분석)
- 모델·지역·앱 버전에 따라 지원 기능 상이

### 사용 절차
1. LG ThinQ 앱 실행
2. 하단의 **"메뉴"** 탭 → **"스마트 진단"** 선택
3. 제품 선택
4. 화면 안내에 따라 진단 진행
5. 결과 확인: 제품 상태·증상·해결책 제시
6. 필요 시 고객센터 연결

### 에어컨과의 차이
- LG 에어컨(스탠드/벽걸이/시스템) 자가진단과 달리, 냉장고 ThinQ 스마트진단은 **앱 기반**
- 형태별 버튼 조합 불필요 (에어컨은 리모콘 조합 필요)

## 글 작성 시 주의

### 1. 에러코드 사용 시
- "FF 에러가 나올 수 있지만 대부분 성에가 원인입니다" 같은 서술은 피할 것
- LG 공식은 "FF는 팬모터 이상이고, 성에가 팬을 구속한 경우도 같은 코드로 표시될 수 있다"로 구분
- 즉, **같은 코드, 다른 원인** 가능성을 명시해야 함

### 2. 냉각 불량 절차
- 온도 설정 → 도어 패킹 → 환기 → 성에 → 적재량 순으로 가이드
- 각 단계마다 "만약 이게 아니라면 다음을 확인하세요" 톤으로 순차적 대안 제시
- "압축기 고장이 아닐 가능성 90%"는 근거 없으므로 피할 것

### 3. 시간 수치
- 설치 후 냉각: 1~2시간(냉기 시작) vs 2~3일(완전 냉각) 구분
- 정전 복구: 성에 제거(24시간) + 냉기 정상화(24~48시간) = 총 2~3일 가능
- "보통" "오래 걸릴 수 있다" 같은 애매한 표현 대신 공식 범위 사용

### 4. 자가점검 한계
- FF/rF/dH 에러 중 성에가 아니면 전문 점검 필요
- CH/CL 에러는 자가점검 후 개선 없으면 반드시 서비스 호출
- "시간이 걸리니 기다려보세요"보다 "2~3일 후에도 안 되면 고객센터 문의"로

### 5. 품질 균등
- 냉동실과 냉장실이 모두 약할 때의 절차 표기 필요
- "냉동만 약할 때" vs "냉장만 약할 때" 증상 분기 가능 (하지만 자가점검은 대부분 겹침)

- **재사용:** 0개 (삼성과 체계가 다름)

## 미흡 사항 (추후 보완)

1. **모델별 에러코드 차이:** 공식 페이지에서 "전 모델 공통"이라고 안내했으나, 실제로 구형/신형 모델 간 코드 변경 여부 미확인
2. **디오스 특화 기능:** 김치냉장고(디오스) 고유의 에러코드나 자가점검법 별도 수집 미완료 → 향후 "디오스-specific" 팩트 추가 필요
3. **씽큐 성능:** LG ThinQ 앱이 실제로 냉장고 문제를 얼마나 정확히 진단하는지는 사용자 후기 수준. 공식은 "결과 제시"만 안내
4. **환기 간격 근거:** 왜 뒷면 10cm, 옆면 5cm인지 열역학적 설명 없음 → 공식 수치이지만 과학적 근거는 LG 문서에 없음
