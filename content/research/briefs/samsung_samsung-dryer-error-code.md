# 삼성 건조기 에러코드와 자가점검 조사 전문

**조사일:** 2026-07-20
**조사자:** Claude Code
**주제:** 삼성 건조기(그랑데 AI 등) 에러코드, 청소 및 배수 관리

---

## 1. 모델명 규칙 및 연식 판독

### 모델명 구조
삼성 건조기 모델명은 다음 구조로 구성된다:
- **DV** (카테고리: 건조기)
- **용량** (9kg, 14kg, 16kg, 21kg, 22kg 등)
- **연식** (알파벳: T, A, B, CB, DB 등)
- **기능코드** (4자리 숫자)
- **색상코드** (뒤 2자리 알파벳)

예시: DV20B8710BV = 20kg, B(2022년), 8710(기능), BV(색상)

### 연식 알파벳 규칙
- **T**: 2020~2021년
- **A**: 2021년
- **B**: 2022년
- **CB**: 2023년
- **DB**: 2024년

**출처:** 커뮤니티·리뷰 자료 (삼성전자서비스 문서에서 명시적 규칙 미확인)
**주의:** 에어컨(AF/AR + N/R/T/A/B)과는 다른 규칙. 모델명 위치는 도어 라벨 또는 에너지 라벨.

## 2. 에러코드 체계

삼성 건조기 에러코드는 8가지로 공식 분류되며, 모두 C로 끝나는 형식이다.

| 코드 | 의미 | 조치 |
|---|---|---|
| **5C / SC** | 배수 안 됨 | 배수펌프 점검 |
| **nC2** | 열교환기 내부 커버 미삽입 | 열교환기 내부 커버 삽입 |
| **nC** | 먼지필터 미삽입 | 먼지필터 삽입 |
| **9C2** | 저전압 | 전원 점검 |
| **HC** | 압축기 과열 | 압축기 온도 센서 점검 |
| **dC** | 도어 열린 채 작동 | 도어 폐쇄 후 재시작 |
| **tC5 / tCS** | 압축기 온도 센서 오류 | 센서 점검 |
| **tC** | 공기 온도 센서 오류 | 필터·열교환기 청소 |

**출처:** 삼성전자서비스 [solution/1578050](https://www.samsungsvc.co.kr/solution/1578050) (2023-09-04 갱신)

**계층 혼동 주의:** 에어컨의 E101/C101 형식(구형 E1 = 신형 3자리)과 완전히 다른 체계. 건조기의 nC 형식은 2자리 또는 3자리.

## 3. 먼지필터 청소

### 청소 주기
- **매 건조 사용 후 매회** 청소 필수

### 청소 방법
1. 도어를 열고 먼지필터 위치 확인 (도어 앞 하단)
2. 필터를 **위쪽으로 잡아당겨** 분리
3. 외부 필터 열기
4. 내부 필터 분리
5. 각 필터의 먼지·보풀 제거
6. 흐르는 물에 씻고 **충분히 말려서** 다시 삽입

**출처:** 삼성전자서비스 [solution/41159](https://www.samsungsvc.co.kr/solution/41159)

**주의:** 필터를 젖은 상태로 삽입하거나 청소하지 않으면 tC 에러(온도 센서 오류) 발생 가능.

---

## 4. 열교환기(콘덴서) 청소

### 청소 주기 (모델별)
모델별로 자동 알림 기준이 다르다:

| 용량·모델 | 청소 주기 | 알림 |
|---|---|---|
| **14kg** | 건조 50회마다 | "열교환기 청소" 표시 |
| **16kg** | 건조 100회마다 | "열교환기 청소" 표시 |
| **그랑데 AI 16kg** | 건조 300회마다 | "열교환기 청소" 표시 |

**출처:**
- 14kg: [solution/28529](https://www.samsungsvc.co.kr/solution/28529)
- 16kg: [solution/28175](https://www.samsungsvc.co.kr/solution/28175)
- AI 16kg: [solution/28789](https://www.samsungsvc.co.kr/solution/28789)

### 청소 방법
1. **전원을 꽂은 채로** 청소 진행
2. 건조기 우측 하단의 전면 뚜껑을 눌러 열기
3. 좌, 우 양쪽의 잠금장치 풀기
4. 열교환기 내부 커버를 꺼내기
5. 청소용 브러시로 **위아래로 긁어** 먼지 제거
6. 진공청소기로 제거된 먼지 수집
7. 청소 후 **30분~1시간 환기** (열풍 상태로 건조)
8. 열교환기 내부 커버 재설치

**주의:** 열교환기 청소 중 커버를 바로 설치하지 말고 충분히 환기시킬 것.

## 5. 물통 및 배수 관리

### 물통(응축수 저장)
- **매 건조 후 매회** 물통 비우기
- 패널에 "물 비움" 알림 표시되면 즉시 비우기

### 직배수 설치 시 호스 점검
직배수를 사용하는 경우:
1. 배수호스 **꺾임** 확인
2. 배수호스 끝부분이 **물에 잠겨 있지 않은지** 확인
3. 하수구에 호스가 **너무 깊이 박혀 있지 않은지** 점검

**출처:** 삼성전자서비스 [solution/41102](https://www.samsungsvc.co.kr/solution/41102), [solution/41189](https://www.samsungsvc.co.kr/solution/41189)

## 6. 건조 시간이 길어질 때 점검 항목

### 주요 원인
1. **주변 온도**: 추운 환경에서는 드럼 내부 온도가 천천히 올라가 건조 시간 증가
2. **옷의 탈수 부족**: 세탁 후 탈수 강도가 약하면 건조 시간 증가
3. **먼지필터 막힘**: 필터에 먼지가 쌓여 공기 순환 저해
4. **열교환기 막힘**: 열교환기에 먼지가 쌓여 열 교환 효율 저하

### 자가점검 절차
1. **세탁기 탈수 재설정**
   - 건조 전 세탁기 탈수 옵션을 **최강**으로 설정
   - 또는 탈수를 **한 번 더** 진행
   
2. **먼지필터 청소**
   - 필터를 깨끗이 청소
   - **충분히 말려서** 삽입 (습한 상태 금지)
   
3. **열교환기 청소**
   - 청소 주기(14kg 50회, 16kg 100회 등) 확인
   - 알림이 없어도 3~6개월에 한 번 정기 청소 권장
   
4. **온도 환경 개선**
   - 실내 온도 20℃ 이상 유지
   - 건조기 주변 환기 확인

**출처:** 삼성전자서비스 [solution/41189](https://www.samsungsvc.co.kr/solution/41189)

## 7. 에어컨 팩트와의 차이 (층위 혼동 방지)

**⚠️ 중요: 에어컨과 건조기는 완전히 다른 규칙이다.**

| 항목 | 에어컨 | 건조기 |
|---|---|---|
| **모델명 시작** | AF(스탠드), AR(벽걸이) | DV |
| **연식 알파벳** | N(2018), R(2019), T(2020), A(2021), B(2022) | T(2020~), A(2021), B(2022), CB(2023), DB(2024) |
| **에러코드 형식** | E/C + 숫자 (E101, C101) | 숫자/알파벳 + C (5C, nC, tC) |
| **자가진단 진입** | 리모컨 메뉴 또는 버튼 조합 | 자가진단 메뉴 없음 (에러 표시만 제공) |

기존 에어컨 팩트(`samsung-model-year-alphabet`, `samsung-errorcode-digit-change` 등)는 **건조기에 적용되지 않는다.**
