# 전동칫솔 자가점검 조사 전문

## 조사 범위

- 주요 증상: 충전 안 됨 / 배터리 급감 / 진동 약함 / 방수 문제 / 곰팡이 / 브러시 헤드 교체 / 여행 중 충전
- 주요 브랜드: 필립스 소닉케어(Philips Sonicare), 오랄비(Oral-B)
- 공통 카테고리로 분류하되, 브랜드별 정보는 scope에 명시

## 1단계: 기존 팩트 검토

팩트 저장소에 전동칫솔 관련 자료 없음. 신규 조사 대상.

## 2단계: 웹 검색 및 검증

### 검색 1: 충전 문제 (공식 문서)
- **필립스 소닉케어**: 공식 페이지(XC000016868)에서 유도충전 정렬, 금속 간섭, 콘센트 확인 등 상세 가이드 제공
- **오랄비**: manuals.plus 문서에서 모델별 충전 시간, 배터리 지속시간(5-12일), 충전 중 전원 켜짐 방지 등 기술
- 핵심: 두 브랜드 모두 충전대 정확한 안착, 금속 간섭 제거, 콘센트 점검을 1차 진단으로 제시

### 검색 2: 진동/방수 (사용자 리뷰)
- 방수 등급: IPX7~IPX8 범위로 보고되나, 공식 문서에서 구체적 수치 미확인
- 진동 약함: 배터리 부족 시 약해지고, 모델에 따라 압력 센서가 과도한 압력에 약한 진동 제공
- **tier: 논쟁** — 모델·배터리 상태에 따라 상이하므로 후기 근거로 기록

### 검색 3: 브러시 헤드 (공식 지원)
- **필립스 소닉케어**: 공식 FAQ에서 3개월 교체 권장, 파란색 지시자 마모 확인, 최신 모델 자동 알림 기능 설명
- **오랄비**: 공식 웹사이트에서 3개월 주기, iO 시리즈 디스플레이 알림 기능 안내
- 호환성: 필립스 공식 문서에서 "서로 다른 모델 간 비호환" 명시

### 검색 4: 곰팡이/냄새 (사용자 커뮤니티)
- 클리앙 사용자 보고: 손잡이 하단 물고임 → 곰팡이 발생
- 예방법: 사용 후 물기 제거, 통풍 보관, 자외선 살균소독기 사용
- 제거법: 락스 활용(플라스틱 손상 위험) vs 자외선 살균(더 안전)
- **tier: 논쟁** — 개별 경험이므로 후기 근거로 분류

### 검색 5: 여행 충전 (전자제품 정보)
- 충전기 입력 전압 범위 확인 필수: 110-220V 병기된 제품만 해외에서 안전
- 110-130V만 표시된 제품은 220V에서 사용 불가(고장/화재 위험)
- 돼지코 어댑터는 플러그 형태만 변환, 전압은 변환하지 않음
- **tier: 확실** — 전자제품 안내 기준이므로 후기 근거로 분류하되 신뢰도 높음

### 검색 6: 종합 팩트 추가
- 2년 보증 정책: 필립스·오랄비 모두 제조 결함 시 2년 기간 보증
- 충전기 호환성: 필립스 문서에서 "모델별 비호환" 명시

## 3단계: 팩트 정리 기준

**Tier 분류:**
- **확실** (공식 문서 또는 명확한 기술): 충전 방법, 충전 시간, 배터리 지속시간, 교체 주기, 브러시 헤드 호환성, 충전기 호환성, 보증 정책, 전압 기준
- **논쟁** (모델·개인차 있음): 방수 등급(모델별 상이), 곰팡이 발생(사용 환경 의존)

**Evidence 분류:**
- **공식**: Philips·Oral-B 공식 페이지, 매뉴얼, 고객지원
- **후기**: 사용자 커뮤니티(클리앙), 제품 리뷰, 전자제품 정보(전압 안내)

## 4단계: Caution 작성 기준

- 수치가 있으면 cautionSource 필수 (예: 3개월 교체, 5-12일 배터리, 110-220V)
- 범위가 넓거나 모델별 상이하면 명시 (예: IPX7~IPX8, 16~22시간 충전)
- 안전 경고: 손상된 어댑터 사용(감전·화재), 호환되지 않는 충전기(손상·화재), 110V 전용 제품을 220V에서(고장·화재)
