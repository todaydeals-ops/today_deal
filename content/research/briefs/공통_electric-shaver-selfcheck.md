## 2. 회전식 vs 왕복식 기계식 차이

### 2-1. 기구적 차이

**회전식 (필립스)**:
- 날이 원을 그리며 **회전**
- 포일 아래 원통형 날 3개가 회전
- 원형 헤드

**왕복식 (브라운·파나소닉)**:
- 날이 **좌우로 왕복** 진동
- 포일 아래 칼날이 수평 왕복
- 직사각형 헤드

### 2-2. 사용 특성 — 신체 반응 차이

**회전식 (필립스)**:
- 피부 자극 **적음** → 민감한 피부 사용자에게 유리
- 적응 시간 필요 (회전 방식에 익숙해져야 함)
- 건식 면도(폼/젤 없음) 가능하나, 아기름 있을 때는 피부가 밀려서 효과 저하
- **결론**: 예민한 피부, 적응 시간 감당 가능한 사용자 추천

**왕복식 (브라운·파나소닉)**:
- 날면도기와 **비슷한 방식** → 익숙하고 빠른 적응
- 굵고 빠르게 자라는 수염을 **더 깊게 깎음**
- 성능상 우수성이 높다는 평가
- **결론**: 깊은 면도, 수염 굵은 사용자 추천

### 2-3. 헤드 움직임 각도 비교

**웹 조사 결과**:
- 파나소닉 LS9AX: 상하 약 26도, 좌우 약 24도 (가장 넓음)
- 필립스 3000: 상하 10도 (가장 제한적)
- 브라운: 중간 수준

## 3. 물세척 가능 여부 — 필립스 공식 아이콘 체계

### 3-1. 필립스 공식 아이콘 확인 방법

**출처**: Philips 공식 고객지원
- https://www.philips.co.kr/c-f/XC000004291/%ED%95%84%EB%A6%BD%EC%8A%A4-%EB%A9%B4%EB%8F%84%EA%B8%B0%EC%97%90-%EB%82%98%ED%83%80%EB%82%98%EB%8A%94-%EA%B8%B0%ED%98%B8%EB%8A%94-%EB%AC%B4%EC%97%87%EC%9D%84-%EC%9D%98%EB%AF%B8%ED%95%A9%EB%8B%88%EA%B9%8C

핸들 뒷면에 인쇄된 **아이콘**으로 판단:

| 아이콘 | 의미 | 세척 방법 |
|---|---|---|
| 수도꼭지 | 물세척 가능 | 물로 헹굼 가능, 샤워/목욕 중 사용 불가 |
| X표시 수도꼭지 | 물세척 불가 | 건식 청소만 가능 |
| 욕조 | 완전 방수 | 샤워/목욕 중 사용, 물세척 모두 OK |

### 3-2. 물세척 방법 (방수 모델)

**출처**: Philips 공식 고객지원
- https://www.philips.co.kr/c-f/XC000004469/%ED%95%84%EB%A6%BD%EC%8A%A4-%EB%A9%B4%EB%8F%84%EA%B8%B0%EB%8A%94-%EC%96%B4%EB%96%BB%EA%B2%8C-%EC%B2%AD%EC%86%8C%ED%95%A9%EB%8B%88%EA%B9%8C

**매번 면도 후 (간단 청소)**:
1. **전원을 켠 상태**로
2. 쉐이빙 유닛 전체를 **따뜻한 흐르는 물에 헹굼**
3. 전원 끔
4. 필요시 유닛을 열어 내부도 헹굼

**월 1회 (깊은 청소)**:
1. 쉐이빙 헤드를 분리
2. 안쪽날과 바깥날을 **따로 세척**
3. 완전히 자연 건조
4. 완전히 건조한 후 조립

### 3-3. 건식 면도기 청소 (X표시 수도꼭지)

- 전원을 끈 상태에서
- 상단을 **불어서** 털 제거
- 또는 **제공된 청소용 브러시** 사용

### 3-4. 조립 주의사항

물세척 후 재조립할 때 **구조 오류가 소음의 주요 원인**:

- 각 커터와 보호대를 올바르게 짝짓기
- 면도날이 **아래를 향하도록** 배치
- 안쪽날을 바깥날 안에 넣기
- 고정 링(또는 지지대)으로 **단단히 고정**

올바르지 않으면 사용 중 덜컹거리는 소리 발생.

## 4. 세척 스테이션 — 브라운과 필립스

### 4-1. 브라운 세척스테이션

**특징**:
- 선택 사항 액세서리 (일부 모델에만 제공)
- 자동 세척·소독·윤활 동시 처리
- 면도 후 면도기를 넣고 청소 버튼 누르기만 하면 됨

**작동 원리**:
- 면도기 본체 모터가 작동하며 세척
- **세척스테이션 소음 ∝ 면도기 소음**
- 내부 잔여물과 미세한 털 조각 제거

### 4-2. 필립스 SmartClean / 무선 클렌징 팟

- 정기적 청소를 간편하게 하는 옵션
- 모델별 호환성 확인 필요
- 공식 페이지에서 호환 정보 확인 가능

## 5. 소음 증가 원인 및 해결

### 5-1. 원인 1: 잘못된 조립 ⭐ **가장 흔함**

**출처**: Philips 공식 고객지원
- https://www.philips.co.kr/c-t/XC000020041

**증상**:
- 면도기를 켤 때 갑작스러운 큰 소음
- 사용 중 덜컹거리는 소리

**원인**:
- 세척 후 재조립 오류
- 각 커터와 보호대 짝짓기 실수
- 면도날 방향 역배치
- 고정 링 미흡

**해결**:
- 제품 사용 설명서 확인
- 공식 조립 영상 참고
- 올바르게 재조립

### 5-2. 원인 2: 날 마모

**증상**:
- 점차 소음 증가 (갑작스럽지 않음)
- 동시에 면도 성능 저하

**원인**:
- 면도망과 면도날 마모
- 권장 교체 주기 경과

**해결**:
- **새로운 헤드로 교체** → 소음과 성능 모두 회복

### 5-3. 원인 3: 세척 부족

**증상**:
- 장기간 세척하지 않으면 쉐이빙 헤드 주변 이물질 축적
- 성능 저하 + 소음 증가 가능

**해결**:
- 정기적 세척 (매번 + 월 1회 깊은 청소)

## 6. 피부 자극 및 당김 현상

### 6-1. 원인 1: 강한 압력 ⭐ **가장 흔함**

**잘못된 사용법**:
- 면도기를 피부에 **세게 누르기**
- 피부가 눌리면서 자극 발생
- 오히려 면도 효과 저하

**올바른 사용**:
- 피부에 **부드럽게 스치듯** 사용
- 압력 없이 가볍게 접촉

### 6-2. 원인 2: 날 마모

**증상**:
- 당김 현상 심화
- 수염이 제대로 잘리지 않고 **잡아 뜯어지는 느낌**

**원인**:
- 날이 예리하지 않음
- 진동 수 저하

**해결**:
1. 먼저 **완전 충전** 확인 (배터리 부족시 진동 저하)
2. 그래도 안 되면 **날 교체**

### 6-3. 원인 3: 건식 면도 + 회전식 조합

**회전식(필립스) 특수성**:
- 아기름이 너무 많으면 피부 표면이 **밀려서** 면도가 제대로 안 됨
- 헤드 접촉면이 넓으나 실제 면도 영역은 극히 일부

### 6-4. 해결책: 윤활

**가장 효과적인 방법**:
1. **세안 후 물기가 조금 남은 상태**에서 면도
2. **프리셰이브 로션** 사용
3. **쉐이빙 폼/젤** 사용 (물면도)
   - 윤활 성분이 날과 피부 사이 마찰 크게 감소
   - 피부 자극 **대폭 감소**

**적응 기간**:
- 날면도기 사용자가 전기면도기로 전환하면 2~3주 적응 기간 필요

## 7. 충전 불량

### 7-1. 초기 진단 (공통)

**출처**: Philips 고객지원, Oral-B 공식 매뉴얼

**확인 순서**:

1. **충전 어댑터 손상 확인**

3. **완전 충전 시도**
   - 충전기가 정상이면 충전 진행

## 조사 소결

### 핵심 인사이트

1. **"안 깎인다" ≠ 고장**
   - 필립스 24개월, 브라운 18개월이 정상 수명
   - 마모 증상이 나타나면 날 교체로 해결

2. **회전식과 왕복식은 다름**
   - 회전식: 피부 자극 적음, 민감한 피부 추천
   - 왕복식: 깊은 면도, 수염 굵은 사용자 추천
   - **관리법도 다름** → 공식 매뉴얼 필독

3. **물세척은 아이콘으로 판단**
   - X표시 수도꼭지 → 물세척 금지
   - 수도꼭지 → 물세척 가능, 샤워 사용 불가
   - 욕조 → 완전 방수

4. **피부 자극은 윤활로 90% 해결**
   - 압력을 빼고, 윤활 성분을 더한다
   - 프리셰이브 로션/쉐이빙 폼이 가장 효과

5. **소음 = 조립 오류 또는 날 마모**
   - 갑작스러운 소음 → 조립 재확인
   - 점진적 소음 증가 → 날 교체 시기

## 미확인 사항 (재사용 팩트에서)

전동칫솔 조사에서 이미 확보한 다음 항목들은 **팩트로는 재사용하지 않았으나**, 필요시 참고 가능:
- 유도충전 방식 특성 (필립스 칫솔)
- 칫솔 헤드 호환성 (면도기 헤드와 구조 다름)
- 칫솔 방수 등급 (면도기와 표기 방식 다름)
