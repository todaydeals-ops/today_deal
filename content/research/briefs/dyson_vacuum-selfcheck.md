# 다이슨 무선청소기 흡입력 약할 때·작동 안 될 때 자가점검

조사 날짜: 2026-07-20  
대상 모델: V15, V12 Detect, V10 (한국 판매 현행 모델)  
범위: 한국(KR), 영문 병행

## 1. 흡입력 저하의 주요 원인

다이슨 공식 가이드는 흡입력 저하를 다음 순서로 점검하도록 안내한다:

### 1-1 필터 오염 (가장 흔함)
- **증상**: LCD 디스플레이에 "필터 확인" 또는 "필터 세척 필요" 알림 표시
- **원인**: 먼지·이물질 축적으로 필터 통과력 감소
- **점검**: 필터를 제거하고 색상 변화 확인

### 1-2 먼지통 과도 채워짐
- **증상**: 흡입력 급감, 입자 감지 정확도 저하
- **원인**: MAX 표시선 이상 먼지 축적
- **해결**: 즉시 먼지통 비우기

### 1-3 사이클론·에어웨이 막힘
- **증상**: LCD "에어웨이 막힘" 알림 또는 기기 자동 종료, 7번 진동 경고
- **원인**: 헤어(머리카락), 실, 큰 입자가 흡입구나 에어웨이를 폐쇄
- **점검 순서**:
  1. 호스 및 먼지통, 클리너 헤드 분리
  2. 각 부위의 이물질 제거
  3. 구부러지거나 찌그러진 호스 확인

### 1-4 브러시바 엉킴
- **증상**: 브러시바 회전 불량 또는 정지
- **원인**: 머리카락·실이 축에 감김
- **해결**: 브러시바를 기기에서 분리하여 찬물로 세척 후 완전 건조 후 재설치

### 1-5 필터 미장착 또는 부정확한 장착
- **증상**: LCD "필터 미장착" 또는 "필터 봉인 안 됨" 표시
- **영향**: 필터가 없으면 모터 직결로 즉시 손상 위험

## 2. 필터 관리 — 완전 건조의 중요성

### 2-1 세척 주기
- **권장 주기**: 평균 월 1회
- **LCD 알림**: 기기가 세척 필요 시점 자동 안내
- **과중 사용**: 더 빈번한 세척 필요 (대형 애완동물 있는 가정, 목재 바닥 등)

### 2-2 필터 세척 절차 [공식]
1. 필터를 싱크대 가장자리나 쓰레기통에 가볍게 두드려 먼지 탈거
2. 따뜻한 수돗물로 손으로 부드럽게 문지르며 세척
3. 완전히 건조될 때까지 기다림

### 2-3 완전 건조 — 필수 안전 기준
- **최소 건조 시간**: **24시간**
- **건조 환경**: 선풍기 앞이나 열린 창가 등 **바람이 잘 통하는 곳**
- **금지 사항**:
  - 헤어드라이어 금지
  - 전자레인지 금지
  - 직접 열원 근처 금지
  
**⚠️ 이유**: 습기가 남은 필터를 장착하면 모터 직결 상태 발생 → 단락/화재 위험

### 2-4 설치 재확인
- 필터를 정확한 위치에 끼우고 완전히 고정되었는지 확인
- LCD 디스플레이에 "필터 봉인됨" 확인 표시 필수

## 3. LED 표시등·경고 패턴

다이슨 무선청소기의 LCD 디스플레이는 다음 알림을 표시한다:

| 알림 메시지 | LED/진동 신호 | 의미 | 대처 |
|---|---|---|---|
| 필터 세척 필요 | LCD 텍스트 | 필터 오염도 높음 | 필터 세척 절차 진행 |
| 필터 미장착 / 봉인 안 됨 | LCD 텍스트 | 필터가 제거됐거나 부정확히 장착 | 필터 재확인 및 재설치 |
| 에어웨이 막힘 | 7번 진동 + LCD 표시 | 호스, 헤드, 또는 사이클론 막힘 | 각 부위 이물질 제거 |
| 배터리 저전 | 3번 진동 | 배터리 용량 거의 소진 | 충전 필요 |

## 4. 배터리 작동시간 저하

### 4-1 정상 런타임 (새 배터리 기준)
- **에코 모드** (통상): 최장 시간 (모델·사용 조건별 15~30분)
- **일반 모드**: 중간 (모델·사용 조건별 10~20분)
- **부스트 모드**: 최단 시간 (모델·사용 조건별 5~10분)

### 4-2 런타임 저하 원인
- **배터리 셀 노후화** (주로 2~3년 이후)
- **모드 설정**: 부스트 모드 사용 시 배터리 빠른 소모
- **온도**: 추운 환경(0℃ 이하)에서 임시 성능 저하
- **청소 도구**: 특정 헤드(동력 브러시바)는 배터리 더 소모

### 4-3 확인 방법
- LCD 디스플레이 배터리 상태 게이지 (%)
- 실제 사용 시간 vs 매뉴얼 명시 시간 비교
- 새로 구매한 배터리로 교체 테스트

## 5. 재시작 방법

### 5-1 즉시 재개 (중단했던 청소 재개)
- **방법**: 트리거(손잡이 누르는 버튼) 다시 누르기
- **효과**: 전원 유지 상태에서 즉시 다시 청소 시작

### 5-2 소프트 리셋 (일시적 오류 해제)
- **방법 1**: 배터리 분리 → 15초 대기 → 재삽입
- **방법 2**: 현재 배터리 상태에서 30분 충전 대기
- **효과**: 입자 카운트 그래프 초기화, 일시적 센서 오류 해제

### 5-3 하드 리셋 (완전 초기화 — 드문 경우)

- **효과**: 모든 설정 초기화 (권장되지 않음)
