# 삼성 TV 화면 안 나옴·신호없음·전원 문제 자가점검 조사

## 조사 개요

삼성 TV의 주요 자가점검 문제 3가지(화면 안 나옴, 신호없음, 전원 문제)에 대해 삼성전자서비스 공식 가이드 기준으로 정리.

## 1. 화면 안 나옴 / 신호없음 — 단계별 점검

### 1.1 최우선: 전원 & LED 확인

**공식 기준:** [삼성전자서비스 — TV화면이 안 나와요](https://www.samsungsvc.co.kr/solution/28337)

- TV 전원 코드를 **완전히 분리 후 60초 대기**, 다시 연결
- 코드 삽입 시 백색/회색 정렬 표시가 거의 보이지 않을 때까지 완전히 밀어넣기
- **절전형 콘센트 사용 중단** — 상시 전원 콘센트로 변경
- TV 하단 LED 표시등 상태 확인(켜짐/꺼짐)

### 1.2 신호 원인 판별: "입력신호가 약하거나 없습니다" 메시지

**공식 기준:** [삼성전자서비스 — TV 화면에 입력신호가 약하거나 없습니다](https://www.samsungsvc.co.kr/solution/39238)

신호 미약의 3가지 원인:
1. **안테나 미연결 또는 불량** — TV 뒷면 안테나 단자(75Ω 동축) 연결 확인
2. **셋톱박스 전원 문제** — 셋톱박스의 초록색 전원 표시등 확인, 전원 코드 재연결
3. **HDMI 연결 불량** — 케이블 양쪽 끝 완전 삽입 확인, 정품 '울트라 하이스피드 HDMI' 케이블 사용 여부 확인

대응 순서:
1. 셋톱박스 전원 확인 → 전원 코드 재연결
2. 안테나 연결 상태 확인
3. HDMI 케이블 분리 후 양쪽 끝을 완전히 재연결
4. 설정 → 채널/방송 → **채널 자동 검색** 실행
5. 호전 없으면 방송사에 신호 강도 문의

### 1.3 원커넥트 박스 사용 TV

**공식 기준:** [삼성전자서비스 — TV 화면이 안보여요, 셋탑박스연결이 안돼요](https://www.samsungsvc.co.kr/solution/996881)

- 원커넥트 박스 → TV 본체 연결 배선(광케이블 또는 HDMI) 확인
- 원커넥트 박스의 전원 표시등 상태 확인
- 원커넥트 박스 전원 코드 재연결 시도

### 1.4 HDMI 케이블 문제의 정체

**공식 기준:** [삼성전자서비스 — HDMI로 2022년TV와 PC연결시 PC화면 안나옴,깜빡](https://www.samsungsvc.co.kr/solution/996820)

- 비정품 HDMI 케이블: 화면 깜박임, 줄, 깨짐, 신호 끊김 유발
- 정품 기준: 케이블에 "**Ultra High Speed HDMI**" 문구 인쇄 또는 각인 필수
  - 없으면 미인증 제품 → 호환성 보장 안 함

## 2. 전원 문제 — 리모컨 vs TV 본체

**공식 기준:** [삼성전자서비스 — TV 전원이 켜지지 않아요](https://www.samsungsvc.co.kr/solution/40598)

### 2.1 LED 켜짐 vs 꺼짐 진단

**LED가 켜져 있으면:**
- TV 본체 정상 → 리모컨 문제 의심 낮음
- 콜드 부팅(전원 코드 분리 30초 이상 → 재연결) 시도
- 설정 → 고객지원 → 초기화 시도

**LED가 꺼져 있으면:**
- 리모컨 우선 의심
- **스마트폰 카메라 모드**로 리모컨 전원 버튼을 누를 때, 카메라 화면에 적외선 표시등이 보이는지 확인
  - 보이면: 리모컨 배터리 교체 후 TV에 IR 신호 수신되는지 재확인
  - 안 보이면: 리모컨 고장(교체 필요)

### 2.2 리모컨 배터리 확인 순서

1. **극성 확인** — 배터리 위/아래 방향(+/−) 정확히 삽입
2. **신형 배터리 교체** — 오래된 배터리일 경우 새 배터리로 즉시 교체
3. **USB-C 리모컨** — 충전 케이블 연결 (삼성 정품 케이블)
4. **솔라 리모컨** — 밝은 곳에서 충전(햇빛 또는 실내 밝은 조명)

### 2.3 콜드 부팅의 정의

- **절차**: 전원 코드 완전 분리 → 30초 이상 대기 → 재연결
- **효과**: 소프트웨어 오류로 인한 일시적 먹통, 화면 표시 오류 해결
- **주의**: 부팅 중 전원 버튼 누르기 금지, 30초 이상 코드 뺀 상태 유지

## 3. 리모컨 페어링 — 모델 연식별 방법

**공식 기준:** [삼성전자서비스 — 리모컨 동작이 안 될때 연결(페어링)하는 방법](https://www.samsungsvc.co.kr/solution/25844)

### 블루투스 스마트 리모컨 페어링 (대부분의 현행 모델)

| TV 연식 | 버튼 조합 | 지속 시간 |
|--------|---------|---------|
| 2016~2025 | Back + Play/Pause | 5초 이상 |
| 2015 | Back/Exit + More Info | 5초 |
| 2014 | Back + Guide | 5초 |
| 2013 | 리모컨 백커버 오픈 후 Pairing 버튼 | 5초 |

**성공 표시**:
- TV 화면에 "스마트 리모컨 연결중" → "스마트 리모컨 연결됨" 메시지 순서 표시

**실패 시**:
- TV 콜드 부팅 후 재시도
- 여전히 실패하면 리모컨 고장 (교체 필요)

## 4. 자가진단 기능 활용 — 화질·신호·HDMI 점검

### 4.1 자가진단 진입 경로

**공식 기준:** [삼성전자서비스 — 리모컨과 자가진단 기능, 제품 안전사용 방법](https://www.samsungsvc.co.kr/solution/239445)

```
홈 버튼 → 설정(또는 기종별로 '고객지원', '지원') 
→ 디바이스 케어(또는 '고객지원') 
→ 자가진단
```

**주의**: 메뉴명과 경로가 TV 모델·연식·펌웨어 버전별로 다를 수 있으므로, 자기 TV의 메뉴를 직접 탐색해야 함

### 4.2 자가진단 내 주요 점검 기능

| 기능 | 확인 항목 | 판별 기준 |
|-----|---------|---------|
| **화질 테스트** (Picture Quality Test) | 색상 이상, 밝기 이상 | 정상 색 재현 시 통과 |
| **화면 테스트** (Screen Test) | 줄, 얼룩, 표시 불가 | 테스트 화면에서 고정 줄이 없으면 패널 정상 |
| **음질 테스트** (Sound Test) | 음성 누락, 노이즈 | 명료한 음성 재현 시 통과 |
| **HDMI 점검** (2024~) | HDMI 포트·케이블 상태 | 정품 케이블 확인 여부 표시 |
| **신호 점검** (Signal Check) | 방송 신호 수신 강도 | 신호 강도 % 표시 |
| **스마트허브 테스트** | 인터넷 연결·앱 동작 | 연결·업로드/다운로드 속도 표시 |

### 4.3 화면 줄 진단의 실제 활용

**공식 기준:** [삼성전자서비스 — TV 화면에 가로 줄이 보여요](https://www.samsungsvc.co.kr/solution/37061)

**자가진단 → 화질 테스트 실행**:

- **테스트 화면에서 줄이 보이는 경우**: TV 패널 문제 → 서비스센터 수리 필요
- **테스트 화면에서 줄이 안 보이는 경우**: TV 패널은 정상
  - 원인: HDMI 케이블 불량, 셋톱박스 신호 오류, 신호 미약
  - 대응: HDMI 케이블 재연결 → 정품 케이블 교체 → 셋톱박스 재부팅 → 신호 점검

## 5. 기존 조사 자료와의 관계

### 재사용하지 않은 이유

삼성 다른 제품(에어컨·세탁기·건조기) 팩트 중 겹치는 것:
- `samsung-model-year-alphabet`: 에어컨 AF/AR 라인 연식 규칙 → TV 모델명 체계 완전히 다름
- `samsung-aircon-selfdiag-entry`: 에어컨 자가진단 진입 방법 → TV는 메뉴 경로 상이
- `samsung-errorcode-digit-change`: 에어컨 에러코드 (E1 vs E101) → TV에 적용 불가
- `samsung-washer-dc-layer-conflict`: 세탁기 코드 → TV와 무관

**결론**: TV는 자체적인 에러·진단 체계가 완전히 독립적이므로, 공식 TV 가이드에서만 정보 수집함

## 6. 주의 사항

### 콘텐츠 작성 시 고려할 점

1. **메뉴 경로의 변동성**: "설정" 진입 후 왼쪽 화살표 네비게이션, 모델별로 "고객지원" 또는 "지원" 또는 "디바이스 케어" 등 다양
   - 글에서 "기종에 따라 메뉴명이 다를 수 있으니 직접 탐색하세요"라는 안내 필수

2. **연식별 자가진단 기능 차이**: 2024년부터 HDMI 문제 점검 기능 추가
   - 함수, 표로 연식 구분 필수

3. **원커넥트 박스**: 고급 모델에만 탑재. TV 일반 사용자도 있으므로 "해당하는 경우" 구분 명시

4. **스마트폰 카메라 IR 확인법**: 리모컨 고장 진단의 가장 빠른 방법. 일반인도 쉽게 따라할 수 있는 팁

5. **정품 HDMI 케이블**: "Ultra High Speed" 인쇄 기준은 공식 기준. 비정품 유입 사고 이력 많으므로 강조

6. **콜드 부팅 vs 리셋**: 서로 다른 개념
   - 콜드 부팅: 전원 코드 분리 (소프트웨어 오류 해결)
   - 팩토리 리셋: 설정 초기화 (데이터 손실, 마지막 수단)
