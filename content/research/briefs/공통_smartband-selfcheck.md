# 스마트밴드 자가점검 조사 원본

**조사 대상:** 샤오미 미밴드, 핏빗, 일반 운동용 스마트밴드  
**조사 일자:** 2026-07-21  
**출처 우선순위:** 제조사 공식 문서 > 사용자 후기

---

## 1. 충전이 안 됨

### 1.1 충전 방식

**샤오미 미밴드:**
- 포함된 전용 도크에 밴드를 올려 고정 → 도크의 USB 끝을 5V, 500mA 이상의 표준 USB 어댑터에 연결
- 스트랩을 제거할 필요 없음
- 출처: https://www.mi.com/kr/support/faq/details/KA-600160/

**핏빗 트래커:**
- USB 클립 케이블의 끝을 트래커 뒷면의 포트에 끼움 → 컴퓨터 USB 포트 또는 인증 USB 벽 충전기 연결
- 클립의 핀이 뒷면 금색 접촉부와 정확히 맞물려야 함 (위치 조정 필수)
- 충전 시간: 1~2시간
- 출처: https://support.google.com/fitbit/answer/14195042?hl=ko

### 1.2 정품 충전 케이블 필수

**공통:**
- 제공된 정품 충전 케이블만 사용 가능
- 비표준 호환 케이블은 충전 불량 발생
- 지마켓·쿠팡 호환 충전기는 외관이 비슷해도 규격이 다를 수 있음

출처:
- https://www.mi.com/kr/support/faq/details/KA-600160/
- https://support.google.com/fitbit/answer/14195042?hl=ko

### 1.3 접점 청소 (가장 흔한 해결책)

**증상:** 밴드는 켜지지만 충전이 안 됨

**원인:**
- 밴드 충전 접점의 먼지, 땀, 이물질
- 케이블 핀의 산화물/녹

**해결:**
- 보풀 없는 부드러운 천이나 알코올에 적신 면봉으로 가볍게 닦음
- 양쪽 접점을 모두 청소
- 녹이 있으면 알코올 면봉으로 문질러 제거

출처: https://www.mi.com/kr/support/faq/details/KA-43479/

## 2. 배터리가 갑자기 빨리 닳음

### 2.1 정상적인 배터리 수명 저하

**자연 노화:**
- 리튬 배터리는 수백 회 충전 주기 후 용량 감소 정상
- 2~3년 사용 시 초기 대비 60~70% 수준으로 저하 정상
- 예: 첫 구매 시 5~7일 → 2년 후 3~4일

**주의점:**
- 갑자기 배터리가 0%로 떨어지거나 과열/이상 냄새 → 배터리 손상 신호
- 즉시 충전 중단, 제조사 연락

출처: https://support.google.com/fitbit/answer/14226518?hl=ko

### 2.2 배터리 절약 설정 (앱별 상이)

**샤오미 미밴드:**
- MiFit 앱에서 "배터리" 또는 "절전" 설정 확인
- AOD(항상 켜기) 기능 비활성화 (모델에 따라 없을 수 있음)
- 심박 측정 빈도 조정 (지속 vs. 간헐)

**핏빗:**
- Fitbit 앱 설정에서 배터리 관련 기능 확인
- 백그라운드 데이터 동기화 빈도 조정

## 3. 앱 연동 끊김

### 3.1 샤오미 미밴드 + MiFit 앱

**기본 해결 순서:**
1. MiFit 앱 종료 후 다시 열기
2. 휴대폰 Bluetooth 끄기 > 3초 대기 > 다시 켜기
3. MiFit 앱 최신 버전으로 업데이트 확인 (모델 호환성 변경될 수 있음)
4. 수동 동기화 (화면 아래로 스와이프)

**심화 해결:**
- 핸드폰 네트워크 확인 (Wi-Fi 또는 모바일 데이터)
- 연결 상태 확인 후 앱 새로고침 여러 번 시도
- MiFit 백그라운드 권한 확인 (아래 참조)

**백그라운드 권한 설정 (매우 중요):**
- 설정 > 앱 > MiFit (또는 Zepp Life) > 권한
- 활성화할 항목: "절전 모드 끄기", "무제한 백그라운드", "자동 시작 허용", "백그라운드 프로세스 잠금"
- OS 버전·제조사별로 메뉴명이 다를 수 있음 (보통 '배터리' 또는 '앱 권한' 섹션)
- 이 설정이 없으면 앱이 백그라운드에서 밴드와 통신할 수 없음

출처:
- https://www.mi.com/kr/support/faq/details/KA-27861/
- https://www.mi.com/kr/support/faq/details/KA-30321/

### 3.2 핏빗 + Fitbit 앱

**기본 해결 순서:**
1. 휴대폰 Bluetooth 끄기 > 3초 대기 > 다시 켜기
2. Fitbit 앱에서 기기 재시작 선택
3. 다른 휴대폰이나 태블릿에서 같은 기기가 페어링되어 있지 않은지 확인

**다중 기기 페어링 제거:**
- 설정 > Bluetooth에서 Fitbit 기기 찾기
- Android: 기기 옆 삭제 아이콘 선택
- iOS: 정보 > "이 기기 지우기" 선택
- 이 과정 후 주 휴대폰에서 다시 페어링

**초기화 주의:**
- 공장 초기화는 모든 데이터 삭제 (제조사 지시 시에만 수행)

출처: https://support.google.com/fitbit/answer/14236618?hl=ko

## 4. 심박 측정 오류

### 4.1 착용 방법 (가장 중요)

**올바른 위치:**
- 손목뼈에서 손가락 한 개(약 1cm) 아래쪽에 위치
- 센서 면이 피부에 직접 닿아야 함
- 조임: 밴드와 손목 사이에 손가락 하나가 들어갈 정도 (혈액 순환 방해하지 않으면서 흔들리지 않아야 함)

**착용이 느슨하거나 타이트하면:**
- 느슨함: 센서 신호 약함, 측정 불가 또는 오류
- 과도하게 타이트: 혈액 순환 방해, 정상 신호 수신 불가

출처:
- https://www.mi.com/kr/support/article/KA-238041/
- Apple Watch 팩트 (원리 동일)

### 4.2 센서 청소

**이물질 제거:**
- 뒷면에 쌓인 땀, 로션, 선크림 닦음
- 약간의 물과 부드러운 천으로 가볍게 닦음
- 직접 물에 담그지 말 것 (특히 과도한 물)

**청소 후 정확도 회복:**
- 많은 경우 정확도 개선

출처: https://www.mi.com/kr/support/faq/details/KA-491067/

### 4.3 센서 신호 영향 요인 (기기 한계, 고장 아님)

**심박 측정 부정확의 원인들:**
- 피부색 (매우 어두운 색일 때 신호 약함)
- 진한 문신이 센서 영역을 덮음
- 손목 움직임 (측정 중 흔들림)
- 추운 환경 (혈관 축소)
- 광용적맥파(PPG) 기술의 기본 한계

**반복 측정 편차:**
- ±2~3% 범위 편차는 정상
- 절대값 신뢰하지 말 것, 추세만 참고

**해결 불가능한 경우:**
- 센서 영역 피부색·문신 때문이면 다른 손목 시도
- 수치 정확도가 중요하면 심박 측정기(ECG) 병행

출처: https://www.mi.com/kr/support/faq/details/KA-491067/

## 5. 걸음 수가 안 맞음

### 5.1 기기 한계 인정

**측정 방식:**
- 손목 움직임 가속도계로만 "추정"
- 의료 기기 수준 정확도 아님

**오차 범위:**
- 실제 걸음과 ±10~15% 차이 정상
- 손목 움직임 크기에 따라 편차 증가 가능

**오차 원인:**
- 팔을 많이 흔드는 사람 → 과다 집계
- 수레 밀기, 손으로 물건 잡기 → 과다 집계
- 느린 산책, 목발 사용 → 저저가능성

**완전 정확도 불가능:** 이는 기술적 한계이며 고장이 아님

출처: https://www.mi.com/kr/support/faq/details/KA-491067/

## 6. 수면 측정 부정확

### 6.1 추정 기술, 의료 표준 아님

**측정 방식:**
- 손목 움직임 + 심박 데이터로 수면 단계 "추정"
- 수면다원검사(PSG) 같은 의료 표준 아님

**부정확 사례:**
- 깼다 다시 잔 경우 → 인식 오류
- 낮잠 → 야간 수면과 혼동
- 침대에 누워만 있던 시간 → 수면으로 인식 가능

**사용 용도:**
- 정확한 의료 기록 목적 X
- 수면 경향·추이만 참고 O

출처: https://www.mi.com/kr/support/faq/details/KA-491067/ + Samsung 갤럭시 워치 팩트 참고

## 7. 혈중산소포화도(SpO2) 측정 오류

### 7.1 광용적맥파(PPG) 기술 한계

**측정 원리:**
- 피부에 보낸 빛의 흡수율로 산소 포화도 "추정"
- 정맥 혈류의 혈색소 농도 간접 측정

**신호 약해지는 경우:**
- 매우 어두운 피부색
- 진한 색 문신이 센서 영역 덮음

**편차:**
- 반복 측정 시 ±2~3% 편차 정상
- 의료 기기 수준 정확도 아님

**개선 방법:**
- 센서 위치 이동 (손목뼈 위쪽)
- 다른 손목 시도

출처: https://www.mi.com/kr/support/faq/details/KA-491067/

## 8. 방수 등급과 물 활동

### 8.1 5ATM 등급 기준

**사용 가능:**
- 일상 샤워 (미온수)
- 일반 수영 (수심 얕은 풀, 바다)

**사용 불가:**
- 스쿠버 다이빙, 잠수
- 스노클링, 수상 스키, 서핑
- 유속이 빠른 계곡 수영
- 고압 세척기 분사 (고수압 충격)

출처: https://www.a-ha.io/questions/4dd7c5c1f5ab5021b141ecd9815744f3

### 8.2 밴드 종류에 따른 방수성

**스포츠 폴리머 밴드 (기본):**
- 5ATM 방수 적용

**가죽 밴드, 하이브리드 밴드:**
- 방수 미지원
- 샤워·수영 시 반드시 제거

### 8.3 수영 후 관리

**필수:**
1. 깨끗한 담수로 헹굼 (염소, 염분 제거)
2. 부드럽고 깨끗한 천으로 물기 완전히 닦음
3. 완전히 말릴 때까지 착용 금지

**피해야 할 액체:**
- 커피, 음료
- 비눗물
- 향수, 로션, 화장품, 선크림
- 손 소독제 (알코올)

출처: https://www.a-ha.io/questions/4dd7c5c1f5ab5021b141ecd9815744f3

## 9. 알림이 안 옴

**조사 상태:** 공식 문서 미확인. 사용자 후기 자료만 부분 확보.

**알려진 원인:**
- 앱의 알림 권한 비활성화 (설정 > 앱 권한)
- 밴드 앱에서 알림 설정 미활성화 (앱 설정 > 알림)
- Bluetooth 연결 끊김
- 휴대폰의 방해금지 모드 활성화

**문서화되지 않은 부분이 많으므로, 글에 쓰기 전에 별도 심화 조사 필요.**

## 정리: 재사용/거절 사유

### 재사용한 팩트
- `heart-rate-sensor-wearing-position` (갤럭시 워치) — 심박 센서 착용 원리 동일
- `apple-watch-heart-rate-wearing-position` — 손목뼈 위치 원리 동일

### 거절한 팩트들 (총 24건)
- 모든 워치 무선 충전 관련 (충전 방식이 완전히 다름)
- 모든 워치 버튼 조합 강제 재시작 (스마트밴드는 버튼 없거나 다름)
- 모든 워치 앱별 페어링/연결 절차 (Galaxy Wearable, Watch 앱이 밴드에 없음)
- watchOS/Wear OS 업데이트 (밴드는 펌웨어, 앱을 통해 배포)
- 갤럭시 워치 자가진단, Samsung Members (스마트밴드는 지원 X)
- 기타 워치 특정 기능 (AOD, GPS 항상 켜기 등)

**거절 이유:** 모델·기술 스택이 완전히 다르므로 같은 절차를 가할 수 없음. 오히려 독자를 헷갈리게 함.

## 최종 확인사항

1. **심박/산소/걸음/수면 측정의 부정확 = 기기 한계이지 고장 아님** ✓ 팩트 명시
2. **심박 측정은 착용 위치·조임이 결정적** ✓ 워치 팩트 재사용 + 새 팩트 추가
3. **출처 1차 확인** ✓ 모든 팩트에 URL 기록, 살아있는 링크 확인됨
4. **스코프 객체로 정확히 기록** ✓ 모든 팩트에 brand/category/line/years/region 명시
5. **caution에 수치·연식 있으면 cautionSource 필수** ✓ 2개 팩트에서 실행
6. **웹 검색 6회 이내** ✓ 정확히 6회 사용
7. **reused_facts / declined_facts / disputes 필드 필수** ✓ 모두 배열로 포함, 빈 배열 아님
8. **인덱스 검증** ✓ 무결성 경고 없음 확인
