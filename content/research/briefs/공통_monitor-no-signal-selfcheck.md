# 모니터 신호 없음·화면 깜박임·색상 이상 자가점검 — 조사 전문

## 조사 개요

모니터 신호 없음, 깜박임, 색상 이상, 해상도 문제, 듀얼 모니터 인식 불가, 밝기 조절 불가 등 모니터의 주요 자가점검 항목을 조사했다. TV와 달리 모니터는 PC 그래픽카드와의 연결이 핵심이므로, **모니터 문제 vs PC 문제 판별법**을 특별히 강조했다.

## 웹 검색 결과 (6회)

1. **모니터 신호 없음 자가점검 (삼성 공식)**
   - 케이블 재연결, 입력 소스 확인, 메뉴 테스트, 전원 리셋 순서
   - 메뉴 문자가 보이면 모니터 정상, PC 문제로 판단

2. **화면 깜박임 원인과 해결 (삼성·LG 공식)**
   - HDMI 케이블 버전 확인 (1.2 vs 2.0 vs 2.1)
   - Windows 주사율 설정 (기본 60Hz로 낮추기)
   - 그래픽카드 드라이버 업데이트
   - LG: SES/DFC 자동 밝기 조정 기능 비활성화

3. **HDMI vs DP vs USB-C 차이 (웹 커뮤니티)**
   - HDMI(5v) vs DP(3.3v) 신호 규격 차이
   - USB-C는 DisplayPort Alt Mode 필수 (영상 출력 기능)
   - HDMI 2.0 이상: 4K 60Hz, HDMI 2.1: HDR 지원

4. **모니터 문제 vs PC 그래픽카드 문제 판별 (커뮤니티)**
   - 스크린샷 테스트: 스크린샷에서도 같은 증상 → 그래픽카드 문제
   - 다른 모니터 연결 테스트
   - 데스크톱: 그래픽카드 제거 후 내장 그래픽 테스트

5. **색상 문제 자가진단 (삼성·LG 공식)**
   - 케이블 확인, 팩토리 리셋
   - RGB 조정, 색온도 변경 (표준 6500K)
   - 온라인 색상 테스트 도구 (monitor.co.kr)
   - Windows 야간 모드 확인

6. **듀얼 모니터·밝기·번인 (Dell·커뮤니티)**
   - 듀얼: 각 모니터 케이블 단단히 재연결, Windows 디스플레이 설정 확인
   - 밝기: 그래픽 드라이버 업데이트 (DDC/CI 지원)
   - 번인: 공식 해결 방법 미확인 → tier 확인실패

## 주요 발견사항

### 1. TV와의 근본적 차이

TV 팩트를 직접 재사용할 수 없는 이유:
- **입력 소스 메뉴**: TV = 방송/HDMI/안테나 / 모니터 = HDMI/DP/USB-C 포트 번호
- **신호 문제 원인**: TV = 방송 신호·셋톱박스 / 모니터 = PC 그래픽카드만 관계
- **자가진단 구조**: TV = 제조사별 다른 경로 / 모니터 = OSD 메뉴 (완전히 다름)

### 2. 모니터 문제 판별의 핵심 = 스크린샷 테스트

부품 교체나 복잡한 작업 없이도 가능:
- Print Screen → 그림판에 붙여넣기 → 같은 증상? → 그래픽카드/드라이버 문제
- 스크린샷 정상? → 모니터 자체 문제 → 케이블, 입력 소스 확인 → 팩토리 리셋

이 단순한 방법이 가장 정확한 판별법이다 (데스크톱 그래픽카드 제거는 복잡하므로 2순위).

### 3. HDMI 케이블 버전 문제의 실제 영향

깜박임·해상도 저하는 종종 무시되는 원인:
- HDMI 1.2: FHD 60Hz, QHD 30Hz만 지원
- 고해상도·고주사율 모니터에서 구형 케이블 사용 = 깜박임
- "케이블 교체"가 가장 간단한 해결책인 경우가 많음

### 4. 듀얼 모니터는 동일 포트 권장

HDMI와 DP 섞어 사용 시:
- 신호 규격 다름 → 호환성 문제 가능
- Windows에서 두 포트를 다르게 취급 → 설정 복잡
- 동일 포트(둘 다 HDMI 또는 둘 다 DP) 권장

### 5. 번인·잔상은 공식 자가진단 방법 부족

- LCD/LED 번인: 이론적으로 불가능하지만, 극단적 조건에서 보고됨
- OLED 번인: 알려진 회복 불가능한 손상
- 공식 제조사 자가진단 절차 미확인 → 예방과 서비스 센터 상담만 가능
