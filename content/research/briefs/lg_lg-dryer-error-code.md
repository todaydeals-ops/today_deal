**조사일**: 2026-07-20  
**주제**: LG 건조기 에러코드별 의미, 콘덴서 자동세척/수동세척, 먼지필터 청소, 물통·배수, 히트펌프 겨울철 성능

## 1. 모델명 체계

LG 건조기는 삼성과 완전히 다른 모델명 구조를 사용한다.

**구조**: R(건조기) + 조작부(H/G/D) + 용량(숫자) + 색상코드 + 스팀(N/S) + 결합제품(F/X)

- R: 건조기를 뜻하는 접두어
- H: 전통적 다이얼식, G: 디스플레이 중심, D: 이지서클 다이얼
- 용량: 16kg, 19kg, 22kg 등
- 색상·스팀·결합제품은 선택사항

**예시**: RH16GVWW (R+H다이얼+16kg), RD19S (R+D다이얼+19kg+스팀)

이는 삼성의 DV+용량+연식+기능코드 구조와 완전히 다르며, LG는 모델명에 연식이 직접 표기되지 않는다.

---

## 2. 에러코드 체계

LG 건조기 에러코드는 삼성의 '모두 C로 끝남' 규칙(5C, SC, nC2 등)과는 완전히 다르다. LG는 D, E, F, H, P, t 등 다양한 문자와 숫자를 혼용한다.

### 2.1 배기 막힘 (D75/D80/D90/D95)

- **의미**: 배기덕트 막힘 정도를 백분율로 표시
- **확인 사항**: 먼지필터 상태, 배기호스 꺾임 확인, 외부 환기구 흙/이물질
- **조치**: 필터 청소, 호스 정리, 환기구 청소

### 2.2 전원·전압 문제

- **nP**: 히터 전류 미감지 → 240V 전원 연결 확인, 차단기 리셋
- **PS**: 부적절한 전압 → 전원 코드 배선 연결 확인

### 2.3 기계식 오류

- **dE**: 도어 미폐쇄 → 필터 재장착, 도어 장애물 확인
- **F0** / **IF**: 필터 미장착 → 외부/내부 필터 올바른 위치 삽입

### 2.4 배수 관련

- **E13**: 배수호스 구부러짐/막힘 (하이브리드 모델만)

### 2.5 센서 오류

- **HS**: 습도 센서 오류 → 필터 아래 센서 청소
- **tE1/tE3**: 온도 센서 오류 → 공장 초기화 시도
- **tE2**: 추운 날씨 중 온도 센서 오류 → 배기/통풍구 확인, 벌크/두께 코스 선택

## 3. 콘덴서(열교환기) 관리

### 3.1 자동 세척 (기본 작동)

LG 건조기는 다음과 같이 **자동으로** 콘덴서를 세척한다:
- 매 건조 코스 사용 시마다, 건조가 끝나기 전에 자동 세척
- 추가로 건조 30회마다 한 번 더 자동 세척

**원리**: 세탁물의 수증기 → 콘덴서에서 응축 → 응축수 → 펌프로 분사(중앙·좌·우 순서) → 배수

사용자는 별도 개입 없이 자동으로 진행되므로 주의할 사항이 없다.

### 3.2 수동 콘덴서케어 (필요시)

추가 세척이 필요할 때 직접 작동하는 기능이다.

**절차**:
1. 물통을 꺼내 물이 비어 있는지 확인
2. 물 1L를 천천히 투입 (단독 설치 시 물통에, 워시타워 상부 설치 시 필터 투입구에)
3. [콘덴서케어] 버튼을 3초 이상 눌러 시작
4. 약 1시간 작동

**권장**: 필터를 먼저 청소한 후 수동 콘덴서케어를 진행하면 효과적이다.

## 4. 먼지필터 청소

### 4.1 내부 필터

- **주기**: 매 사용 전/후 (항상)
- **방법**:
  1. 세탁물 제거 후 필터 꺼내기
  2. 화살표 방향으로 펼치기
  3. 큰 먼지 제거
  4. 청소기 또는 흐르는 물로 세척
  5. **완전히 건조 후** 중앙부를 눌러 닫기

**주의**: 덜 마른 필터를 사용하면 수막현상으로 냄새 발생

### 4.2 외부 필터

- **주기**: 사용 전/후 또는 보풀 쌓일 때 (약 10회 사용 후)
- **방법**:
  1. 투입구 주변 먼지 제거
  2. 필터 꺼내기
  3. 측면을 당겨 펼치기
  4. 청소기 또는 물로 세척
  5. 우측면 표시부를 누르고 닫기
  6. 내부 필터와 결합 후 장착

**필수**: 필터 없이는 건조기 작동 불가

---

## 5. 물통·배수 관리

### 5.1 물통(배수통)

- **주기**: 건조 후 매회 비우기
- **알림**: 디스플레이에 '물 비우기' 표시 시 즉시 비우기
- **중요**: 물통이 가득 차면 건조기는 자동 중단

**주의**: 양손으로 잡고 수평 유지하며 천천히 꺼내기

### 5.2 배수호스 (직배수 모델)

- **높이**: 바닥으로부터 1m 이하 (자연 낙하 유지)
- **위치**: 단단하고 고른 바닥 위에 위치
- **호스 끝**: 물에 잠기지 않아야 함 (5~10cm 위에 위치)
- **깊이**: 하수구에 너무 깊이 박혀 있지 않아야 함

**위반 시 결과**: 배수 역압 발생 → 배수 문제

## 6. 건조 불량 자가점검 (시간 증가)

건조 시간이 평소보다 길어진 경우, **다음 순서**로 점검해야 한다:

### 6.1 습도 센서 청소

센서는 필터 아래에 위치하며, 먼지가 붙으면 습기 감지 실패

### 6.2 필터 청소

필터 막힘은 건조 성능 저하의 주된 원인

### 6.3 세탁물 분류

- 두께가 다른 옷을 함께 건조하면, 얇은 옷이 먼저 건조되어 두꺼운 옷이 덜 마감
- **개선**: 비슷한 두께 옷끼리 분류, 용량의 절반까지만 채우기

### 6.4 기온 확인

- 겨울철: 건조 시간 최대 1시간 증가 가능
- 설치 공간 5℃ 이하: 심각한 성능 저하

### 6.5 전문가 점검

30분 이상 운전 후에도 열감이 없으면 서비스 필요

## 7. 히트펌프 방식의 겨울철 특성

### 7.1 원인

LG 건조기 히트펌프는 외부 공기를 활용하여 건조한다. 겨울철 낮은 기온에서:
- 건조기와 세탁물 온도가 낮아짐
- 예열 시간 증가
- 기온 5℃ 이하: 건조 시간 길어짐, 효율 저하

### 7.2 완화 방법

1. **세탁 단계**: 탈수를 [최강] 또는 [건조맞춤]으로 설정, 필요시 추가 탈수
2. **세탁물 분류**: 두께별 분류, 용량 절반까지만 채우기
3. **코스 선택**: 두꺼운 옷은 강력 모드
4. **필터 관리**: 내부 필터는 매회, 외부 필터는 10회마다 청소 → 공기 순환 최대화
