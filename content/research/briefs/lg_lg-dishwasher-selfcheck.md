# LG 디오스 식기세척기 자가점검 조사 원본

**조사 날짜:** 2026-07-21  
**출처:** LG전자 스스로해결(lge.co.kr/support), LG ThinQ 고객지원

## 개요

LG 디오스 식기세척기 사용자가 마주치는 에러코드, 세척 불량, 배수 문제에 대한 자가점검 방법을 LG 공식 문서를 통해 조사했다.

### 삼성 팩트와의 비교

기존 삼성 식기세척기 조사에서 다룬 에러코드(E03, FE, LC, 4C, 5C, 3C, HC)는 제조사 고유 체계이므로, LG 에러코드(IE, OE, FE, AE/RE, bE, tE, HE, LE, PE, CL)와는 완전히 다르다. 따라서 삼성 에러코드 5건은 모두 `declined_facts`에 기록했다.

반면 **배수 필터 청소, 배수 호스 높이, 잔수의 정상성, 세척 원리(그릇 배치), 건조 성능(린스 사용)**은 제조사를 불문하고 동일한 물리·화학 원리에 기초하므로, 삼성 팩트 3건(residual-water-normal, rinse-aid-water-tension, vinegar-rinse-warning)을 `reused_facts`에 기록했다.

## LG 에러코드 체계 (IE, OE, FE, AE/RE, bE 등)

LG 식기세척기의 에러코드는 영문 2자리로 표기되며, 각각 다음을 의미한다:

### 급수 문제
- **IE (Inlet Error)**: 물이 식기세척기 내부로 들어오지 못함
  - 확인 항목: 싱크대 급수 밸브 개방, 급수 호스 눌림/막힘/동파

### 배수 문제
- **OE (Outlet Error)**: 세척통 안의 물이 배출되지 못함
  - 주요 원인: 배수 필터 막힘, 배수 호스 꺾임
  - 자가점검: 필터 분리·청소(반시계 분리 → 반시계 회전으로 설치), 호스 높이(30cm 이상), 호스 꺾임 확인
  - 강제 배수: 전원 켜고 시작 버튼 3초 누르기

### 과급수/누수 문제
- **FE (overfilling)**: 표준량보다 많은 물이 급수됨
  - 확인 항목: 큰 그릇이 노즐을 막고 있지 않은지, 세제 종류(일반 세제 거품 → FE 유발)

- **AE/RE (누수 감지)**: 제품 내부로 누수 감지
  - 주의사항: 열풍건조 모델의 경우 연수장치 덮개와 에어필터 덮개를 혼동하면 누수 발생 가능
  - 자동 배수 작동으로 2차 피해 방지

### 거품 에러
- **bE (Bubble Error)**: 세제 거품 과다 발생
  - 원인: 일반 주방용 세제 사용 (전용 세제 필수)
  - 안전 기능: 누수 방지를 위해 자동 배수

### 센서/모터 문제
- **tE**: 온도감지센서 이상
- **HE**: 히터 작동 이상
- **LE**: 세척 모터(순환 펌프) 이상
- **PE**: 수위감지센서 이상
- **CL**: 버튼잠금 표시 (정상 기능, 오작동 아님)

## 배수 불량 (OE 에러) 자가점검

### 필터 청소
- **주기**: 1~2주에 1회 (사용 빈도에 따라 조정)
- **방법**:
  1. 전원 끔 → 30분 대기
  2. 아래 식기 바구니 제거
  3. 필터를 반시계 방향으로 회전하여 분리
  4. 흐르는 물로 세척
  5. 시계 방향으로 회전하여 재설치
- **확인**: 필터에 이물질(음식물, 음료수 등)이 없는지 육안 확인

### 배수 호스
- **높이**: 호스 끝부분이 바닥으로부터 최소 30cm 이상 떨어져 있어야 함
  - 30cm 이하 → 하수구 냄새 역류 또는 역류 물 발생 가능
- **형태**: 눌리거나 꺾인 부분이 없는지 확인
  - 호스가 구부러져 있으면 물이 흐르기 어려움
- **연결**: 수도 밸브 → 호스 연결 상태 확인

### 강제 배수 테스트
물이 필터 아래를 넘어갈 정도로 고여 있으면:
1. 전원 켜기
2. 시작 버튼 3초 누르기
3. 2~3분 동안 강제 배수 사이클 작동

---

## 물이 안 들어옴 (IE 에러)

### 확인 순서
1. 싱크대 급수 밸브가 완전히 열려 있는지 확인
2. 급수 호스가 눌리거나 꺾여 있지 않은지 확인
3. 급수 호스 연결부가 안전하게 연결되어 있는지 확인
4. 겨울철 급수 호스 동파 여부 확인

## 세척 불량 원인 및 해결책

### 그릇 배치 (중요)
- **올바른 배치**:
  - 접시: 세로 세우기
  - 냄비/그릇: 오목한 부분이 아래를 향하도록 배치
  - 회전 노즐에 닿지 않도록 바구니 높이 조절
  - 겹치지 않기

- **피해야 할 배치**:
  - 큰 그릇으로 노즐 차단
  - 식기 겹치기 (물이 닿지 않음)
  - 배수 필터를 덮을 정도로 높이 배치

### 음식물 처리
- 큰 음식물 조각 미리 제거 (고추, 뼈, 견과류 등)
- 눌어붙은 음식물: 물에 불린 후 스펀지로 가볍게 애벌세척

### 세제 선택 및 사용
- **종류**: 고체 세제 권장, 액체 세제는 세척 및 건조 성능 저하
- **양**: 제조사 권장량을 식기 양과 더러움 정도에 맞춰 조절
  - 적게 → 효과 부족
  - 많이 → 거품 발생
- **투입 방법**:
  - 투입 전 투입구와 손의 물기 닦기
  - 덮개가 "딸깍" 소리 날 때까지 밀기
- **금지**: 일반 주방용 세제 절대 금지 (거품 과다 → bE 에러 또는 누수 위험)

### 세척 코스 선택
- 경도: 약한 세척 → 강력 세척 (오염 정도에 맞춤)
- 특수 옵션: 불림, 살균세척, 스팀 등 활용

### 내부 청소
- 필터 정기 청소 (1~2주)
- 통살균 코스 + 구연산 40g (냄새 제거, 내부 세척)
  - 월 1~2회 권장
  - 일반 세제 금지, 세제 투입 금지, 구연산만 투입

## 건조 불량 원인 및 해결책

### 건조 성능에 영향을 주는 요소

#### 1. 린스 부족 (가장 중요)
린스(Rinse Aid)가 없거나 레벨이 낮으면 건조 성능이 현저히 감소한다.

**린스의 역할**:
- 물 표면장력 감소 → 물이 빠르게 흘러내림
- 건조 시간 단축, 얼룩 방지

**주의**:
- 식초나 베이킹소다는 효과 없음
- 오히려 헹굼 분배기 부품 손상 초래 가능
- **반드시 식기세척기 전용 린스만 사용**

#### 2. 식기 배치
- 겹치지 않기: 겹쳐 있으면 물이 닿지 않아 건조 불가
- 플라스틱/얇은 소재: 온도가 빠르게 떨어져 건조 어려움 (근본적 해결 어려움)
- 뒷굽이 있는 그릇(국그릇 등): 기울여서 물이 고이지 않게 배치

#### 3. 건조 방식 이해
응축 건조 + 자동 문열림 + 자연 건조의 3단계로 진행:
1. 뜨거운 수증기가 냉각되면서 물로 응축
2. 세척 완료 후 자동으로 도어 열림
3. 수증기가 천천히 자연 배출됨

따라서 문을 바로 열어도 즉시 건조가 완료되지 않으며, 자동 문열림 후 5~10분 정도 두는 것이 효과적이다.

## 냄새 제거

### 원인
필터에 음식물 잔여물이 고이면서 부패/발효하여 냄새 발생

### 해결책
**구연산(시트르산) 통살균 코스** (월 1~2회 권장):
1. 세제 투입 금지 (중요)
2. 구연산 40g을 세제 투입 위치에 투입
3. 통살균(또는 강력 세척) 코스 실행
4. 세척 후 배수 필터 및 내부 확인

### 주의
- 일반 세제로 내부 청소하면 거품 과다로 배수 불량 유발
- 식초나 베이킹소다는 권장하지 않음

## 잔수(고여 있는 물)

### 정상성
필터 아래에 **80~150ml (손가락 두세마디 높이)의 물이 고여 있는 것**은 정상이다.

**의의**:
- 설계상 구조로 하수 냄새 역류 방지
- 이전 세척에서 살균된 물이므로 재오염 우려 없음
- 다음 세척 시작 시 자동 배수됨

### 예외
물이 필터 아래를 넘어가는 경우만 강제 배수 필요 (위 참조)

## 정리: 우선 순위별 자가점검 체크리스트

### 배수 안 됨 (OE 에러)
1. ✓ 필터 분리 → 음식물 확인 → 세척 → 재설치
2. ✓ 배수 호스 높이 확인 (30cm 이상)
3. ✓ 배수 호스 눌림/꺾임 확인
4. → 해결 안 되면 강제 배수 테스트

### 물이 안 들어옴 (IE 에러)
1. ✓ 급수 밸브 개방 확인
2. ✓ 급수 호스 눌림/막힘/동파 확인

### 세척 불량
1. ✓ 그릇 배치 확인 (겹침, 노즐 차단 등)
2. ✓ 큰 음식물 제거 + 애벌세척
3. ✓ 세제 종류 확인 (고체, 전용 제품)
4. ✓ 세제량 조절
5. ✓ 세척 코스 선택 (강력 옵션 등)
6. → 필터 정기 청소 (1~2주)

### 건조 불량
1. ✓ **린스 레벨 확인** (가장 중요)
2. ✓ 식기 겹침 제거
3. ✓ 플라스틱 식기 분리 (온도 저하 문제)
4. ✓ 뒷굽 있는 그릇 기울임 배치

### 냄새
1. ✓ 구연산 40g + 통살균 코스 (월 1~2회)

## 미해결 항목

### 확인 못 한 것
- 정확한 필터 청소 주기 (근거: LG는 "1~2주", 삼성은 "주기적" 표기만 함)
- 각 에러코드별 정확한 자가 수리 가능 여부 (센서/모터 이상은 대부분 서비스 호출 필요할 가능성)
- 세제 브랜드별 성능 차이 구체적 수치
- 린스 브랜드별 건조 효과 차이 (일반론만 명시)
- 강제 배수 실패 시 다음 단계 (공식 문서에서 명확한 안내 없음 → 서비스 호출로 추정)
