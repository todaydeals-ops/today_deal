# 비데 자가점검 2차 — 물·온수·누수·동파·노즘 기계 고장

## 2단계: 웹 검색 결과 정리

### 검색 1: 물이 안 나옴 진단
**출처:** 렌트리, 삼성전자서비스
- 급수밸브 폐쇄가 가장 흔한 원인 → 최근 필터 교체·수도 작업 후 확인 필수
- 거름망 미청소·필터 미교체 → 이물질 쌓임 → 수압 약화
- 순서: (1) 급수밸브 확인 → (2) 거름망 청소 → (3) 초기화

### 검색 2: 온수 방식 차이
**출처:** 비데 온수 방식 비교 가이드, 인터텍(순간온수 모듈 제조사)
**순간온수식:**
- 가열 시간: 5초 이내
- 온도: 37~38도
- 장점: 세균 번식 없음, 전기료 최대 60% 절약
- 단점: 초기 찬물, 전력 소비 큼

**저수식:**
- 방식: 탱크에 물 저장, 지속 가열
- 장점: 저렴, 설치 쉬움, 수압 변화 적음
- 단점: 세균 번식 쉬움, 연속 사용 시 온도 저하(타이밍 이슈)

### 검색 3: 온수가 미지근한 원인
**출처:** AJD, 비데 사용법 가이드
- 계절 변화: 겨울(수온 낮음) → 더 높이 데워야 함 / 여름(수온 높음) → 적게 데움
- 저수식 연속 사용: 탱크 물이 점차 식어서 미온수 → 대기 후 재사용 권장(30초~1분)
- 해결: 온도 설정 높이기(37→40도), 대기 후 사용

### 검색 4: 누수 원인
**출처:** 삼성전자서비스, 렌트리
**호스 연결부:** 나사 헐거움 → 간단히 조임으로 해결
**레버 드레인:** 사용 중 흔들림 → 버튼 눌러 원위치로 돌림
**필터 막힘:** 물이 약하게 흐름 → 거름망 청소·필터 교체
**자동 세척:** 세정 후 노즘이 복귀하며 세척 물이 흐름 → 정상(10~15초 후 멈춤) [기존 팩트 재사용]

### 검색 5: 정전 후 초기화
**출처:** 삼성전자서비스, VOVO 매뉴얼
- 일반: 전원 코드 10초 이상 분리 후 재연결
- **삼성: 60초 이상 대기 필수**(60초 이내 재연결 시 리셋 안 됨)
- 코웨이: 10초 대기로 충분
- 버튼식: 세정+비데 동시 5초(코웨이) 또는 세정 3초(삼성) → 자동·어린이 불 깜빡 시 초기화 완료
- 초기화 후: 온도·수압 세기가 기본값으로 돌아감

### 검색 6: 겨울철 동파 & 노즘 안 나옴
**동파 방지:**
- 핵심: 보온 + 최소 난방 유지(10~15도)
- 계량기함: 에어캡·헌 옷으로 이중 보온
- 외부 배관: 동파 방지 열선·보온재로 감쌈
- 극저온 지역: 비데 내 잔수 배출(호스 연결부 풀어서) 또는 부동액 사용

**노즘 안 나옴:**
- 센서 감지 실패: 바지를 무릎까지 내리지 않음 / 너무 앞에 앉음
- 해결: (1) 전원 리셋 30초 → (2) 올바른 착좌 → (3) 강제 작동(세정 3초)
- 노즘 안 들어감: 모터·기어 고장 → **자가 수리 불가** → 고객센터

**구형 모델(~2010년 4월):**
- 기계식 센서(무게로 작동)
- 변좌 기울어짐 → 센서 오작동
- 해결: 변좌 하단 나사 조정해 수평 맞춤
