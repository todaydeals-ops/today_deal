# LG 그램 노트북 안 켜짐·발열·배터리 자가점검 — 조사 전문

**조사일:** 2026-07-21  
**주제:** LG 그램 노트북 전원 문제·발열·배터리 자가점검  
**출처:** LG전자 공식 고객지원 문서 (스스로 해결)

---

## 1. 전원이 켜지지 않을 때

### 1-1. 전원 어댑터 확인
LG 그램 노트북의 전원이 켜지지 않을 때 가장 먼저 확인할 것은 노트북의 AC 전원 표시등이다. 좌측 측면이나 전원 버튼 옆에 있는 표시등이 꺼져 있으면 전원 어댑터 자체 문제일 가능성이 높다.

### 1-2. 배터리 리셋 절차

**내장형 배터리 모델:**
1. AC 어댑터를 완전히 분리
2. 전원 버튼을 10초 이상 누르고 해제
3. 어댑터를 다시 연결하고 전원 켜기

일부 모델은 6초 이상이면 충분하지만, 10초 이상 누르는 것을 권장한다.

**외장형 배터리 모델:**
1. AC 어댑터와 배터리 모두 분리
2. 약 30초 대기하여 잔류 전원 제거
3. 배터리를 바닥면의 락 스위치로 다시 장착
4. 전원 켜기

외장형 배터리는 노트북 바닥면에 있는 배터리 락(Lock) 스위치로 제거한다.

### 1-3. 외부 장치 분리
LG 그램이 LG 로고에서 멈추거나 부팅이 안 될 때는 연결된 USB 기기, SD 메모리 카드, 외장 하드드라이브 등이 원인이 될 수 있다. 모든 외부 장치를 분리한 후 재부팅을 시도한다.

### 1-4. 적응형 대기 모드 무효화
일부 모델의 경우 BIOS의 적응형 대기 모드(Adaptive Standby) 기능이 활성화되어 있으면 부팅이 느려질 수 있다. 지속되는 문제는 전문 서비스 점검이 필요하다.

## 2. 배터리 80% 충전 제한 — 배터리 수명 연장 기능

### 2-1. 기능 설명
LG 그램 노트북의 배터리가 80%까지만 충전되는 것은 고장이 아니다. 배터리 수명 연장 기능(Battery Life Extension)이 활성화된 상태로, 충전 기준을 100%에서 80%로 낮춤으로써 배터리의 화학 노화를 지연시킨다. 이 기능이 켜져 있으면 어댑터를 계속 꽂아도 80% 이상 충전되지 않는다.

### 2-2. 모델별 설정 방법

**2025년 이후 Gram 모델:**
- 작업 표시줄 검색 → "My gram" 실행
- PC 관리 → 전원 및 성능 → 배터리 수명 연장
- ON/OFF 선택

**2022년 4월 ~ 2024년 모델:**
- 작업 표시줄 검색 → "LG Smart Assistant" 실행
- 전원 설정 → 배터리 수명 연장
- ON/OFF 선택

**2022년 4월 이전 모델:**
- 작업 표시줄 검색 → "LG Control Center" 실행
- 전원관리 설정 → 배터리 수명 연장
- ON/OFF 선택

LG Update 프로그램을 통해 앱이 자동으로 설치되며, 데스크톱 비OS 제품은 Windows 설치 후 수동 설치가 필요하다.

### 2-3. 사용 권장사항
배터리 케어를 원한다면 배터리 수명 연장을 ON으로 두면 되고, 단기간 최대 용량이 필요하다면 OFF로 설정한다.

## 3. 발열 및 팬 소음

### 3-1. 냉각 모드 설정
발열과 팬 소음의 가장 흔한 원인은 냉각 모드가 저소음(Silent) 모드로 설정되어 있는 것이다. 저소음 모드는 팬 회전을 줄여 조용하지만 온도 관리가 약해지므로, 발열이 심하면 권장 모드로 변경한다.

**냉각 모드 변경:**
- Fn+F1 단축키 (모델별 상이할 수 있음)
- 또는 My gram/LG Smart Assistant 앱 → 냉각 설정에서 변경

### 3-2. 물리적 환경 확인
- 노트북을 평탄하고 단단한 표면(책상, 하드 쿠션)에 놓을 것
- 침대, 소파, 카펫 위에서 사용하면 통풍구가 막혀 발열이 심해짐
- 노트북 모든 측면에서 최소 15cm 이상 통풍 간격 확보
- 히터·에어컨 출구 근처 설치 회피

### 3-3. 통풍구 청소
먼지 축적이 발열의 또 다른 원인이다. 통풍구 주변의 눈에 보이는 먼지를 부드럽게 제거한다.

**주의:**
- 압축공기 금지 (팬 블레이드 손상 위험)
- 부드러운 천이나 솔로 가볍게 제거만 할 것
- 내부 팬 청소가 필요하면 고객센터(1544-7777) 전문 서비스 이용

"드르륵", "덜거덕" 같은 비정상 소음이 나거나 CPU 냉각팬의 안쪽 청소가 필요하면 분해하지 말고 전문가에게 의뢰한다.

### 3-4. 백그라운드 프로세스 정리
- 작업 관리자 열기 (Ctrl+Shift+Esc)
- 불필요한 백그라운드 프로세스 종료
- Windows Defender 빠른 검사 실행
- 필요시 Microsoft 악성코드 제거 도구 다운로드

### 3-5. BIOS 업데이트
일부 모델의 경우 BIOS를 최신 버전으로 업데이트하면 팬 속도 조절 기능이 개선되어 소음 문제가 해결될 수 있다. LG Update 프로그램에서 확인할 수 있다.

## 4. 화면이 안 나올 때

### 4-1. 외부 모니터로 진단
화면 문제의 원인을 빠르게 파악하려면 외부 모니터를 HDMI로 연결하는 것이 가장 유효하다.

- **외부 모니터에 화면이 나옴:** 노트북 디스플레이(LCD 패널, 연결 케이블, 디스플레이 컨트롤러) 고장 → 전문 서비스 필요
- **외부 모니터에도 화면이 안 나옴:** 그래픽 카드, 메인보드, 전원 등 상위 시스템 문제 → 전문 서비스 필요

### 4-2. HDMI 다중 디스플레이 설정
HDMI 케이블이 제대로 연결되었는데도 화면이 안 나올 때:

1. Fn+F7 단축키로 다중 디스플레이 설정 활성화
2. 디스플레이 설정에서 [확장] 또는 [복제] 선택
   - [PC 화면만]으로 설정되어 있으면 외부 기기에는 신호가 가지 않음

### 4-3. HDMI 케이블 검증
- 현재 HDMI 케이블을 다른 케이블로 교체 후 테스트
- 손상된 케이블과 정상 포트를 구분하기 위함

### 4-4. 그래픽 드라이버 재설치
디스플레이 설정과 HDMI 케이블이 정상이어도 화면이 안 나올 때:

- LG Update 프로그램 실행 → 그래픽 드라이버 항목 확인 및 재설치
- 또는 장치 관리자에서 디스플레이 어댑터 > 그래픽 카드 드라이버 우클릭 > 업데이트

## 5. 터치패드 작동이 안 될 때

### 5-1. 단축키로 활성화

**2025년 Gram 모델:**
- Fn + F4 누르기 → [Touchpad On] 표시

**2024년 이전 모델:**
- Fn + F5 누르기 → [Touchpad On] 표시

일부 모델은 터치패드 아이콘 표시 F 키를 사용할 수도 있다.

### 5-2. Windows 터치패드 설정 확인
1. Windows 설정 검색 → "터치 패드 켜기/끄기"
2. 토글이 [켬]으로 되어 있는지 확인
3. "마우스 연결 시 터치패드 비활성화" 옵션 확인
   - 체크되어 있으면 외부 마우스 연결 시 터치패드가 자동으로 꺼짐

### 5-3. HotKeyManager 활성화
Fn+F 단축키가 작동하지 않을 때:

1. 작업 관리자 열기 (Ctrl+Shift+Esc)
2. [시작프로그램] 탭 확인
3. HotKeyManager가 "비활성화" 상태면 우클릭 > [활성화]
4. 시스템 재부팅

HotKeyManager가 비활성화되면 모든 Fn+F 함수 키 단축키가 작동하지 않는다.

### 5-4. 터치패드 드라이버 재설치
위의 모든 단계를 거쳤는데도 작동하지 않을 때:

- 작업 표시줄 검색 → 모델별 LG Update 프로그램 실행
  - LG Update & Renew
  - LG Update & Recovery
  - LG DnA Center Plus
  - 등 (모델마다 상이)
- 터치패드 관련 드라이버 항목 확인 및 재설치
- 시스템 재부팅

## 6. LG Smart Assistant 활용 (2022년 4월 이후 모델)

### 6-1. 시스템 진단 기능
LG Smart Assistant는 2022년 4월 이후 출시된 모든 LG 그램에 기본 탑재된 기능이다(데스크톱 비OS 제품은 별도 설치).

**진단 실행:**
1. 작업 표시줄 검색 → "LG Smart Assistant" 실행
2. 시스템 진단 섹션 → [진단하기] 클릭
3. 자동으로 6가지 항목 점검:
   - CPU
   - 메모리(RAM)
   - 저장공간(SSD/HDD)
   - 배터리
   - 네트워크
   - 기타 시스템 상태

문제가 감지되면 [해결하기] 버튼이 활성화되어 자동 해결을 시도한다.

### 6-2. 배터리 건강도 확인
전원 설정 → [배터리 수명 상태]에서 현재 배터리 용량이 정상 용량의 몇 %인지 확인할 수 있다. 80% 이하면 수명이 다한 상태로 교체를 고려해야 한다.

### 6-3. 사용 목적별 프리셋 모드
사용 환경에 맞게 미리 설정된 모드를 선택하면 화면 밝기, 냉각 모드, 전원 설정이 자동으로 최적화된다:
- 공부 (저밝기, 저소음)
- 업무 (고밝기, 고성능)
- 영화 (중밝기, 영상 최적화)
- 야외 (최고밝기)
- 사용자 정의

### 6-4. 고급 기능
- USB-C 오프라인 충전
- 부재중 자동 화면 꺼짐
- AI 노이즈 제거 (화상회의)
- 키보드 백라이트 조절
