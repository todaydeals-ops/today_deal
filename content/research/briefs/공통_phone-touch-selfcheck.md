# 스마트폰 터치 불량·고스트터치·액정 반응 없음 자가점검 조사 전문

조사 기간: 2026-07-21
조사자 노트: 공통 팩트 저장소. 갤럭시·아이폰 공통 원인과 기종별 절차 분리.

---

## 개요

스마트폰 터치 불량은 크게 두 가지 분류다:
1. **터치 미반응**: 손가락으로 화면을 눌러도 반응 없음
2. **고스트터치**: 손을 대지 않았는데 화면이 저절로 반응함

두 증상 모두 **소프트웨어 오류**(앱 충돌, 설정값)와 **하드웨어 손상**(센서/디스플레이 불량)으로 나뉘므로, 자가점검 순서가 중요하다.

---

## 자가점검 순서 (우선순위)

### 1단계: 환경 요소 확인 (무료)

#### 보호필름·케이스 제거
- **갤럭시**: 삼성전자서비스(39691)에서 명시적으로 "비정품 케이스의 높이 차이로 인한 가장자리 터치 미인식" 언급
- **아이폰**: Apple은 공식 문서에서 케이스·필름 제거 후 테스트를 권장
- 저품질 보호필름이 가장 흔한 원인 중 하나

#### 온도 확인
- **안전 작동 온도**: 0°C ~ 35°C (Apple 공식)
- 45°C 이상: 성능 저하, 터치 반응성 감소
- 극저온(-10°C 이하): 배터리 성능 저하로 터치 센서 전력 부족
- **겨울 야외**: 터치 안 됨 → 실온으로 옮겨 대기
- **차량 내**: 직사광선 아래 여름 차량은 70°C 이상 가능 → 즉시 꺼내 냉각

#### 습기 확인
- **물 노출 직후**: 고스트터치 가능성 높음
- **추운→따뜻한 환경 이동 시**: 기기 내부 결로(습기) 발생
- 해결: 자연 건조(몇 시간~1일). **냉동실 금지** (결로 악화)

#### 충전기 상태
- **증상**: 충전 중에만 터치 오류·고스트터치 발생
- **원인**: 비정품 충전기의 미세 전류 노이즈가 정전기식 터치 센서 교란
- **테스트**: 정품 충전기로 교체 또는 충전 제거 후 테스트
- **결과**: 다른 충전기에서 정상 = 기존 충전기 불량

---

### 2단계: 소프트웨어 오류 확인 (재부팅)

#### 강제 재시작
**갤럭시**: 측면 버튼 + 음량(−) 버튼 7초 이상
**아이폰**: 기종별로 다름
- iPhone 8 이후: 볼륨 업 빠르게 → 볼륨 다운 빠르게 → 전원 버튼 길게
- iPhone 7/7 Plus: 전원 + 음량(−) 버튼 7초 이상

일시적 소프트웨어 오류의 대부분은 강제 재시작으로 해결된다.

---

### 3단계: 터치 센서 자체 진단 (갤럭시만)

#### 다이얼러 진단 코드: `*#0*#`
1. 전화 앱 → 키패드에서 `*#0*#` 입력
2. 진단 모드 실행 → [Touch] 선택
3. 화면에 격자무늬(그리드) 표시
4. 손가락으로 그으면 녹색으로 채워짐
5. **모든 칸이 녹색** = 센서 정상
6. **특정 칸이 반응 없음** = 해당 영역 불량

이 테스트는 터치 센서의 물리적 작동만 확인하며, 필름/케이스 간섭이나 소프트웨어 오류는 구분하지 못한다.

#### 삼성 Members 앱 진단
1. Members 앱 실행 → 자가진단 → 폰 진단
2. 전체 진단 또는 터치 화면 항목만 선택
3. 진단 결과 참고 (정식 진단은 서비스센터에서)

---

### 4단계: 앱 오류 vs 하드웨어 손상 구분 (안전모드)

#### 안전모드 진입
**갤럭시**: 부팅 시 삼성/갤럭시 로고 나타날 때 음량(−) 버튼 계속 누르기
또는 빠른 설정 → 전원 아이콘 > 다시 시작 → "안전모드로 시작" 선택

이 모드에서는 타사 앱이 모두 비활성화된다.

#### 안전모드 테스트 결과 해석
- **안전모드에서 터치 정상** = 설치된 앱 오류 → 최근 설치/업데이트 앱 삭제
- **안전모드에서도 터치 불량** = 하드웨어 결함 가능성 높음 → 서비스센터 필요

---

### 5단계: 터치 설정 확인

#### 갤럭시 터치 민감도
- 설정 > 디스플레이 > 터치 민감도 > 켜기
- 보호필름 사용 시 민감도를 높이면 도움이 될 수 있음

#### TalkBack 혹은 접근성 기능 확인
- 음성 가이드가 나오면 터치가 접근성 기능으로 인식될 수 있음
- 설정 > 접근성에서 TalkBack, 돋보기 등 확인 후 비활성화

## 공장 초기화 (마지막 수단)

### 사전 필수 조건
1. **반드시 백업** (아래 참조)
2. 터치가 완전히 작동하지 않으면 음성 제어 또는 접근성 메뉴 사용

### 백업 방법 (터치 불량 상황)

#### 터치 가능한 경우
- 갤럭시: 설정 > 계정 > Samsung 계정 (또는 Google) → 동기화
- 아이폰: 설정 > [사용자명] > iCloud → 백업 지금

#### 터치 거의 불가능한 경우
- **갤럭시**: Bixby 음성 명령 또는 음량 버튼만 사용 가능한 설정 메뉴로 진입
- **아이폰**: Siri 음성 제어 ("설정 열기" 등)로 iCloud 백업 강제 실행
- 또는 컴퓨터에 연결하여 클라우드 동기화 대기

### 초기화 유형
1. **설정값 초기화** → 네트워크 설정 초기화 → **공장 초기화** (순서대로 시도)
2. 설정값 초기화는 데이터를 보존하므로 먼저 시도

## 터치 불량의 일반적 원인 정리표

| 원인 | 증상 | 진단 방법 | 해결책 |
|---|---|---|---|
| 보호필름/케이스 | 가장자리 터치 미반응, 전반적 둔한 반응 | 제거 후 테스트 | 정품/고품질 교체 또는 제거 |
| 충전기 노이즈 | 충전 중만 터치 오류 | 정품 충전기로 교체 | 정품 충전기 사용 |
| 습기 | 고스트터치, 버튼 저절로 눌림 | 드라이 환경에서 24시간 대기 후 테스트 | 자연 건조, 서비스센터 수리 |
| 온도 | 극저온/극고온에서만 반응 없음 | 온도 조절 후 테스트 | 환경 온도 조절 |
| 앱 충돌 | 특정 앱 사용 시만 터치 오류 | 안전모드 테스트, 최근 앱 삭제 | 앱 재설치 또는 삭제 |
| 디스플레이 하드웨어 | 다이얼러 진단에서 특정 영역 미반응 | `*#0*#` 진단 코드 | 서비스센터 수리 |
| 터치 센서 하드웨어 | 전체 화면 터치 작동 안 함 | 다이얼러 진단, 안전모드 | 서비스센터 수리 |

## 주의사항 (금지사항)

### 절대 금지
1. **분해**: 터치 센서는 정밀 부품. 자가 분해는 취급 부주의로 손상 가능.
2. **냉동실 냉각**: 급격한 온도 변화는 내부 결로를 악화시킴.
3. **강한 세제/알코올**: 디스플레이 코팅 손상.
4. **무리한 누르기**: 이미 손상된 센서를 추가 손상 유발.

### 신중히 접근
- **공장 초기화**: 데이터 전부 삭제. 꼭 필요할 때만.
- **비정품 부품**: 저가 터치 센서 교체는 신뢰성 낮음. 공식 서비스 권장.

## 서비스센터 방문 체크리스트

다음 중 하나라도 해당되면 서비스센터 필수:

- [ ] 위의 1~5단계 모두 완료했는데도 터치 불량 지속
- [ ] 다이얼러 진단에서 특정 영역이 반응하지 않음
- [ ] 안전모드에서도 터치 불량 지속
- [ ] 물에 노출 후 고스트터치 발생
- [ ] 시각적 화면 균열·손상이 보임

**준비물**: 
- 구매 영수증 또는 보증서
- 제품명·시리얼 번호 확인
- 백업 완료 확인
