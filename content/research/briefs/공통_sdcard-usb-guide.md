# SD카드·USB메모리 속도 등급 표기 읽는 법 및 선택 가이드

## 조사 개요
- **핵심 문제**: SD카드 및 USB 메모리의 속도 등급 표기가 여러 체계(Class, UHS, V30, A2 등)로 혼재되어 있어 소비자 혼동 심각
- **스마트가이드 목표**: 표기 의미를 표로 정리하고, 용도별 필요 등급을 명확히 제시

## 주요 조사 결과

### 1. 속도 등급 표기 체계 (표로 정리 가능)

**Speed Class (클래스 체계)**
- 최소 쓰기 속도를 보장하는 가장 기초적 표기
- Class 2 = 최소 2MB/s, Class 10 = 최소 10MB/s
- 원 안에 숫자로 표시 (C10 등)

**UHS Speed Class (울트라 하이스피드 체계)**
- 더 현대적 표기로, 읽기/쓰기 성능이 높은 카드에 사용
- U1 = 최소 10MB/s, U3 = 최소 30MB/s
- U자 안에 숫자로 표시

**Video Speed Class (비디오용 체계)**
- 4K 영상 촬영, 블랙박스, 드론처럼 연속 녹화가 필요한 용도 표기
- V10 = 최소 10MB/s, V30 = 최소 30MB/s, V60 = 최소 60MB/s, V90 = 최소 90MB/s
- V자 안에 숫자로 표시
- V30과 U3는 같은 속도(최소 30MB/s)를 의미

**App Performance Class (앱 성능 클래스)**
- 스마트폰 등에서 앱 설치·실행 성능에 영향
- A1 = 최소 읽기 1500 IOPS, 쓰기 500 IOPS
- A2 = 최소 읽기 4000 IOPS, 쓰기 2000 IOPS (더 빠름)

**읽기 속도 vs 쓰기 속도**
- 패키지의 큰 숫자는 보통 최대 읽기 속도 (예: 170MB/s)
- 실제 사용에서 중요한 것은 쓰기 속도 (녹화·저장 속도)
- 규격으로 명시된 속도는 최소/최대가 아니라 보장 속도

### 2. 정품 vs 가짜 구별 방법

**외관 검사 (육안 구분)**
- 정품: 카드 위 글씨가 선명하고 깔끔
- 가짜: 글씨가 거칠고 흐림, 제조국이 대만 등 이상
- 라벨이 품질 저가형으로 부착된 제품은 의심

**용량 뻥튀기 판별 (H2testw 사용)**
- 가짜 카드의 가장 흔한 사례: 실제 32GB를 128GB로 표시
- Windows용 무료 도구 H2testw 사용: 더미 파일로 실제 용량 테스트
- 테스트 완료 후 "Test finished without errors"면 정상
- 오류 발생하면 용량 뻥튀기 확정

**구매 안전지침**
- 공인 판매처(Amazon, 공식 한국 대리점)에서 구매
- 비정상적 저가 제품 피할 것
- 중고·리퍼브 제품은 가짜 위험 높음

### 3. 용도별 필요 등급 (표로 정리 권장)

**4K 영상 촬영 (카메라/캠코더/액션캠)**
- 필수: V30, U3 이상 (최소 쓰기 30MB/s)
- 권장: V60, U3 (더 안정적)
- 이유: 4K는 초당 11MB 이상 기록 필요

**블랙박스**
- 추천: Class 10, U1, V10 등급
- 고온 주의: 여름철 차내 90℃ 이상에서 카드 수명 단축 (6개월~1년 교체)
- 주기적 포맷: 2주~1개월 권장

**드론**
- 4K 촬영: V30, U3 이상
- 용량: 32GB 이상 (4K는 저장 공간 빠르게 소모)

**스마트폰**
- A2 등급이 있으면 앱 로딩 빠름
- 최소: Class 10, U1 이상

**일반 파일 백업**
- Class 10으로 충분
- 속도보다는 용량(32GB 이상) 중요

### 4. microSD vs 표준 SD 카드

**크기 차이**
- microSD: 11mm × 15mm (초소형)
- SD: 24mm × 32mm (표준)
- 어댑터로 microSD를 SD 슬롯에 사용 가능

**속도 성능**
- 제조 과정의 차이로 같은 등급 표기해도 microSD가 다소 느릴 수 있음 (예: KIOXIA 최대 쓰기 90MB/s vs SanDisk 140MB/s)
- 용도(폰, 태블릿, 드론, 게임기): microSD 표준
- 용도(카메라, 비디오): 표준 SD 카드가 일반적

**호환성**
- 카드리더기: 반드시 SDXC 지원 제품 선택 (SDHC는 64GB 카드 미지원)
- 가격: 동일 용량 시 microSD가 약 1/3 저렴

### 5. USB 규격과 카드리더기

**USB 3.0 vs 3.1 vs 3.2 속도**
- USB 3.0 = USB 3.1 Gen1 = USB 3.2 Gen1: 5Gbps (약 625MB/s)
- USB 3.1 Gen2 = USB 3.2 Gen2: 10Gbps (약 1.25GB/s)
- USB 3.2 Gen 2×2: 20Gbps (약 2.5GB/s)
- 카드리더기는 USB 3.0 이상 권장 (USB 2.0은 최대 20MB/s로 느림)

**카드리더기 선택 기준**
- SDXC 지원 필수 (64GB 이상 카드 사용 가능)
- USB 3.0 이상 (고속 전송)
- 호환성: "SD / SDHC / SDXC 지원" 명시 제품 선택

### 6. 용량별 파일시스템 선택

**FAT32 (구형)**
- 호환성: 모든 OS 지원
- 제한: 단일 파일 4GB 제한, 전체 용량 32GB 이상 부권장
- 용도: 레거시 장비(구형 카메라) 호환 필요할 때만

**exFAT (권장, 크로스 플랫폼)**
- 호환성: Windows, Mac, Linux 모두 읽기·쓰기 가능
- 장점: 파일 크기 제한 없음, 포터블 용도에 최적화
- 추천: Windows·Mac 함께 사용할 계획이면 필수

**NTFS (Windows 전용)**
- 호환성: Mac에서는 기본 읽기만 가능 (쓰기 불가)
- 기능: 암호화, 권한 관리 지원
- 크로스 플랫폼 사용 시 부적합

### 7. 수명과 백업 주의사항

**메모리카드 수명**
- 일반적 수명: 6개월~1년 (특히 블랙박스)
- 고온 영향: 60℃에서도 화질 저하 보고, 90℃에서는 대부분 손상
- 반복 쓰기로 인한 셀 손상: 포맷 반복 시 가속화

**정기 유지보수**
- 주기적 포맷: 2주~1개월 (오버라이트 데이터 정리)
- 포맷 전 반드시 백업
- 중요 영상은 별도 저장소(PC, 클라우드)에 즉시 백업

**구매 시 안전 팁**
- 신규 카드보다는 평가 많은 기존 모델 추천 (신형 부작용 미알려)
- 블랙박스용 카드는 고내구 전문 제품 검토 (Samsung PRO 등)
- 여름철 야외주차 피하거나 그늘막 사용

## 작성 주의사항
- 표의 속도 수치는 보장(minimum) 속도임을 명시할 것
- "최대 읽기 속도"와 "보장 쓰기 속도" 구분 강조
- 높은 숫자 표기만 보고 판단하는 소비자 심리 고려 → 표기 의미 풀이 강조
- V30과 U3의 동일성 설명 (혼동 방지)
- 사용자 후기와 공식 사양의 차이 명시 (예: 선언된 속도 vs 실제 측정)
